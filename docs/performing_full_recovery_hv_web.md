# Restoring Entire VMs Using Web UI

In this article

To restore an entire VM using the web UI, use the Entire VM Restore wizard.

1. [Launch the wizard](full_restore_launch_hv_web.md).
2. [Select VMs and restore points](full_restore_vms_hv_web.md).
3. [Select a restore mode](full_restore_mode_hv_web.md).
4. [Select target hosts](full_restore_host_hv_web.md).
5. [Select target datastores](full_restore_datastore_hv_web.md).
6. [Specify network mapping](full_restore_network_hv_web.md).
7. [Change VM Names and UUIDs](full_restore_names_hv_web.md).
8. [Specify a restore reason](full_restore_reason_hv_web.md).
9. [Verify restore settings](full_restore_review_hv_web.md).
