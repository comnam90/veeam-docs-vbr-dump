# Performing Failover

In this article

For more information on failover, see [Failover](failover.md) and [Failover and Failback for Replication](failover_failback.md).

To perform failover, use the VMware Failover wizard.

1. [Check prerequisites](failover_before_you_begin.md).
2. [Launch the Failover wizard](failover_launch_vm.md).
3. [Select VMs and restore points](failover_vms_vm.md).
4. [Specify a failover reason](failover_reason_vm.md).
5. [Review the summary and finish working with the wizard](failover_summary_vm.md).
