# Publishing Disks

In this article

To publish a disk, use the Publish Disks wizard.

1. [Launch the wizard](publishing_disks_launch.md).
2. [Select a workload whose disks you want to restore](publishing_disks_workloads.md).
3. [Select a restore point](publishing_disks_point.md).
4. [Select disks](publishing_disks_disks.md).
5. [Specify the target server](publishing_disks_target.md).
6. [Specify a reason](publishing_disks_reason.md).
7. [Finish working with the wizard](publishing_disks_summary.md).
