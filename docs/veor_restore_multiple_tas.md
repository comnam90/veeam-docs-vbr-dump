# Restoring to Another Server


To restore multiple Oracle databases to another server, do the following:

1. [Launch the Restore wizard](veor_restore_multiple_tas_wizard.md).
2. [Specify a restore point](veor_restore_multiple_tas_specify_restore_point.md).
3. [Specify a target server](veor_restore_multiple_tas_specify_target_server.md).
4. [Specify Oracle settings](veor_restore_multiple_tas_specify_oracle_settings.md).
5. [Review the restore summary](veor_restore_multiple_tas_summary.md).
