# Mail Settings


Veeam Explorer for Microsoft Teams allows you to send Microsoft Teams posts and files as attachments in email messages.

Veeam Explorer for Microsoft Teams sends email messages on behalf of your Google or Microsoft 365 account using [OAuth 2.0 Authorization Framework](https://datatracker.ietf.org/doc/html/rfc6749), or you can specify connection settings of an SMTP server that uses basic authentication.

For more information, see the following sections:

* [SMTP Server with Basic Authentication](vet_configuring_smtp.md)
* [Google Account](vet_google_account.md)
* [Microsoft 365 Account](vet_m365_account.md)
