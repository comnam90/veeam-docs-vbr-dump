# Step 1. Launch Restore Wizard

In this article

To launch the Restore wizard, do the following:

1. In the navigation pane, select the SAP HANA system.
2. On the System tab, select Restore Database > Restore the latest state to <original\_location>.

Alternatively, you can right-click the server and select Restore latest state to <original\_location>.
