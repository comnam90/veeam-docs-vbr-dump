# Adding Lenovo ThinkSystem DM/DG Series

In this article

Before you add a Lenovo ThinkSystem DM/DG series storage system to the backup infrastructure, check [Planning and Preparation](storage_limitations.md). To add the storage system to the backup infrastructure, use the New Lenovo ThinkSystem DM Series Storage wizard.

1. [Launch the wizard](lenovo_type.md).
2. [Specify Lenovo ThinkSystem server name or address and storage role](lenovo_add_name.md).
3. [Specify credentials and protocol type](lenovo_add_creds.md).
4. [[For Microsoft Windows-based backup server] Specify VMware access options](specify_vmware_access.md).
5. [[For Microsoft Windows-based backup server] Specify Veeam Agent access options](specify_agents_access.md).
6. [Specify NAS access options](specify_nas_access.md).
7. [Apply settings](lenovo_add_apply.md).
8. [Finish working with the wizard](lenovo_add_rescan.md).
