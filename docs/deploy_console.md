# Veeam Backup & Replication Console Installation

In this article

The Veeam Backup & Replication console is a client-side component that provides access to the backup server. Install a Veeam Backup & Replication console on a dedicated Windows server to access the backup server remotely.By default, the Veeam Backup & Replication console is installed on the backup server automatically when you install Veeam Backup & Replication. You do not need to install the console manually.

However, in addition to the default console, you can install the Veeam Backup & Replication console on a dedicated machine to access the backup server remotely. You can install as many remote consoles as you need. For more information, see [Backup & Replication Console](backup_console.md).
