# Launching Veeam Explorer from Plug-in or MongoDB Backup


To restore application items from plug-in or MongoDB backups, perform the steps specified in the following topics:

* [Restoring from Backup with Veeam Explorer MongoDB](mongo_application_backups_restore.md) — to restore from MongoDB replica sets backed up with MongoDB Backup.

* [Restore with Veeam Explorer for SAP HANA](restore_vehana.md) — to restore SAP HANA databases backed up with Veeam Plug-in for SAP HANA.

* [Restore with Veeam Explorer for Microsoft SQL Server](mssql_db_restore_vesql.md) — to restore Microsoft SQL Server databases backed up with Veeam Plug-in for Microsoft SQL Server.
* [Restore with Veeam Explorer for Oracle](restore_veor.md) — to restore Oracle databases backed up with Veeam Plug-in for Oracle RMAN.

For more information, see [Veeam Explorers Overview](explorers_introduction.md).
