# Instant Recovery from Storage Snapshots


You can instantly recover a VM from a storage snapshot without prior de-staging and intermediate restores. Instant Recovery accelerates VM restore, improves RTOs and decreases downtime of production VMs. For more information, see the [Instant Recovery to VMware vSphere](instant_recovery.md) section in the Veeam Backup & Replication User Guide.

Requirements for Instant Recovery from Storage Snapshots

Before you perform backup from storage snapshots, check requirements specific for data recovery in [Data Recovery from Storage Snapshots Requirements and Limitations](storage_limitations_general.md#boss).

Performing Instant Recovery from Storage Snapshots

For information on performing Instant Recovery, see section the [Performing Instant Recovery to VMware vSphere](instant_recovery.md) section in the Veeam Backup & Replication User Guide.
