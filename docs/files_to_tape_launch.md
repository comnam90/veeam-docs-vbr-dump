# Step 1. Launch New File to Tape Job Wizard


To run the New File to Tape Job wizard, do either of the following:

* On the Home tab, click Tape Job and select Files.
* Open the Home view, right-click the Jobs node and select Tape Job > Files to Tape.

Alternatively, you can:

* Open the Files view, browse to the necessary files, select the files and choose Add to Tape Job > New job from the ribbon menu.
* Open the Files view, browse to the required files, right-click the necessary files and choose Add to Tape Job > New job.
