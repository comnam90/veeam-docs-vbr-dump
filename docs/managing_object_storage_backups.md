# Managing Object Storage Backups


You can manage your object storage backups in various ways: import, detach or remove object storage backups or seed them to Azure Data Box or AWS Snowball Edge storage devices.

* [Seeding Backups to Azure Blob Storage](data_box_seeding.md).
