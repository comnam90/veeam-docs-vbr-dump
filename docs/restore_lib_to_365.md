# Restore to Microsoft 365 Organizations

In this article

You can restore backed-up data to Microsoft 365 organizations with enabled [multi-factor authentication](https://docs.microsoft.com/en-us/microsoft-365/admin/security-and-compliance/set-up-multi-factor-authentication?view=o365-worldwide) (MFA). For more information, see the [Adding Microsoft 365 Organizations](https://helpcenter.veeam.com/docs/vbo365/guide/vbo_add_office365_org.html?ver=80) section of the Veeam Backup for Microsoft 365 User Guide.

To restore Microsoft SharePoint document libraries and lists to another Microsoft 365 organization, do the following:

1. [Launch the Restore wizard](vesp_restore_lib_launch_wizard_m365.md).
2. [Select a target site location](restore_lib_target_location_m365.md).
3. [Select an authentication method](vesp_select_authentication_method_lib_m365.md).
4. [Log in to Microsoft 365](vesp_login_lib_m365.md).
5. [Specify a target site web address and a target list](specify_target_site_list_m365.md).
6. [Specify restore options](vesp_specify_restore_options_lib_m365.md).
7. [Review the operation summary](summary13.md).
