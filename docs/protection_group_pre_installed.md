# Creating Protection Group for Computers with Pre-installed Backup Agents


Before you create a protection group for computers with pre-installed backup agents, [check prerequisites](pg_pre_installed_before.md). Then use the New Protection Group wizard to configure a protection group.

1. [Launch the New Protection Group wizard](pg_pre_installed_launch.md).
2. [Specify protection group name and description](pg_pre_installed_name.md).
3. [Specify packages](protection_group_packages.md).
4. [Specify advanced protection group settings](pg_pre_installed_advanced.md).
5. [Assess results](pg_pre_installed_results.md).
6. [Finish working with the wizard](pg_pre_installed_finish.md).
