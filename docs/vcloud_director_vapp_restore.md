# Restoring Cloud Director vApps


To restore a vApp to VMware Cloud Director, use the vCloud Full vApp Restore wizard.

1. [Launch the Full vApp Restore wizard](vcloud_director_vapp_restore_launch.md).
2. [Select an object to restore](vcloud_director_vapp_restore_object.md).
3. [Select a restore point](vcloud_director_vapp_restore_point.md).
4. [Select a restore mode](vcloud_director_vapp_restore_mode.md).
5. [Select the vApp location](vcloud_director_vapp_restore_location.md).
6. [Select a destination network](vcloud_director_vapp_restore_network.md).
7. [Select a template to link](vcloud_director_vapp_template.md).
8. [Select a storage policy and datastores](vcloud_director_vapp_restore_profile.md).
9. [Specify secure restore settings](vcloud_director_vapp_av.md).
10. [Specify a restore reason](vcloud_director_vapp_restore_reason.md).
11. [Verify recovery settings and finish working with the wizard](vcloud_director_vapp_restore_verify.md).
