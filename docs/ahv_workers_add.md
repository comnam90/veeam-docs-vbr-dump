# Adding Workers


To deploy a worker and add it to the backup infrastructure, do the following:

1. [Check prerequisites and limitations](ahv_workers_add_byb.md).
2. [Launch the Add Worker wizard](ahv_workers_add_launch.md).
3. [Specify worker VM configuration](ahv_workers_add_vm.md).
4. [Specify worker network settings](ahv_workers_add_network.md).
5. [Finish working with wizard](ahv_workers_add_summary.md).
