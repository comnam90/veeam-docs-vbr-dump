# Restore with Restore Database Wizard


You can restore Microsoft SQL Server databases using the Restore Database wizard.

If you want to restore databases from a Veeam Plug-In backup on another server, launch the wizard on the target server, where you want to place the new database backup.

Before restoring a database, [check prerequisites](planning_and_preparation_mssql.md). Then use the wizard to define restore settings and start the restore process.

1. [Launch the Restore Database wizard](mssql_db_restore_launch.md).
2. [Select the database to restore and choose the restore point](mssql_db_restore_source.md).
3. [Specify the point in time to which you want to restore the database](mssql_db_restore_pit.md).
4. [Specify where and how to restore the database](mssql_db_restore_target.md).
5. [Monitor the restore process](mssql_db_restore_process.md).
