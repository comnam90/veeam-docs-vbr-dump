# Retention of Microsoft SQL Server Backups


When using Veeam Plug-In for Microsoft SQL Server, you can configure the retention policy in the Back Up Database wizard. For details, see [Configuring Retention Policy](plugins_mssql_retention_policy.md#studio). Also, you can manually delete backups from a backup repository using the Veeam Backup & Replication console and force delete backups with the command-line interface. For details, see [Force Deleting Backups](plugins_mssql_retention_force.md).

Alternatively, delete backups from a backup repository using the Veeam Backup & Replication console. For details, see [Deleting Backup Using Veeam Backup & Replication Console](delete_backups_mssql.md).
