# Step 8. Specify Advanced Backup Settings

In this article

At the Storage step of the wizard, you can specify the advanced settings for the backup job:

* [Backup settings](backup_job_advanced_backup_hv.md)
* [Maintenance settings](backup_job_advanced_maintenance_hv.md)
* [Storage settings](backup_job_advanced_storage_hv.md)
* [Notification settings](backup_job_advanced_notify_hv.md)
* [Hyper-V settings](backup_job_advanced_hv_hv.md)
* [Script settings](backup_job_advanced_script_hv.md)

|  |
| --- |
| Tip |
| After you specify necessary settings for the backup job, you can save them as default settings. To do this, click Save as Default at the bottom left corner of the Advanced Settings window. When you create a new backup job, Veeam Backup & Replication will automatically apply the default settings to the new job. |
