# Creating Tenant Backup Jobs


To create a Microsoft Entra ID tenant backup job, do the following:

1. [Launch the New Microsoft Entra ID Tenant Backup Job wizard](entra_id_backup_launch.md).
2. [Specify a job name and description](entra_id_job_name.md).
3. [Specify a tenant and retention settings](entra_id_job_tenant.md).
4. [Specify backup copy settings](entra_id_tenant_backup_copy.md).
5. [Define a job schedule](entra_id_job_schedule.md).
6. [Finish working with the wizard](entra_id_job_summary.md).
