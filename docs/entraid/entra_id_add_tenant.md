# Adding Microsoft Entra ID Tenants


To add a Microsoft Entra ID tenant, do the following:

1. [Launch the Microsoft Entra ID Tenant wizard](entra_id_add_tenant_launch.md).
2. [Specify the tenant ID and select a cache repository](entra_id_add_tenant_id.md).
3. [Define a protection scope](entra_id_tenant_protection_scope.md).
4. [Choose an application to access the tenant](entra_id_tenant_connection.md).
5. [Wait for the operation to complete](entra_id_add_tenant_apply.md).
6. [Finish working with the wizard](entra_id_add_tenant_summary.md).
