# Connecting to Remote Microsoft Entra ID Backup Repository

In this article

To use for the Microsoft Entra ID backup repository a remote PostgreSQL instance instead of the local one, use the Veeam Configuration Database Connection Utility.

1. [Launch the Veeam Configuration Database Connection Utility](entra_id_db_utility_launch.md)
2. [Select a product](entra_id_db_utility_product.md)
3. [Configure the connection](entra_id_db_utility_connection.md)
4. [Finish working with the wizard](entra_id_db_utility_summary.md)

Page updated 11/3/2025

Page content applies to build 13.0.1.1071
