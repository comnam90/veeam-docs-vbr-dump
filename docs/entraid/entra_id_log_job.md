# Creating Log Backup Jobs


To create a Microsoft Entra ID log backup job, do the following:

1. [Launch the New Microsoft Entra ID Log Backup Job wizard](entra_id_log_job_launch.md).
2. [Specify a job name and description](entra_id_log_job_name.md).
3. [Specify a tenant](entra_id_log_job_tenant.md).
4. [Specify backup repository settings](entra_id_log_job_repo.md).
5. [Specify secondary repository settings](entra_id_log_job_secondary.md).
6. [Define a job schedule](entra_id_log_job_schedule.md).
7. [Finish working with the wizard](entra_id_log_job_summary.md).
