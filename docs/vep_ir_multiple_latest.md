# Instant Recovery of Latest State


To perform instant recovery of multiple instances to their latest state, use the Instant Recovery wizard.

1. [Launch the Instant Recovery wizard](vep_ir_multiple_latest_wizard.md).
2. [Specify switchover settings](vep_ir_multiple_latest_specify_switchover_settings.md).
3. [Finalize the Instant Recovery session](vep_ir_multiple_latest_finalize_ir_session.md).
