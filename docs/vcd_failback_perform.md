# Performing Failback

In this article

For more information on failback, see [Failover and Failback for Cloud Director](vcd_failover_failback.md) and [Failback](vcd_failback.md).

To perform failback, do the following:

1. [Check prerequisites](byb_failback_vcd.md).
2. [Launch the Failback wizard](vcd_failback_launch.md).
3. [Select vApp replicas to fail back](vcd_vapps_failback_select.md).
4. [Select the failback destination](vcd_failback_destination.md).
5. [Select the target vApp](vcd_target_vapp.md).
6. [Specify the organization VDC](vcd_orgvdc_failback.md).
7. [Specify the storage policy](vcd_failback_storage_policy.md).
8. [Specify the network](vcd_network_failback.md).
9. [Define Failback Mode](vcd_switch_to_production_vapps.md).
10. [Review summary and finish working with the wizard](vcd_failback_finish.md).
