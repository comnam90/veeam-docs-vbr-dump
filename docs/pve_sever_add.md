# Adding Proxmox VE Server to Backup Infrastructure


To add a Proxmox VE server to the backup infrastructure, do the following:

1. [Launch the New Proxmox VE Server wizard](pve_server_add_launch.md).
2. [Specify the Proxmox VE server domain name or IP address](pve_server_add_name.md).
3. [Enter credentials to access the Proxmox VE server](pve_server_add_credentials.md).
4. [Select storage for snapshots](pve_server_add_storage.md).
5. [Apply Proxmox VE server settings](pve_server_add_apply.md).
6. [Finish working with the wizard](pve_server_add_summary.md).
