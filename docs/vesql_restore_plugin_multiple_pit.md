# Restoring Point-in-Time State


To restore databases backed up with Veeam Plug-In for Microsoft SQL Server as of a point-in-time state, use the Restore wizard.

1. [Launch the Restore wizard](vesql_restore_plugin_multiple_pit_wizard.md).
2. [Select databases you want to restore](vesql_restore_plugin_multiple_pit_select_databases.md).
3. [Specify a restore point](vesql_restore_plugin_multiple_pit_specify_restore_point.md).
