# Performing Failback


For more information on failback, see [Failback](failback_hv.md) and [Failover and Failback for Replication](failover_failback_hv.md).

To switch back to the source VM, use the Failback wizard.

1. [Check prerequisites](failback_before_you_begin_hv.md).
2. [Launch the Failback wizard](failback_launch_hv.md).
3. [Select VM replicas to fail back](failback_replica_hv.md).
4. [Select a failback destination](failback_destination_hv.md).
5. [Select a failback target host](failback_host_hv.md).
6. [Select a failback target datastore](failback_datastore_hv.md).
7. [Select a target network](failback_network_hv.md).
8. [Select a failback target folder](failback_name_hv.md).
9. [Map the replica to the restored VM](failback_target_hv.md).
10. [Review summary and finish working with the wizard](failback_summary_hv.md).
