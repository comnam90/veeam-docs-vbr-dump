# Exporting Disks


You can export disks of different workloads from backups and convert disks to the VMDK, VHD or VHDX format. For the list of backups that you can use for export, see [Supported Backup Types](disk_export.md).

To export disks, use the Export Disk wizard.

1. [Launch the Export Disk wizard](disk_export_launch.md).
2. [Select a backup](disk_export_machine.md).
3. [Select a restore point](disk_export_restore_point.md).
4. [Select disks](disk_export_disks.md).
5. [Select the destination and disk format](disk_export_format.md).
6. [Specify secure restore settings](disk_export_av.md).
7. [Specify export reason](disk_export_reason.md).
8. [Finish working with the wizard](disk_export_complete.md).
