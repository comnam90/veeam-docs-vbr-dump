# Restoring to Amazon EC2


To restore workloads to Amazon EC2, use the Restore to Amazon EC2 wizard.

1. [Check prerequisites](restore_amazon_byb.md).
2. [Launch the Restore to Amazon EC2 wizard](restore_amazon_launch.md).
3. [Select a workload and restore point](restore_amazon_vm.md).
4. [Specify credentials and region settings](restore_amazon_account.md).
5. [Specify an instance name](restore_amazon_name.md).
6. [Specify an instance type](restore_amazon_instance.md).
7. [Configure network settings](restore_amazon_network.md).
8. [Specify secure restore settings](restore_amazon_av.md).
9. [Specify a helper appliance](restore_amazon_proxy.md).
10. [Specify a restore reason](restore_amazon_reason.md).
11. [Start the restore process](restore_amazon_start.md).
