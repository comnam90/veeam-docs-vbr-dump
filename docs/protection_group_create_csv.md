# Creating Protection Group for Computers from CSV File

In this article

Before you create a protection group for computers from a CSV file, [check prerequisites](protection_group_before_csv.md). Then use the New Protection Group wizard to configure a protection group.

1. [Launch the New Protection Group wizard](protection_group_launch_csv.md).
2. [Specify protection group name and description](protection_group_name_csv.md).
3. [Specify CSV file](protection_group_scope_csv.md).
4. [Specify credentials](protection_group_accounts_csv.md).
5. [Specify discovery and deployment options](protection_group_options_csv.md).
6. [Specify advanced protection group settings](protection_group_advanced_csv.md).
7. [Review components](protection_group_review_csv.md).
8. [Assess results](protection_group_apply_csv.md).
9. [Finish working with the wizard](protection_group_finish_csv.md).
