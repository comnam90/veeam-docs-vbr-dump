# Publishing to Specified Server

In this article

To publish a PostgreSQL instance to the specified PostgreSQL server, use the Publish wizard.

1. [Launch the Publish wizard](vep_publish_launch.md).
2. [Specify a restore point](vep_publish_restore_point.md).
3. [Specify the target PostgreSQL server](vep_publish_target_server.md).
4. [Specify PostgreSQL settings](vep_publish_postgresql_settings.md).
