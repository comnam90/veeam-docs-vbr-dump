# Restoring Point-in-Time State


To restore a point-in-time state of multiple SAP HANA tenant databases, use the Restore wizard.

1. [Launch the Restore wizard](vehana_restore_multiple_pit_wizard.md).
2. [Select databases to restore](vehana_restore_multiple_pit_select_databases.md).
3. [Specify server credentials](vehana_restore_multiple_pit_specify_server_credentials.md).
4. [Specify system credentials](vehana_restore_multiple_pit_specify_system_credentials.md).
5. [Specify a point in time](vehana_restore_multiple_pit_specify_pit.md).
6. [Specify SYSTEM user password](vehana_restore_multiple_pit_specify_password.md).
