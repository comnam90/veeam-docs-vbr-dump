# Creating File to Tape Jobs


To write files to tape, you should create a file to tape job using the New File to Tape Job wizard. This section will guide you through all steps of the wizard and provide explanation on available options.

With this wizard, you can create a file to tape job or a GFS file to tape job.

To create a file to tape job, follow the next steps:

1. [Launch the New File to Tape Job wizard](files_to_tape_launch.md).
2. [Specify a job name and description](files_to_tape_name.md).
3. [Choose files and folders to archive](files_to_tape_files.md).
4. [Choose a media pool for full backup](files_to_tape_pool.md).
5. [Specify a media pool for incremental backups](files_to_tape_pool_increments.md).
6. [Specify archiving options](files_to_tape_options.md).
7. [Specify advanced job settings](files_to_tape_advanced.md).
8. [Define a job schedule](gfs_file_to_tape_schedule.md).
9. [Finish working with the wizard](files_to_tape_review.md).
