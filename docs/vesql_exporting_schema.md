# Exporting to Custom Location

In this article

To export database schema and data to a custom location, do the following:

1. [Launch the Export wizard](vesql_exporting_schema_1.md).
2. [Specify a restore point](vesql_exporting_schema_2.md).
3. [Fine-tune the restore point](vesql_exporting_schema_3.md).
4. [Select database objects](vesql_exporting_schema_4.md).
5. [Specify names for objects](vesql_exporting_schema_5.md).
6. [Specify export location](vesql_exporting_schema_6.md).
7. [Review the export summary](vesql_export_schema_tal_summary.md).
