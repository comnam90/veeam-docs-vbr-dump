# Instant Recovery of Latest State

In this article

To perform instant recovery of multiple Oracle databases to their latest state on the backup file, perform the following steps:

1. [Launch the Instant Recovery wizard](veor_ir_multiple_latest_wizard.md).
2. [Specify switchover settings](veor_ir_multiple_latest_specify_switchover_settings.md).
3. [Finalize the Instant Recovery session](veor_ir_multiple_latest_finalize_ir_session.md).
