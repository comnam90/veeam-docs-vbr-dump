# Step 5. Specify Home User Password

In this article

This step of the wizard is available if you restore a database to a Windows-based Oracle server and the account specified in [Specify Target Server](rman_target.md) is not the Oracle home user.

At this step of the wizard, enter the password for the Oracle home user.
