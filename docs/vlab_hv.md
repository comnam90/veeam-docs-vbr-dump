# Creating Virtual Lab


Before you create a new virtual lab, [check prerequisites](vlab_before_you_begin_hv.md). Then use the New Virtual Lab wizard to create a virtual lab.

1. [Launch the New Virtual Lab wizard](vlab_launch_hv.md).
2. [Specify a name and description for the virtual lab](vlab_name_hv.md).
3. [Select a host](vlab_host_hv.md).
4. [Set up a proxy appliance](vlab_proxy_hv.md).
5. [Select the networking mode](vlab_network_hv.md).
6. [Select isolated networks](vlab_isolated_network_hv.md).
7. [Specify network settings](vlab_net_settings_hv.md).
8. [Specify static IP mapping rules](vlab_ip_hv.md).
9. [Apply parameters](vlab_apply_hv.md).
