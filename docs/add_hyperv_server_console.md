# Adding Microsoft Hyper‑V Servers Using Console

In this article

This section describes how to add a Microsoft Hyper-V server as a backup infrastructure component.

Before you add a Microsoft Hyper-V server using Veeam Backup & Replication console, [check prerequisites](hv_server_before_begin.md). Then use the New Hyper-V Server wizard to add the server.

1. [Launch the New Hyper-V Server wizard](hv_server_launch.md).
2. [Specify a server name or address](hv_server_name.md).
3. [Specify credentials](hv_server_credentials.md).
4. [Select hosts](hv_server_hosts.md).
5. [Review components](hv_server_apply.md).
6. [Finish working with the wizard](hv_server_results.md).
7. [Specify settings for connected volumes](hv_server_volumes.md).
