# Instant Recovery to Another Server


To perform instant recovery of a database to another server, use the Instant Recovery wizard.

1. [Launch the Instant Recovery wizard](vesql_instant_recovery_launch.md).
2. [Specify a restore point](vesql_instant_restore_point.md).
3. [Fine-tune the restore point](vesql_instant_point_define.md).
4. [Specify the target server](vesql_instant_target.md).
5. [Specify database files location](vesql_instant_files_location.md).
6. [Specify switchover settings](vesql_instant_switchover.md).
7. [Finalize the Instant Recovery session](vesql_instant_finish.md).
