# Before You Begin

In this article

Before you create a virtual lab, check the following prerequisites:

* A valid license for Enterprise edition of Veeam Backup & Replication must be installed on the backup server.
* You can configure a virtual lab on the following types of hosts:

+ Microsoft Windows Server 2022 with the Hyper-V role enabled
+ Microsoft Windows Server 2019 with the Hyper-V role enabled
+ Microsoft Hyper-V Server 2019
+ Microsoft Windows Server 2016 with the Hyper-V role enabled
+ Microsoft Hyper-V Server 2016
+ Microsoft Windows Server 2012 R2 with the Hyper-V role enabled
+ Microsoft Hyper-V Server 2012 R2
+ Microsoft Windows Server 2012 with the Hyper-V role enabled
+ Microsoft Hyper-V Server 2012
+ Microsoft Windows 10 with the Hyper-V role enabled
+ Microsoft Windows 11 with the Hyper-V role enabled
