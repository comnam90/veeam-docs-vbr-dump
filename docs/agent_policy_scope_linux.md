# Step 6. Specify Backup Scope Settings

In this article

The Objects step of the wizard is available if you chose to create volume-level or file-level Veeam Agent backups. Specify backup scope for the Veeam Agent backup policy:

* [Specify volumes to back up](agent_policy_volumes_linux.md) — if you have selected the Volume level backup option at the [Backup Mode](agent_policy_mode_linux.md) step of the wizard.
* [Specify directories to back up](agent_policy_folders_linux.md) — if you have selected the File level backup option at the [Backup Mode](agent_policy_mode_linux.md) step of the wizard.

Page updated 6/19/2024
