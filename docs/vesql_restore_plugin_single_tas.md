# Restoring to Another Server

In this article

To restore a single Microsoft SQL Server database to another server, use the Restore wizard.

1. [Launch the Restore wizard](vesql_restore_plugin_single_tas_wizard.md).
2. [Specify a restore point](vesql_restore_plugin_single_tas_specify_restore_point.md).
3. [Specify a point-in-time state](vesql_restore_plugin_single_tas_specify_pit.md).
4. [Specify the target server](vesql_restore_plugin_single_tas_specify_target_server.md).
5. [Specify files location](vesql_restore_plugin_single_tas_specify_files_location.md).
6. [Specify the recovery state](vesql_restore_plugin_single_tas_specify_recovery_state.md).
