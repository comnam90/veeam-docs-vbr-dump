# Creating Failover Plans


To create a failover plan, use the New Failover Plan wizard.

1. [Check prerequisites](uni_cdp_failover_plan_before_you_begin.md).
2. [Launch the New Failover Plan wizard](uni_cdp_creating_failover_plan_launch.md).
3. [Specify a failover plan name and description](uni_cdp_creating_failover_plans_name.md).
4. [Select VMs](uni_cdp_creating_failover_plan_vms.md).
5. [Set time delay](uni_cdp_creating_failover_plan_delay_time.md).
6. [Review the summary and finish working with the wizard](uni_cdp_creating_failover_plan_summary.md).
