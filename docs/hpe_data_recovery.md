# Performing Restore


In various disaster recovery scenarios, Veeam Backup & Replication allows you to perform the following operations using backed-up data:

* [Entire VM restore](hpe_restore_entire_vm.md) — recover HPE Morpheus VM Essentials VMs to the original location or to a new location.

* [Instant VM recovery](hpe_restore_instant.md) — instantly start an HPE Morpheus VM Essentials VM directly from a backup.
* [Disk publishing](hpe_disk_publish.md) — mount specific disks of a backed-up HPE Morpheus VM Essentials VMs to any server added to the backup infrastructure.
* [File-level restore](hpe_vm_guest_restore.md) — recover individual VM guest OS files and folders.

* [Application items restore](hpe_restore_app_items.md) — restore applications, such as Microsoft Active Directory, Microsoft Exchange, Microsoft SharePoint, and Microsoft SQL Server.

* [VM disk export](hpe_disk_export.md) — restore VM disks and convert them to disks of the VMDK, VHD or VHDX format.
* [VM restore to Amazon Web Services](hpe_restore_to_amazon_ec2.md) — restore HPE Morpheus VM Essentials VMs to Amazon Web Services as EC2 instances.
* [VM restore to Microsoft Azure](hpe_restore_to_microsoft_azure.md) — restore HPE Morpheus VM Essentials VMs to Microsoft Azure as Azure VMs.
* [VM restore to Google Cloud](hpe_restore_to_google.md) — restore HPE Morpheus VM Essentials VMs to Google Cloud as VM instances.

You can restore VM data to the most recent state or to any available restore point.
