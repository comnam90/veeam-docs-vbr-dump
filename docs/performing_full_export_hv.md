# Performing Export

In this article

To export data, do the following:

1. [Launch the Export Backup wizard](launching_new_export_wizard_hv.md).
2. [Select restore points to export](specify_restore_points_to_export_hv.md).
3. [Specify a destination](specify_export_target_hv.md).
4. [Specify an export reason](specify_export_reason_hv.md).
5. [Finish working with the wizard](finishing_working_with_export_wizard_hv.md).
