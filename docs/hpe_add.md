# Adding Hewlett Packard Enterprise


In this section, you will learn how to add HPE Nimble and HPE Alletra 5000, HPE Alletra 6000, HPE 3PAR StoreServ, HPE Primera, HPE Alletra 9000 and HPE Alletra Storage MP B10000 built-in storage systems.

To learn how to add HPE XP Universal Storage API integrated systems, see [Universal Storage API Integrated Systems](universal_storage_integration_api.md).
