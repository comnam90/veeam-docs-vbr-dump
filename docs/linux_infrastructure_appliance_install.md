# Installing Veeam Infrastructure Appliance with ISO

In this article

Before you configure a backup infrastructure component with Veeam Infrastructure Appliance, check the [prerequisites](linux_infrastructure_appliance_byb.md). Then, perform the following steps:

1. [Mount the ISO image](jeos_iso_install_mount.md).
2. [Select the product](jeos_install_select_product.md).
3. [Begin installation](jeos_install_type.md).
4. [Read and accept license agreements](jeos_install_license.md).
5. [Specify the hostname](jeos_install_hostname.md).
6. [Review network settings](jeos_install_network.md).
7. [Review server time settings](jeos_install_time.md).
8. [Configure the default host administrator account](jeos_install_host_administrator.md).
9. [Configure the default security officer account](jeos_install_security_officer.md).
10. [Finish the configuration](jeos_install_summary.md).
