# Performing Disk Restore


In case a disaster strikes, you can restore a disk of an oVirt VM from a backup. Veeam Plug-in for oVirt KVM allows you to attach the restored disk to the original VM or any other oVirt VM in the oVirt KVM environment. To restore a disk attached to a protected VM, do the following:

1. [Launch the Virtual Disk Restore wizard](ovirt_restore_disks_launch.md).
2. [Select a VM](ovirt_restore_disks_select_disks.md).
3. [Select a restore point](ovirt_restore_disks_restore_point.md).
4. [Configure mapping settings](ovirt_restore_disks_mapping.md).
5. [Specify a reason for the restore](ovirt_restore_disks_reason.md).
6. [Finish working with the wizard](ovirt_restore_disks_summary.md).
