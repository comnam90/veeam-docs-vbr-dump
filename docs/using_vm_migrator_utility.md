# Using Veeam VM Migrator Utility


Each procedure step includes running utility cmdlets in Veeam Backup PowerShell module.

To use Veeam VM Migrator utility, check the [prerequisites](vm_migrator_utility_byb.md) and start Veeam PowerShell session.

1. [Start Veeam PowerShell session](vm_migrator_powershell_session.md).
2. [Optional] [Update existing VMs BIOS UUIDs](vm_migrator_uuid_update.md).
3. [Optional] [Modify old vCenter name](vm_migrator_modify_vcenter_name.md).
4. [Optional] [Add new vCenter](vm_migrator_add_vcenter.md).
5. [Generate migration task file](vm_migrator_task_generate.md).
6. [Review migration task file](vm_migrator_task_review.md).
7. [Start the migration](vm_migrator_start_migration.md).
8. [Finish the migration](vm_migrator_finish_migration.md).
