# Before You Begin


Before you install an update for Veeam Backup & Replication 13.0.1, check the following prerequisites:

* Make sure you have Veeam Backup & Replication 13.0.1 (build 13.0.1.180) or later installed.

For information on how to upgrade from product version 12.3.1 (build 12.3.1.1139) or later, see [Upgrading to Veeam Backup & Replication 13 on Windows](upgrade_vbr.md).

* Review the list of required [Trusted Root Certificates](trusted_root_certificates.md), and ensure all are installed.

* If you use Veeam Backup Enterprise Manager, start the update procedure with this component. For details, see [Updating Veeam Backup Enterprise Manager](https://helpcenter.veeam.com/docs/vbr/em/em_updating.html?ver=13). Veeam Backup & Replication should be updated after that.
* Stop all restore processes in the Veeam Backup & Replication Console.
* Stop all jobs and disable them in the Veeam Backup & Replication Console.
* Close the Veeam Backup & Replication Console for all users.
