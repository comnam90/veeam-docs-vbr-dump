# Creating Job for Windows Computers


To back up data of a computer protected with Veeam Agent for Microsoft Windows, you can configure a Veeam Agent backup job in Veeam Backup & Replication.

Before configuring a Veeam Agent backup job, [check prerequisites](agent_job_before.md). Then use the New Agent Backup Job wizard to define settings for the backup job.

1. [Launch the New Agent Backup Job wizard](agent_job_launch.md).
2. [Select the type of protected computers and the job mode](agent_job_protection_mode.md).
3. [Specify job name and description](agent_job_name.md).
4. [Select computers to back up](agent_job_scope.md).
5. [Select backup mode](agent_job_mode.md).
6. [Specify backup scope](agent_job_backup_scope.md).
7. [Specify backup storage settings](agent_job_target_settings.md).
8. [Specify advanced backup settings](agent_job_advanced.md).
9. [Specify secondary backup target](agent_job_secondary_target.md).
10. [Specify guest processing settings](agent_job_vss.md).
11. [Specify the backup schedule](agent_job_schedule.md).
12. [Review backup job settings](agent_job_review.md).
