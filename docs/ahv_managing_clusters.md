# Connecting Nutanix AHV Server


The Nutanix AHV server (a Prism Central or standalone cluster) allows the backup server to access Nutanix AHV resources such as VMs, storage containers and networks. After you add the Nutanix AHV server to the backup infrastructure, you will be able to deploy workers and to manage data protection tasks for Nutanix AHV VMs and protection domains.
