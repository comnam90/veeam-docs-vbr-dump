# Adding Dell


In this section, you will learn how to add Dell PowerScale (formerly Isilon) and Dell Unity, Unity XT built-in storage systems.

To learn how to add Dell PowerMax, Dell PowerStore or Dell SC series Universal Storage API integrated systems, see [Universal Storage API Integrated Systems](universal_storage_integration_api.md).
