# Recovering Guest OS Files Using Console


To recover guest OS files and folders, use the File Level Restore wizard.

1. [Launch the wizard](guest_restore_launch_vm.md).
2. [Select a workload](guest_restore_vms_vm.md).
3. [Select a restore point](guest_restore_point_vm.md).
4. [Specify the mount server](multios_restore_host_vm.md).
5. [Specify a location for helper appliance](multios_restore_proxy_vm.md).
6. [Configure the staging host](guest_restore_storage_location.md).
7. [Specify a restore reason](guest_restore_reason_vm.md).
8. [Verify recovery settings](guest_restore_review_vm.md).
9. [Finalize recovery](guest_restore_save.md).
