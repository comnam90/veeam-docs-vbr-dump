# Restore from Veeam Backup for Microsoft 365 Backups

In this article

You can restore Microsoft SharePoint data from backups created by Veeam Backup for Microsoft 365 to the original location or other [Microsoft 365 organizations](https://helpcenter.veeam.com/docs/vbo365/guide/vbo_add_office365_org.html?ver=80) and [on-premises Microsoft organizations](https://helpcenter.veeam.com/docs/vbo365/guide/vbo_add_onpremises_org.html?ver=80).

Consider the following:

* If your Veeam Explorer for Microsoft SharePoint comes as part of Veeam Backup for Microsoft 365, the backup content is loaded automatically when you open Veeam Explorer for Microsoft SharePoint.
* If your Veeam Explorer for Microsoft SharePoint comes as part of Veeam Backup & Replication, add a Veeam Backup for Microsoft 365 server or a Veeam Backup for Microsoft 365 backup file. For more information, see [Adding Veeam Backup for Microsoft 365 Server](vesp_adding_vbo_server.md) and [Adding Veeam Backup for Microsoft 365 Databases](vesp_adding_vbo_databases.md).

Before restoring data, read the [Considerations and Limitations](vesp_recovery_specials.md) section.
