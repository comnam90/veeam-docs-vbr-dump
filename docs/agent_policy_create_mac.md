# Creating Policy for Mac Computers

In this article

To back up data of a computer protected with Veeam Agent for Mac, you must configure a Veeam Agent backup policy in Veeam Backup & Replication. This backup policy will be applied to Veeam Agent computers to create individual backup jobs. Using these jobs, Veeam Agents will perform backup operations.

Before configuring a backup policy, [check prerequisites](agent_policy_before_mac.md). Then use the New Agent Backup Job wizard to define settings for the backup policy.

1. [Launch the New Agent Backup Job wizard](agent_policy_launch_mac.md).
2. [Select the type of protected computers](agent_policy_protection_mac.md).
3. [Specify policy name and description](agent_policy_name_mac.md).
4. [Select computers to back up](agent_policy_comp_mac.md).
5. [Select backup mode](agent_policy_mode_mac.md).
6. [Specify backup scope](agent_policy_scope_mac.md).
7. [Select backup destination](agent_policy_target_mac.md).
8. [Specify backup storage settings](agent_policy_target_set_mac.md).
9. [Specify advanced backup settings](agent_policy_advanced_mac.md).
10. [Specify secondary backup target](agent_policy_secondary_target_mac.md).
11. [Specify guest processing settings](agent_policy_guest_mac.md).
12. [Specify the backup schedule](agent_policy_schedule_mac.md).
13. [Review backup policy settings](agent_policy_review_mac.md).
