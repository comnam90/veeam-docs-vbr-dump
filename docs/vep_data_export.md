# Data Export


This section explains how to export PostgreSQL data using Veeam Explorer for PostgreSQL.

|  |
| --- |
| Note |
| If you are using a Linux-based backup server, you can only export PostgreSQL data to a Linux machine. |

Before exporting data, read the [Considerations and Limitations](vep_considerations.md) section.
