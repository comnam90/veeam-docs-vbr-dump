# Uninstalling Veeam Plug-In for Microsoft SQL Server


To uninstall Veeam Plug-In for Microsoft SQL Server, do the following:

1. On Microsoft SQL Server with Veeam Plug-In, open the Control Panel and click Programs and Features.
2. In the list of programs, select Veeam Plug-In for Microsoft SQL Server and click Uninstall.
