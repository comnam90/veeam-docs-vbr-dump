# Data Copy

In this article

You can use Veeam Explorer for Microsoft OneDrive for Business to copy OneDrive data to a different location.

To copy OneDrive data to the same or different user, do the following:

1. [Launch the Restore wizard](veod_launch_copy_wizard.md).
2. [Select an authentication method](veod_select_authentication_method_copy.md).
3. [Log in to Microsoft 365](veod_login_copy.md).
4. [Specify a target user](veod_copy_target_user.md).
5. [Specify a target folder](veod_copy_target_folder.md).
6. [Specify restore options](veod_copy_restore_options.md).
7. [Review the operation summary](summary23.md).
