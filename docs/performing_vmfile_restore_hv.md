# Restoring VM Files


To restore VM files from a backup, use the Hyper-V Restore  wizard.

1. [Check prerequisites](vmfile_restore_before_you_begin.md).
2. [Launch the Restore wizard](vmfile_restore_launch_hv.md).
3. [Select a VM](vmfile_restore_vms_hv.md).
4. [Select a restore point](vmfile_restore_point_hv.md).
5. [Select VM files and destination](vmfile_restore_files_hv.md).
6. [Specify a restore reason](vmfile_restore_reason_hv.md).
7. [Finish working with the wizard](vmfile_restore_summary_hv.md).
