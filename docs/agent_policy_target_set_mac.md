# Step 8. Specify Backup Storage Settings


Specify backup storage settings for the backup policy at one of the following steps of the wizard:

* [Local storage settings](agent_policy_mac_drive.md) — if you have selected the Local storage option at the [Destination](agent_policy_target_mac.md) step of the wizard.
* [Shared folder settings](agent_policy_mac_share.md) — if you have selected the Shared folder option at the [Destination](agent_policy_target_mac.md) step of the wizard.
* [Veeam backup repository settings](agent_policy_mac_vbr.md) — if you have selected the Veeam backup repository option at the [Destination](agent_policy_target_mac.md) step of the wizard.
* [Cloud repository settings](agent_policy_mac_cloud.md) — if you have selected the Veeam Cloud Connect repository option at the [Destination](agent_policy_target_mac.md) step of the wizard.
