# Configuring Backup Settings

In this article

To back up Microsoft SQL Server data with Veeam Plug-In, you must configure backup settings. You can select what databases to back up, choose the backup type, specify data retention and data processing settings.

Before configuring backup settings, [check prerequisites](planning_and_preparation_mssql.md). Then use the Backup Up Database wizard to define backup settings.

1. [Launch the Back Up Database wizard](backup_job_launch.md).
2. [Select databases to back up and specify the backup type](backup_job_scope.md).
3. [Specify data retention and data processing options](backup_job_options.md).
4. [Monitor the backup process](backup_job_process.md).
