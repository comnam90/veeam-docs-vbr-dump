# Creating Backup Jobs


To create a backup job, do the following:

1. [Check prerequisites and limitations](pve_backup_job_create_prerequisites.md).
2. [Launch the New Backup Job wizard](pve_backup_job_create_launch.md).
3. [Specify a job name and description](pve_backup_job_create_general_settings.md).
4. [Select VMs to back up](pve_backup_job_create_assign_vms.md).
5. [Specify a backup repository and configure backup settings](pve_backup_job_create_destination.md).
6. [Enable guest processing](pve_backup_job_create_guest_processing.md).
7. [Create a schedule for the backup job](pve_backup_job_create_schedule.md).
8. [Finish working with the wizard](pve_backup_job_create_summary.md).
