# Overview


Starting from version 12.2, Veeam Backup & Replication comes with a plug-in that extends the solution functionality and allows you to add Microsoft Entra ID tenants to the backup infrastructure, as well as to manage data protection and recovery operations for Microsoft Entra ID data from a single console.
