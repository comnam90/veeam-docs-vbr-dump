# Deployment


Starting from version 12.3.2, the Veeam Backup & Replication solution allows you to add Scale Computing HyperCore servers to the backup infrastructure, and to manage data protection and recovery operations for Scale Computing HyperCore workloads from a single console.

To access the Veeam Plug-in for Scale Computing HyperCore v2 functionality, you can either deploy a new backup server as described in the [Deployment](deployment.md) section or use a backup server that already exists in your backup infrastructure if it meets the [Veeam Plug-in for Scale Computing HyperCore system requirements](platform_support_sch.md).

To install Veeam Plug-in for Scale Computing HyperCore v3:

* For Windows-based servers, download the installation file and follow instructions in section [Installing Veeam Plug-in for Scale Computing HyperCore Manually](sch_install_plugin.md).
* For Linux-based servers, install new plug-in version as described in section [Installing Updates](update_appliance_install_updates.md).
