# Creating Failover Plans

In this article

To create a failover plan, use the New Failover Plan wizard.

1. [Check prerequisites](failover_plan_before_you_begin.md).
2. [Launch the New Failover Plan wizard](creating_failover_plan_launch.md).
3. [Specify a failover plan name and description](creating_failover_plans_name.md).
4. [Select VMs](creating_failover_plan_vms.md).
5. [Set time delay](creating_failover_plan_delay_time.md).
6. [Review the summary and finish working with the wizard](creating_failover_plan_summary.md).
