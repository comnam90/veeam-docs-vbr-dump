In this article

The example below describes the steps you should follow to get a list of backup servers connected to Veeam Backup Enterprise Manager. The example will help you start working with Veeam Backup Enterprise Manager REST API and perform basic operations such as logon, retrieving a resource representation and so on.

To follow this exercise, you will perform the next steps:

1. [Access Veeam Backup Enterprise Manager REST API](accessing_em_web_api.md)
2. [Perform logon](logging_on.md)
3. [Get a list of backup servers](getting_a_list_of_veeam_backup_servers.md)
4. [Perform logout](logging_out.md)
