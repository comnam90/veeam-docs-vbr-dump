This Veeam Backup Enterprise Manager REST API reference contains a description of resources and their collections which stand for Veeam Backup & Replication and Veeam Backup Enterprise Manager entities. Every resource and collection has an XML and JSON data representation and one or more methods that represent operations available for the resource or collection.
