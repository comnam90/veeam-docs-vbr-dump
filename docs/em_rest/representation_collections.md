In this article

For collections of key resources, Veeam Backup Enterprise Manager REST API provides the same two types of representations as for specific entities of [key resources](resource_representation_for_ke.md):

* Entity reference resource representation
* Entity resource representation

|  |
| --- |
| GET  https://localhost:9398/api/cloud/tenants?format=Entity |
