In this article

Every request to the server returns a response. The response returned from the server includes:

* [HTTP response code](http_response_codes.md) that indicates the status of the operation, for example: 200 OK
* [HTTP response header](em_web_api_specifications.md)
* HTTP response body (if applicable to the operation)
