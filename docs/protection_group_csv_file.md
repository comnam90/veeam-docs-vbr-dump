# Creating Protection Group for Computers from CSV File


Before you create a protection group for computers from a CSV file, [check prerequisites](pg_csv_file_before.md). Then use the New Protection Group wizard to configure a protection group.

1. [Launch the New Protection Group wizard](pg_csv_file_launch.md).
2. [Specify protection group name and description](pg_csv_file_name.md).
3. [Specify CSV file](protection_group_csv.md).
4. [Specify credentials](pg_csv_file_credentials.md).
5. [Specify discovery and deployment options](pg_csv_file_options.md).
6. [Specify advanced protection group settings](pg_csv_file_advanced.md).
7. [Review components](pg_csv_file_review.md).
8. [Assess results](pg_csv_file_results.md).
9. [Finish working with the wizard](pg_csv_file_finish.md).
