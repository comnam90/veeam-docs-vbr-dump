# Installing Veeam Agent for Linux from Generated Setup Files


To install Veeam Agent for Linux on a computer added to a protection group for pre-installed Veeam Agents, do the following:

1. Upload the setup files generated by Veeam Backup & Replication to the computer you want to protect — for example, to a directory on a local drive or USB drive, or to a network shared folder.
2. On the computer you want to protect, navigate to the directory where you have saved the setup files and install Veeam Agent:

* [Installing Veeam Agent for Linux in RHEL / Rocky Linux / AlmaLinux](#rhel)
* [Installing Veeam Agent for Linux in Oracle Linux](#oracle)
* [Installing Veeam Agent for Linux in SLES](#sles)
* [Installing Veeam Agent for Linux in Debian / Ubuntu](#deb)

Installing Veeam Agent for Linux in RHEL / Rocky Linux / AlmaLinux

To install Veeam Agent for Linux, use the following commands:

RHEL 8 / Rocky Linux 8.10 / AlmaLinux 8.10

|  |
| --- |
| rpm -i <...>/kmod-veeamsnap-13.0.1.404-1.el8.x86\_64.rpm  rpm -i <...>/veeam-libs-13.0.1.404-1.x86\_64.rpm |

RHEL 9 / Rocky Linux 9.4 – 9.7 / AlmaLinux 9.4 – 9.7

|  |
| --- |
| rpm -i <...>/kmod-blksnap-13.0.1.404-1.el9.x86\_64.rpm  rpm -i <...>/veeam-libs-13.0.1.404-1.x86\_64.rpm |

RHEL 10 / Rocky Linux 10 / AlmaLinux 10

|  |
| --- |
| rpm -i <...>/kmod-blksnap-13.0.1.404-1.el10.x86\_64.rpm  rpm -i <...>/veeam-libs-13.0.1.404-1.x86\_64.rpm rpm -i <...>/veeam-13.0.1.404-1.el10.x86\_64.rpm |

where <...> is the path to the directory where you have saved the Veeam Agent packages.

Installing Veeam Agent for Linux in Debian / Ubuntu

To install Veeam Agent for Linux, use the following commands:

For Debian 10 / Ubuntu 16.04, 18.04, 20.04 (kernel 5.4)

|  |
| --- |
| apt-get install <...>/veeamsnap\_13.0.1.404\_all.deb  apt-get install <...>/veeam-libs\_13.0.1.404\_amd64.deb apt-get install <...>/veeam\_13.0.1.404\_amd64.deb |

For Debian 11 – 13 / Ubuntu 22.04 and 24.04

|  |
| --- |
| apt-get install <...>/blksnap\_13.0.1.404\_all.deb  apt-get install <...>/veeam-libs\_13.0.1.404\_amd64.deb apt-get install <...>/veeam\_13.0.1.404\_amd64.deb |

where <...> is the path to the directory where you have saved the Veeam Agent packages.
