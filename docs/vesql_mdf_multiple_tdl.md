# Exporting to Different Location

In this article

To export multiple Microsoft SQL Server databases to a custom location, use the Export wizard.

1. [Launch the Export wizard](vesql_mdf_multiple_tdl_wizard.md).
2. [Select databases you want to export](vesql_mdf_multiple_tdl_select_databases.md).
3. [Specify a restore point](vesql_mdf_multiple_tdl_restore_point.md).
4. [Specify export location](vesql_mdf_multiple_tdl_files.md).
5. [Review the export summary](vesql_export_multiple_tal_summary.md).
