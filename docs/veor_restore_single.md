# Restoring Single Database

In this article

With Veeam Explorer for Oracle, you can restore a standalone database or a Data Guard database from image-level backups created by Veeam Backup & Replication.

To restore an Oracle database, follow one of these scenarios:

* [Restoring Latest State](veor_restore_single_latest.md)
* [Restoring Point-in-Time State](veor_restore_single_pit.md)
* [Restoring to Another Server](veor_restore_single_tas.md)
