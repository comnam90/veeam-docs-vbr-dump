# Creating Backup Jobs


To create a backup job, do the following:

1. [Check prerequisites and limitations](ovirt_backup_job_prerequisites.md).
2. [Launch the New Backup Job wizard](ovirt_backup_job_launch_wizard.md).
3. [Specify a job name and description](ovirt_backup_job_general_settings.md).
4. [Selects VMs to backup](ovirt_backup_job_assign_vms.md).
5. [Specify a backup repository where backups will be stored and configure backup settings](ovirt_backup_job_destination.md).
6. [Create a schedule for the backup job](ovirt_backup_job_schedule.md).
7. [Finish working with the wizard](ovirt_backup_job_review.md).
