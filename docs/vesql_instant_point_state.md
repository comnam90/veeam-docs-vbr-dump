# Instant Recovery of Point-in-Time State

In this article

To perform instant recovery of a database to a selected point-in-time state, use the Instant Recovery wizard.

1. [Launch the Instant Recovery wizard](vesql_instant_point_launch.md).
2. [Specify a restore point](vesql_instant_point_select.md).
3. [Fine-tune the restore point](vesql_instant_point_tune.md).
4. [Specify switchover settings](vesql_instant_point_switchover.md).
5. [Finalize the Instant Recovery session](vesql_instant_point_finish.md).
