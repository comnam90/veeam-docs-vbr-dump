# Performing Instant Disk Recovery

In this article

To recover disks using Instant Disk Recovery, use the Instant Disk Recovery wizard.

1. [Check prerequisites](instant_disk_recovery_before_you_begin.md).
2. [Launch the Instant Disk Recovery wizard](instant_disk_recovery_launch.md).
3. [Select a source VM](instant_disk_recovery_vm.md).
4. [Select a restore point](instant_disk_recovery_restore_point.md).
5. [Select a virtual disk to restore](instant_disk_recovery_map.md).
6. [Specify secure restore settings](instant_disk_recovery_secure_restore.md).
7. [Specify a restore reason](instant_disk_recovery_reason.md).
8. [Verify recovery settings](instant_disk_recovery_summary.md).
