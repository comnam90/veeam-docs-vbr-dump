# Adding Hardened Repositories Using Console

In this article

This section describes how to add a hardened repository.

To add a hardened repository using the Veeam Backup & Replication console, use the New Backup Repository wizard.

1. [Launch the new backup repository wizard](hardened_repository_add_console_launch.md).
2. [Specify hardened repository name and description](hardened_repository_add_console_name.md).
3. [Specify server settings](hardened_repository_add_console_server.md).
4. [Configure hardened repository settings](hardened_repository_add_console_repository.md).
5. [Specify mount server settings](hardened_repository_add_console_mount.md).
6. [Review properties and components](hardened_repository_add_console_review.md).
7. [Apply hardened repository settings](hardened_repository_add_console_apply.md).
8. [Finish working with wizard](hardened_repository_add_console_finish.md).
