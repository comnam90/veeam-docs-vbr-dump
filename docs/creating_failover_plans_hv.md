# Creating Failover Plans


To create a failover plan, use the New Failover Plan wizard.

1. [Check prerequisites](failover_plan_before_you_begin_hv.md).
2. [Launch the New Failover Plan wizard](creating_failover_plan_launch_hv.md).
3. [Specify a failover plan name and description](creating_failover_plans_name_hv.md).
4. [Select VMs](creating_failover_plan_vms_hv.md).
5. [Set time delay](creating_failover_plan_delay_time_hv.md).
6. [Review the summary and finish working with the wizard](creating_failover_plan_summary_hv.md).
