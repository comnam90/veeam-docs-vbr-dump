# Adding CDP Proxies

In this article

You must add to the backup infrastructure CDP proxies that you plan to use for continuous data protection of VMs.

To add a CDP proxy, use the New CDP Proxy wizard.

1. [Launch the New VMware Proxy wizard](cdp_proxy_launch.md).
2. [Choose a server and traffic ports](cdp_proxy_server.md).
3. [Configure cache](cdp_proxy_cache.md).
4. [Configure traffic rules](cdp_proxy_traffic.md).
5. [Review settings and install components](cdp_proxy_install_components.md).
6. [Wait for the CDP proxy to be added to the infrastructure](cdp_proxy_apply.md).
7. [Finish working with the wizard](cdp_proxy_finish.md).
