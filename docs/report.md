# Reporting

In this article

You can view real-time statistics for rescan jobs, application backup policies and restore operations configured in Veeam Backup & Replication. You can generate reports with statistics data only for performed rescan job or backup job sessions. You can generate reports manually in the Veeam Backup & Replication console or set up Veeam Backup & Replication to send reports automatically by email.
