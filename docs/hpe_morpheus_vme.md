# HPE VM Morpheus Essentials


Veeam Backup & Replication offers various data protection and disaster recovery features for HPE Morpheus VM Essentials environments. Specifically, you can perform the following operations:

* Create backups of HPE Morpheus VM Essentials VMs and store them in backup repositories.

* Create VeeamZIP backups of HPE Morpheus VM Essentials VMs.

* Create several instances (copies) of the same backup data in different locations.
* Restore VMs from HPE Morpheus VM Essentials VM backups to the HPE Morpheus VM Essentials environment.

* Restore VMs from VMware ESXi and Microsoft Hyper-V to the HPE Morpheus VM Essentials environment.
* Restore VMs from Nutanix AHV, Proxmox VE, Scale Computing HyperCore and oVirt KVM backups to the HPE Morpheus VM Essentials environment.

* Restore VMs from Microsoft Azure, Amazon Web Services (AWS) and Google Cloud backups to the HPE Morpheus VM Essentials environment.
* Restore physical machines from backups created by Veeam Agents to the HPE Morpheus VM Essentials environment.

* Restore VMs from HPE Morpheus VM Essentials backups to Microsoft Azure, Amazon Web Services (AWS) and Google Cloud environments.
* Restore VMs from HPE Morpheus VM Essentials backups to Nutanix AHV, Proxmox VE, oVirt KVM, Scale Computing HyperCore environments.
* Perform Instant Recovery of HPE Morpheus VM Essentials VMs to VMware vSphere, Microsoft Hyper-V and Nutanix AHV environments.
* Restore application items (such as Microsoft Active Directory, Microsoft Exchange, Microsoft SharePoint, Microsoft DNS, PostgreSQL, Oracle Database and Microsoft SQL Server).
* Restore files and folders of HPE Morpheus VM Essentials VM guest OSes.
* Export disks of backed-up HPE Morpheus VM Essentials VMs to VMDK, VHD and VHDX formats.
* Mount disks of backed-up HPE Morpheus VM Essentials VMs to any server and access data in read-only mode.
