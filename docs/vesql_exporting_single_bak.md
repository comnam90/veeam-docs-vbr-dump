# Exporting Single Database


You can export a single Microsoft SQL Server database to the machine where Veeam Explorer for Microsoft SQL Server is running by following one of these scenarios:

* [Exporting Latest State](vesql_exporting_bak_latest_state.md)
* [Exporting Point-in-Time State](vesql_exporting_point_in_time_bak.md)
* [Exporting to Different Location](vesql_exporting_single_bak_tdl.md)
