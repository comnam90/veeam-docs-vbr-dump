# Performing Instant Recovery to Microsoft Hyper-V Using Web UI

In this article

With Instant Recovery, you can recover workloads from backups and register them as Microsoft Hyper-V VMs.

To perform Instant Recovery, use the Instant Recovery to Microsoft Hyper-V wizard.

1. [Launch the wizard](ir_launch_hv_web.md).
2. [Select workloads and restore points](ir_workloads_hv_web.md).
3. [Select a restore mode](ir_mode_hv_web.md).
4. [Select target hosts](ir_target_host_hv_web.md).
5. [Select target datastores](ir_target_datastore_hv_web.md).
6. [Configure network mapping](ir_netwrk_mapping_hv_web.md).
7. [Change names and UUIDs](ir_vm_name_hv_web.md).
8. [Specify a restore reason](ir_reason_hv_web.md).
9. [Verify Instant Recovery settings](ir_verify_hv_web.md).
10. [Finalize Instant Recovery to Microsoft Hyper-V](ir_finalize_hv_web.md).

|  |
| --- |
| Note |
| At the moment, web UI has a limited functionality. It allows recovering only Microsoft Hyper-V VMs to Microsoft Hyper-V. |
