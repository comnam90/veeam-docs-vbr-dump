# Installing Updates


Before you update Veeam Backup & Replication, [check prerequisites](update_vbr_byb.md). Then use the Veeam Backup & Replication update wizard to run the product update.

1. [Start the update wizard](update_vbr_launch.md).
2. [Select the component to update](update_vbr_select_component.md).
3. [Read and accept the license agreement](update_vbr_license_agreement.md).
4. [Perform the configuration check](update_vbr_configuration_check.md).
5. [Review the components that will be upgraded and begin the update process](update_vbr_review_components.md).

After you upgrade Veeam Backup & Replication, perform the [finalizing steps](update_vbr_after_update.md).

For information on how to update Veeam Backup & Replication in the unattended mode, see [Upgrading Veeam Backup & Replication in Silent Mode](upgrade_vbr_answer_file.md).
