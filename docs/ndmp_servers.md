# NDMP Servers

In this article

If your NAS device supports the NDMP protocol, you can back up data from it to tape. To do this, you need to add the NAS device as an NDMP server. For more information, see [NDMP Servers Backup to Tape](ndmp_servers_backup_to_tape.md).
