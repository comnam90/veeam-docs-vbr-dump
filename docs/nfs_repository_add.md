# Adding NFS Repositories


This section describes how to add an NFS share as a backup repository.

To add a backup repository, use the New Backup Repository wizard.

1. [Launch the New Backup Repository wizard](nfs_repository_launch.md).
2. [Specify backup repository name and description](nfs_repository_name.md).
3. [Specify shared folder settings](nfs_repository_server.md).
4. [Configure backup repository settings](nfs_repository_repository.md).
5. [Specify mount server settings](nfs_repository_mount_server.md).
6. [Review properties and components](nfs_repository_review.md).
7. [Apply backup repository settings](nfs_repository_apply.md).
8. [Finish working with the wizard](nfs_repository_finish.md).
