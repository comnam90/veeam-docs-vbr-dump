# Step 1. Launch Export Wizard


To launch the Export wizard, do the following:

1. In the navigation pane, select a database.
2. On the Database tab, select Export Files > Export point-in-time state to Desktop\<db\_name>.

Alternatively, you can right-click a database and select Export files > Export point-in-time state to Desktop\<db\_name>.
