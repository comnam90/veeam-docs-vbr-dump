# Restoring Documents and List Items


You can restore Microsoft SharePoint documents and list items to the same or a different site.

To restore a Microsoft SharePoint document or list item, take the following steps:

1. [Launch the Restore wizard](vesp_vbr_restore_items_wizard.md).
2. [Specify a target SharePoint site](vesp_vbr_restore_items_specify_target_site.md).
3. [Specify a target list](vesp_vbr_restore_items_specify_target_list.md).
4. [Specify restore options](vesp_vbr_restore_items_specify_restore_options.md).
5. [Review the operation summary](summary20.md).
