# Overview


Starting from version 12.2, Veeam Backup & Replication comes with a plug-in that extends the solution functionality and allows you to add Nutanix AHV servers to the backup infrastructure, as well as to manage data protection and recovery operations for Nutanix AHV workloads from a single console.
