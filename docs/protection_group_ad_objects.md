# Creating Protection Group for Microsoft Active Directory Objects

In this article

Before you create a protection group for Microsoft Active Directory objects, [check prerequisites](ad_objects_before.md). Then use the New Protection Group wizard to configure a protection group.

1. [Launch the New Protection Group wizard](ad_objects_launch.md).
2. [Specify protection group name and description](ad_objects_name.md).
3. [Specify Active Directory objects](protection_group_ad.md).
4. [Exclude objects from the protection group](protection_group_exclusions.md).
5. [Specify credentials](protection_group_accounts.md).
6. [Specify discovery and deployment options](ad_objects_options.md).
7. [Specify advanced protection group settings](ad_objects_advanced.md).
8. [Review components](ad_objects_review.md).
9. [Assess results](ad_objects_results.md).
10. [Finish working with the wizard](ad_objects_finish.md).
