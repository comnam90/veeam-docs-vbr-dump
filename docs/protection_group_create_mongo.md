# Creating Protection Group for MongoDB


Before you create a protection group for MongoDB, [check prerequisites](mongo_protection_group_before.md). Then use the New Protection Group wizard to configure a protection group.

1. [Launch the New Protection Group wizard](mongo_protection_group_launch.md).
2. [Specify protection group name and description](mongo_protection_group_name.md).
3. [Specify MongoDB replica sets you want to back up](mongo_protection_group_scope_deployments.md).
4. [Select computers from the MongoDB replica sets](mongo_protection_group_scope_computers.md).
5. [Specify discovery and deployment options](mongo_protection_group_options.md).
6. [Specify advanced protection group settings](mongo_protection_group_advanced.md).
7. [Review components](mongo_protection_group_review.md).
8. [Assess results](mongo_protection_group_apply.md).
9. [Finish working with the wizard](mongo_protection_group_finish.md).
