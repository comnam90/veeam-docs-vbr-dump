# Retrieving Backup Files

In this article

To start the retrieval of archived backup files, use the Retrieve Backup wizard.

1. [Launch Retrieve Backup wizard](archive_retrieval_launch.md).
2. [Select Retrieval Mode](archive_retrieval_mode.md).
3. [Select Availability Period](archive_retrieval_availability.md).
4. [Finish Working with the wizard](archive_retrieval_summary.md).
