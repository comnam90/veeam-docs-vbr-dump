# Adding Linux Repositories Using Web UI


This section describes how to add a Linux server as a backup repository.

To add a backup repository using the Veeam Backup & Replication web UI, use the New Backup Repository wizard.

1. [Launch the New Backup Repository wizard](linux_repository_launch_web.md).
2. [Specify backup repository name and description](linux_repository_name_web.md).
3. [Specify server settings](linux_repository_server_web.md).
4. [Configure backup repository settings](linux_repository_repository_web.md).
5. [Specify mount server settings](linux_repository_mount_server_web.md).
6. [Review properties and components](linux_repository_review_web.md).
7. [Finish working with the wizard](linux_repository_finish_web.md).
