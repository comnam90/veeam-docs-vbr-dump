# Exporting to Custom Location

In this article

This section explains how to export a standalone database or a Data Guard database as database files to a selected point-in-time state on the backup file. Your Oracle data will be restored to the specified location.

To export database files to a custom location, do the following:

1. [Launch the Export wizard](veor_df_export_custom_wizard.md).
2. [Specify a restore point](veor_df_export_custom_srp.md).
3. [Fine-tune the restore point](veor_df_export_custom_frp.md).
4. [Specify export location](veor_df_export_custom_stl.md).
5. [Review the export summary](veor_export_dbfiles_tas_summary.md).
