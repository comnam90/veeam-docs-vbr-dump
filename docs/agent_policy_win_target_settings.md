# Step 8. Specify Backup Storage Settings

In this article

Specify backup storage settings for the Veeam Agent backup policy:

* [Local storage settings](agent_policy_win_target_drive.md) — if you have selected the Local storage option at the [Destination](agent_policy_win_destination.md) step of the wizard.

* [Shared folder settings](agent_policy_win_target_share.md) — if you have selected the Shared folder option at the [Destination](agent_policy_win_destination.md) step of the wizard.

* [Veeam backup repository settings](agent_policy_win_target_vbr.md) — if you have selected the Veeam backup repository option at the [Destination](agent_policy_win_destination.md) step of the wizard.

* [Cloud repository settings](agent_policy_win_target_cloud.md) — if you have selected the Veeam Cloud Connect repository option at the [Destination](agent_policy_win_destination.md) step of the wizard.
