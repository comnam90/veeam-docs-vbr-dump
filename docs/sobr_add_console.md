# Adding Scale-Out Backup Repositories Using Console

In this article

To add the scale-out backup repository using console, use the New Scale-out Backup Repository wizard.

1. [Launch the New Scale-Out Backup Repository Wizard](sobr_launch.md).
2. [Specify scale-out backup repository name](sobr_add_name.md).
3. [Add performance extents](sobr_add_extents.md).
4. [Specify backup placement policy](sobr_add_policy.md).
5. [Add capacity tier](new_capacity_tier.md).
6. [Add archive tier](new_archive_tier.md).
7. [Finish working with wizard](sobr_add_summary.md).
