# Step 1. Launch Backup Repository Wizard

In this article

To launch the New Veeam Data Cloud Vault wizard using web UI, in the Infrastructure section of the main menu, on the Repositories page, select the Veeam Data Cloud Vault tab, and click Get Veeam Vault.
