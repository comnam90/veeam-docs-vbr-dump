# Performing Instant Recovery to Microsoft Azure

In this article

With Instant Recovery to Microsoft Azure, you can recover different workloads from backups and register them as Microsoft Azure VMs.

To perform Instant Recovery, use the Instant Recovery to Microsoft Azure wizard.

1. [Launch the wizard](ir_azure_launch.md).
2. [Select workloads and restore points](ir_azure_workload.md).
3. [Select a subscription and location](ir_azure_subscription.md).
4. [Specify names and tags](ir_azure_name.md).
5. [Specify VM sizes and disks](ir_azure_disks.md).
6. [Specify a resource group](ir_azure_group.md).
7. [Configure network settings](ir_azure_network.md).
8. [Configure helper appliance settings](ir_azure_helper.md).
9. [Specify a restore reason](ir_azure_reason.md).
10. [Verify recovery settings](ir_azure_summary.md).
11. [Finalizing Instant Recovery to Microsoft Azure](ir_azure_finalize.md).
