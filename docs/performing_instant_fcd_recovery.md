# Performing Instant FCD Recovery

In this article

To register VM disks as FDCs, use the Instant Disk Recovery wizard.

1. [Check prerequisites](byb_fcd.md).
2. [Launch the Instant Disk Recovery wizard](instant_fcd_recovery_launch.md).
3. [Select a source VM](instant_fcd_recovery_vm.md).
4. [Select a restore point](instant_fcd_recovery_restore_point.md).
5. [Select virtual disks to restore](instant_fcd_recovery_disks.md).
6. [Specify a cluster to register FCDs](instant_fcd_recovery_target_host.md).
7. [Specify a location for redo logs](instant_fcd_recovery_write_cache.md).
8. [Specify a restore reason](instant_fcd_recovery_reason.md).
9. [Verify recovery settings](instant_fcd_recovery_summary.md).
