# Installing Veeam Software Appliance from ISO


To install and configure Veeam Software Appliance, perform the following steps:

1. [Mount the ISO image](deployment_linux_iso_install_mount.md).
2. [Select the product](deployment_linux_iso_install_select_product.md).
3. [Begin installation](deployment_linux_iso_install_type.md).
4. [Read and accept license agreements](deployment_linux_iso_install_license.md).
5. [Specify the hostname](deployment_linux_iso_install_hostname.md).
6. [Review network settings](deployment_linux_iso_install_network.md).
7. [Review server time settings](deployment_linux_iso_install_time.md).
8. [Configure the default host administrator account](deployment_linux_iso_install_host_administrator.md).
9. [Configure the default security officer account](deployment_linux_iso_install_security_officer.md).
10. [Finish the configuration](deployment_linux_iso_install_summary.md).
