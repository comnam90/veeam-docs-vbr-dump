# Unstructured Data Recovery


Veeam Backup & Replication offers the following types of recovery:

* [File share data recovery](file_share_data_recovery.md) — to restore data previously backed up with file backup jobs.
* [Object storage data recovery](os_data_recovery.md) — to restore data previously backed up with object storage backup jobs.
