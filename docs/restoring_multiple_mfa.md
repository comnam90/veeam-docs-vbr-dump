# Restoring Multiple Mailboxes

In this article

This scenario is only available for data restore from backups created by Veeam Backup for Microsoft 365.

To restore multiple Exchange mailboxes, do the following:

1. [Launch the Restore wizard](launch_restore_mailboxes_mfa.md).
2. [Select an authentication method](select_authentication_method_mailboxes_mfa.md).
3. [Log in to Microsoft 365](login_on_to_office_365_mfa_mailboxes.md).
4. [Select mailboxes to restore](select_mailboxes_to_restore_mfa.md).
5. [Select restore options](vex_restoring_multiple_options.md).
6. [Review the operation summary](summary4.md).
