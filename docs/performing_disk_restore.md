# Restoring Virtual Disks

In this article

To restore virtual disks, use the Virtual Disk Restore wizard.

1. [Check prerequisites](disk_restore_before_you_begin.md).
2. [Launch the Virtual Disk Restore wizard](disk_restore_launch_vm.md).
3. [Select a VM](disk_restore_vms_vm.md).
4. [Select a restore point](disk_restore_point_vm.md).
5. [Select virtual hard disks to restore](disk_restore_map_vm.md).
6. [Specify secure restore settings](disk_restore_av_vm.md).
7. [Specify a restore reason](disk_restore_reason_vm.md).
8. [Finish working with the wizard](disk_restore_review_vm.md).
