# Restoring Multiple Mailboxes


To restore multiple Exchange mailboxes, do the following:

1. [Launch the Restore wizard](vex_vbr_restore_onprem_multiple_launch_wizard.md).
2. [Specify a domain and credentials](vex_vbr_restore_onprem_multiple_specify_domain_creds.md).
3. [Select mailboxes to restore](vex_vbr_restore_onprem_multiple_select_mailboxes.md).
4. [Specify a target mailbox server](vex_vbr_restore_onprem_multiple_specify_server.md).
5. [Select restore options](vex_vbr_restore_onprem_multiple_restore_options.md).
6. [Review the operation summary](summary7.md).
