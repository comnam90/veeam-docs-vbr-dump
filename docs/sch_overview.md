# Overview


Starting from version 13.0.1, Veeam Backup & Replication comes with a plug-in that extends the solution functionality and allows you to add Scale Computing HyperCore servers to the backup infrastructure, as well as to manage data protection and recovery operations for Scale Computing HyperCore workloads from a single console.
