# Instant Recovery

In this article

Instant recovery combines capabilities of the data restore and data publishing features. If you want the database to be online as quickly as possible, you can use the Instant Recovery option and restore it in the background.

Before performing instant recovery, read the [Considerations and Limitations](veor_considerations.md) section.
