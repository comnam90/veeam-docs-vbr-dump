# Object Restore from Tape

In this article

You can restore object storage data previously archived with object to tape jobs. Entire buckets or selected objects and prefixes can be restored to the following locations:

* Original object storage location.
* Alternative object storage location.
* Microsoft Windows or Linux servers.

* SMB (CIFS) and NFS file shares.

You can restore objects to any restore point available on tape.
