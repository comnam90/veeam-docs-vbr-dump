# Adding VMware vSphere Server Using Console


This section describes how to add a VMware vSphere server as a backup infrastructure component.

To add a VMware vSphere server using the Veeam Backup & Replication console, use the New VMware Server wizard.

1. [Launch the New VMware Server wizard](vmware_server_launch.md).
2. [Specify a server name or address](vmware_server_name.md).
3. [Specify credentials](vmware_server_credentials.md).
4. [Apply settings](vmware_server_apply.md).
5. [Finish working with the wizard](vmware_server_summary.md).
