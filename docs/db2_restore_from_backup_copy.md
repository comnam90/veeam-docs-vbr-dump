# Restore from Backup Copy

In this article

You can restore IBM Db2 databases from backup copies of backups created with Veeam Plug-In for IBM Db2. The procedure of database restore from a backup copy is the same as database restore to another server. For more information, see [Restore to Another Server](db2_restore_to_another.md).

Page updated 4/2/2024

Page content applies to build 13.0.1.1071
