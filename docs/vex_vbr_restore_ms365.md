# Restore to Microsoft 365 Organizations


You can restore your Exchange data to Microsoft 365 organizations with enabled [multi-factor authentication](https://docs.microsoft.com/en-us/microsoft-365/admin/security-and-compliance/set-up-multi-factor-authentication?view=o365-worldwide) (MFA).

To restore a single mailbox, folder or item, do the following:

1. [Launch the Restore wizard](vex_vbr_restore_ms365_single_launch_wizard.md).
2. [Specify a target mailbox](vex_vbr_restore_ms365_single_target_mailbox.md).
3. [Select an authentication method](vex_vbr_restore_ms365_single_select_authentication_method.md).
4. [Log in to Microsoft 365](vex_vbr_restore_ms365_single_log_in.md).
5. [Specify a user account to impersonate with a public folder mailbox](vex_vbr_restore_ms365_single__specify_impersonation_account.md).
6. [Specify a CAS server](vex_vbr_restore_ms365_single_mfa_cas_server.md).
7. [Specify a target folder](vex_vbr_restore_ms365_single_target_folder.md).
8. [Specify restore options](vex_vbr_restore_ms365_single_restore_options.md).
9. [Review the operation summary](summary8.md).
