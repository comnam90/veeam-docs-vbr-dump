# Restoring to Another Server


To restore a single Microsoft SQL Server database to another server, use the Restore wizard.

1. [Launch the Restore wizard](vesql_restore_wizard_single.md).
2. [Specify a restore point](vesql_restore_point_single.md).
3. [Fine-tune the restore point](vesql_fine_tune_single.md).
4. [Specify the target server](vesql_target_server_single.md).
5. [Specify Always On restore options](vesql_always_on_single.md).
6. [Specify files location](vesql_files_location_single.md).
7. [Specify the recovery state](vesql_recovery_state_single.md).
8. [Review the restore summary](vesql_restore_single_tas_summary.md).
