# Restoring Multiple OneDrives


To restore multiple OneDrives, do the following:

1. [Launch the Restore wizard](veod_bunch_launch.md).
2. [Select an authentication method](veod_select_authentication_method_restore_bunch.md).
3. [Log in to Microsoft 365](veod_login_restore_bunch.md).
4. [Select OneDrives](select_onedrives.md).
5. [Review the operation summary](summary22.md).
