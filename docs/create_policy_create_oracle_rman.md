# Creating Oracle RMAN Backup Policy

In this article

To back up databases protected with Veeam Plug-In for Oracle RMAN, you must configure an application backup policy in Veeam Backup & Replication.

Before configuring an application policy for Veeam Plug-In for Oracle RMAN, [check prerequisites](policy_oracle_rman_before.md). Then use the New Application Backup Policy wizard to define settings for the application policy.

1. [Launch the New Application Backup Policy wizard](policy_oracle_rman_launch.md).
2. [Specify policy name and description](policy_oracle_rman_name.md).
3. [Select databases to back up](policy_oracle_rman_databases.md).
4. [Specify backup storage settings](policy_oracle_rman_repository.md).
5. [Specify advanced backup settings](policy_oracle_rman_advanced.md).
6. [Specify database processing settings](policy_oracle_rman_database_processing.md).
7. [Specify the backup schedule](policy_oracle_rman_schedule.md).
8. [Review backup policy settings](policy_oracle_rman_summary.md).
