# Managing Backups in Veeam Backup & Replication


In the Veeam Backup & Replication console, backups created by Veeam Plug-Ins are displayed in the Backups node of the Home view. In the working area, backups created by Veeam Plug-In for Oracle RMAN are listed under the Oracle RMAN node.

You can use the Veeam Backup & Replication console to perform the following operations with Veeam Plug-In backups:

* [Restore from Oracle RMAN backup](plugins_rman_restore_from.md)
* [Create a recovery token for a backup](plugins_rman_backup_management_token.md)
* [Remove a backup from configuration](plugins_rman_backup_management_remove.md)
* [Delete a backup from the backup repository](plugins_rman_backup_management_delete.md)
