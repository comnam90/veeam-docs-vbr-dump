# Adding Microsoft Windows Servers Using Console


This section describes how to add a Microsoft Windows server as a backup infrastructure component.

To add a Microsoft Windows server using the Veeam Backup & Replication console, use the New Windows Server wizard.

1. [Launch the New Windows Server wizard](windows_server_launch.md).
2. [Specify a server name or address](windows_server_name.md).
3. [Specify credentials](windows_server_credentials.md).
4. [Review components](windows_server_review.md).
5. [Apply settings](windows_server_apply.md).
6. [Finish working with the wizard](windows_server_summary.md).
