# Creating SAP HANA Backup Policy


To back up databases protected with Veeam Plug-In for SAP HANA, you must configure an application backup policy in Veeam Backup & Replication.

Before configuring an application backup policy, [check prerequisites](policy_sap_hana_before.md). Then use the New Application Backup Policy wizard to define settings for the application backup policy.

1. [Launch the New Application Backup Policy wizard](policy_sap_hana_launch.md).
2. [Specify policy name and description](policy_sap_hana_name.md).
3. [Select databases to back up](policy_sap_hana_databases.md).
4. [Specify backup storage settings](policy_sap_hana_repository.md).
5. [Specify advanced backup settings](policy_sap_hana_advanced.md).
6. [Specify credentials](policy_sap_hana_credentials.md).
7. [Specify log backup settings](policy_sap_hana_log_backup.md).
8. [Specify the backup schedule](policy_sap_hana_schedule.md).
9. [Review backup policy settings](policy_sap_hana_summary.md).
