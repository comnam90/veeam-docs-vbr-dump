# Configuring Veeam Plug-In for Nutanix AHV


To start working with Veeam Plug-in for Nutanix AHV, perform a number of steps for its configuration:

1. [Configure backup repositories](ahv_configure_repository.md) where Veeam Plug-in for Nutanix AHV will store backups of Nutanix AHV VMs.
2. [Add to the backup infrastructure the Nutanix AHV cluster or Prism Central](ahv_add_ahv_cluster.md) that administers Nutanix AHV resources you want to protect.
3. [Deploy workers](ahv_workers.md) that will transfer backup traffic.
4. [[Optional] Configure email settings and notifications](ahv_general_settings.md).
