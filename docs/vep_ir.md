# Instant Recovery


Instant recovery combines capabilities of the data restore and data publishing features. If you want instances to be online as quickly as possible, you can use the Instant Recovery option and restore the instances in the background.

Before performing instant recovery, read the [Considerations and Limitations](vep_considerations.md) section.
