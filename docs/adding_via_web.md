# Adding Veeam Infrastructure Appliance Using Web UI


This section describes how to add a host deployed from Veeam Infrastructure Appliance as a backup infrastructure component.

Before you add a Veeam Infrastructure Appliance, [check prerequisites](linux_infrastructure_appliance_byb.md) and configure the machine with Veeam Infrastructure Appliance. To add it as a backup infrastructure components using the Veeam Backup & Replication web UI, use the New Linux Server wizard for Veeam Infrastructure Appliances.

1. [Launch the New Linux Server wizard](add_via_launch_web.md).
2. [Specify a server name or address](add_via_server_name_web.md).
3. [Review components](add_via_review_web.md).
4. [Finish working with the wizard](add_via_summary_web.md).
