# Assembling High Availability Cluster

In this article

To be able to use continuous replication between 2 Veeam Software Appliance, you must submit a request to enable High Availability for both servers, and then assemble these servers into a cluster using the New High Availability Cluster wizard.

Before you assemble a cluster, [check the prerequisites](high_availability_limitations.md). Then, use the New High Availability Cluster wizard to assemble the cluster.

In This Section

1. [Request the necessary permissions](high_availability_configuration_byb.md).
2. [Launch the New High Availability Cluster wizard](high_availability_configuration_launch.md).
3. [Specify the cluster endpoint](high_availability_configuration_endpoint.md).
4. [Specify the cluster nodes](high_availability_configuration_nodes.md).
5. [Finish working with wizard](high_availability_configuration_summary.md).
