# Installing Veeam Backup & Replication on Microsoft Windows


Before you install Veeam Backup & Replication, [check prerequisites](installation_byb.md). Then use the Veeam Backup & Replication setup wizard to install the product.

1. [Start the setup wizard](install_vbr_launch.md).
2. [Select the component to install](install_vbr_select_component.md).
3. [Read and accept the license agreement](install_vbr_eula.md).
4. [Provide a license file](install_vbr_license.md).
5. [Install missing software](install_vbr_missing.md).
6. [Review the default installation settings](install_vbr_settings.md).
7. [Specify service account settings](install_vbr_account.md).
8. [Specify the database engine and instance](install_vbr_sql.md).
9. [Specify data locations](install_vbr_data.md).
10. [Specify service ports](install_vbr_service_ports.md).
11. [Begin installation](install_vbr_summary.md).
