# Restoring Multiple Databases


With Veeam Explorer for Oracle, you can restore multiple standalone databases from image-level backups created by Veeam Backup & Replication.

To restore multiple Oracle databases, follow one of these scenarios:

* [Restoring Latest State](veor_restore_multiple_latest.md)
* [Restoring Point-in-Time State](veor_restore_multiple_pit.md)
* [Restoring to Another Server](veor_restore_multiple_tas.md)
