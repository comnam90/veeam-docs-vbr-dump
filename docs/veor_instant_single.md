# Instant Recovery of Single Database

In this article

You can use Veeam Explorer for Oracle to perform instant recovery to the original server or to any other server in your network where Oracle Database is installed.

To instantly recover a standalone database or a Data Guard database, follow one of these scenarios:

* [Instant Recovery of Latest State](veor_ir_single_latest.md)
* [Instant Recovery of Point-in-Time State](veor_ir_single_pit.md)
* [Instant Recovery to Another Server](veor_ir_single_tas.md)

Page updated 12/9/2024

Page content applies to build 13.0.1.1071
