# Performing Instant Recovery to Microsoft Hyper-V Using Console

In this article

With Instant Recovery, you can recover different workloads from backups and register them as Microsoft Hyper-V VMs. For the list of backups that you can use for recovery, see [Supported Backup Types](instant_recovery_to_hv.md#backups).

5. [Select a recovery mode](instant_recovery_mode_hv.md).
6. [Select target hosts](ir_target_host_hv.md).
7. [Select target datastores](ir_target_datastore_hv.md).
8. [Specify network mapping](ir_netwrk_mapping_hv.md).
9. [Change VM names and UUIDs](ir_vm_name_hv.md).

12. [Specify a restore reason](ir_reason_hv.md).
13. [Verify Instant Recovery settings](ir_verify_hv.md).
