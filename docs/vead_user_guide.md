# Veeam Explorer for Microsoft Active Directory

In this article

Veeam Explorer for Microsoft Active Directory allows you to restore and export Active Directory objects and containers from backups, snapshot replicas, CDP replicas, VeeamZIP backups or storage snapshots created by Veeam Backup & Replication.
