# Instant Recovery of Point-in-Time State


To perform instant recovery of multiple instances to a selected point-in-time state, use the Instant Recovery wizard.

1. [Launch the Instant Recovery wizard](vep_ir_multiple_pit_wizard.md).
2. [Specify a restore point](vep_ir_multiple_pit_specify_restore_point.md).
3. [Specify switchover settings](vep_ir_multiple_pit_specify_switchover_settings.md).
4. [Finalize the Instant Recovery session](vep_ir_multiple_pit_finalize_ir_session.md).
