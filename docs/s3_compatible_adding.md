# Adding S3 Compatible Object Storage

In this article

This section describes how to add S3 compatible object storage to the backup infrastructure.

To add S3 compatible object storage, use the New Object Storage Repository wizard.

1. [Launch the New Object Storage Repository wizard](new_object_repository_s3.md).
2. [Specify object storage name](compatible_repository_name.md).
3. [Specify object storage account](compatible_repository_account.md).
4. [Specify object storage settings](compatible_storage_details.md).
5. [Specify mount server settings](compatible_mount_server.md).
6. [Review components](compatible_review.md).
7. [Apply settings](compatible_apply.md).
8. [Finish working with the wizard](compatible_finishing_wizard.md).
