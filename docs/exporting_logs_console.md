# Exporting Logs Using Console


To aggregate all log files in the same location, use the Export Logs wizard in the Veeam Backup & Replication console.

1. [Launch the Export Logs wizard](launch_export_logs_wizard.md).
2. [Select virtual infrastructure scope](export_logs_servers.md).
3. [Specify the time interval](export_logs_date.md).
4. [Specify a destination folder](export_logs_destination.md).
5. [Review the results](export_logs_results.md).

|  |
| --- |
| Note |
| If you do not have access to the Veeam Backup & Replication console and you cannot use the built-in log export, export logs manually as described in [this Veeam KB article](https://www.veeam.com/kb1832). |
