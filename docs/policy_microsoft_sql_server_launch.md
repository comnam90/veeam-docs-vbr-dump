# Step 1. Launch New Application Backup Policy Wizard


To create an application policy for Veeam Plug-In for Microsoft SQL Server, you must launch the New Application Backup Policy wizard. To do this, on the Home tab, click Jobs > Backup > Application > Microsoft SQL Server.
