# Creating Protection Group for Microsoft Active Directory Objects

In this article

Before you create a protection group for Microsoft Active Directory Objects, [check prerequisites](protection_group_before_ad.md). Then use the New Protection Group wizard to configure a protection group.

1. [Launch the New Protection Group wizard](protection_group_launch_ad.md).
2. [Specify protection group name and description](protection_group_name_ad.md).
3. [Specify protection scope for the protection group](protection_group_scope_ad.md).
4. [Exclude objects from the protection group](protection_group_exclusions_ad.md).
5. [Specify credentials](protection_group_accounts_ad.md).
6. [Specify discovery and deployment options](protection_group_options_ad.md).
7. [Specify advanced protection group settings](protection_group_advanced_ad.md).
8. [Review components](protection_group_review_ad.md).
9. [Assess results](protection_group_apply_ad.md).
10. [Finish working with the wizard](protection_group_finish_ad.md).
