# Instant Recovery


Instant recovery combines capabilities of the data restore and data publishing features. If you want databases to be online as quickly as possible, you can use the Instant Recovery option and restore databases in the background.

Before performing instant recovery, read the [Considerations and Limitations](vesql_considerations.md) section.
