# Step 1. Launch Instant Recovery Wizard

In this article

To launch the Instant Recovery wizard, do the following:

1. In the navigation pane, select a database.
2. On the Database tab, select Instant Recovery > Instant recovery of the point-in-time state to <original\_location>.

Alternatively, you can right-click a database and select Instant recovery > Instant recovery of the point-in-time state to <original\_location>.
