# Adding VMware Backup Proxies Using Console

In this article

Before adding a VMware backup proxy using the Veeam Backup & Replication console, [check prerequisites](backup_proxy_requirements.md). Then use the New VMware Proxy wizard.

1. [Launch the New VMware Proxy wizard](vmware_proxy_launch.md).
2. [Choose a server](vmware_proxy_server.md).
3. [Configure traffic rules](vmware_proxy_traffic.md).
4. [Review components to install](vmware_proxy_review.md).
5. [Apply VMware backup proxy settings](vmware_proxy_apply.md).
6. [Finish working with the wizard](vmware_proxy_summary.md).
