# Adding External Google Cloud Storage

In this article

To add an external Google Cloud storage repository to the backup infrastructure, do the following:

1. [Launch the New External Repository wizard](external_google_cloud_launching_wizard.md).
2. [Specify the repository name](external_google_cloud_name.md).
3. [Specify the cloud account](external_google_cloud_account.md).
4. [Specify cloud storage details](external_google_cloud_details.md).
5. [Apply settings](external_google_cloud_apply.md).
6. [Finish working with the wizard](external_google_cloud_finish.md).

Page updated 4/15/2024
