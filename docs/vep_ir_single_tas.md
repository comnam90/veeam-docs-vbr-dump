# Instant Recovery to Another Server


To perform instant recovery of an instance to another server, use the Instant Recovery wizard.

1. [Launch the Instant Recovery wizard](vep_ir_single_tas_wizard.md).
2. [Specify a restore point](vep_ir_single_tas_specify_restore_point.md).
3. [Specify the target server](vep_ir_single_tas_specify_target_server.md).
4. [Specify PostgreSQL settings](vep_ir_single_tas_specify_postgresql_settings.md).
5. [Specify tablespaces](vep_ir_single_tas_specify_tablespaces.md).
6. [Specify switchover settings](vep_ir_single_tas_specify_switchover_settings.md).
7. [Finalize the Instant Recovery session](vep_ir_single_tas_finalize_ir_session.md).
