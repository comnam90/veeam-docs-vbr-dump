# Job Encryption

In this article

Job-level encryption is supported for the following jobs:

* Backup jobs. For more information, see [Encrypting Backup Jobs](encrypting_backup_jobs.md).
* Backup copy jobs. For more information, see [Encrypting Backup Copy Jobs](encrypting_backup_copy_jobs.md).
* File backup jobs. For more information, see the [Storage Settings](file_share_backup_job_advanced_storage.md) section in the job configuration.
* Object storage backup jobs. For more information, see the [Storage Settings](os_backup_job_advanced_storage.md) section in the job configuration.
* VeeamZIP sessions. For more information, see [Encrypting VeeamZIP Backups](encrypting_veeam_zip.md).
* Configuration backup jobs. For more information, see [Creating Encrypted Configuration Backups](config_backup_encrypted.md).

|  |
| --- |
| Note |
| If you plan to store backups in [Veeam Data Cloud Vault](osr_adding_veeam_data_cloud_vault.md), you must enable job encryption. |
