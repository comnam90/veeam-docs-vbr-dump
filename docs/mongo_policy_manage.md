# Managing Application Backup Policy


You can use the Veeam Backup & Replication console to perform the following operations with an application backup policy:

* [Start and stop backup operations on computers added to the backup policy](mongo_policy_manage_start_stop.md).
* [Retry a backup policy run](mongo_policy_manage_retry.md).
* [Perform active full backup on computers added to the backup policy](mongo_policy_manage_active_full.md).
* [Edit backup policy settings](mongo_policy_manage_edit.md).
* [Enable and disable a backup policy](mongo_policy_manage_disable.md).
* [Clone a backup policy](mongo_policy_manage_clone.md).
* [Delete a backup policy](mongo_policy_manage_delete.md).
