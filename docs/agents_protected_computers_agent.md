# Managing Veeam Agents


You can use the backup console to manage the following Veeam Agents on a specific computer in the inventory:

* Veeam Agent for Microsoft Windows
* Veeam Agent for Linux
* Veeam Agent for Unix

Keep in mind that Veeam Agents for computers that you plan to add to a protection group for pre-installed Veeam Agents require a different installation approach. To learn more, see [Deploying Veeam Agents Using Generated Setup Files](agents_deploy_package.md).

You can perform the following management tasks for Veeam Agent:

* [Install Veeam Agent](agents_protected_computers_install.md).
* [Upgrade Veeam Agent](agents_protected_computers_upgrade.md).
* [Manage the Veeam CBT Driver](agents_protected_computers_driver.md).
* [Uninstall Veeam Agent](agents_protected_computers_uninstall.md).
