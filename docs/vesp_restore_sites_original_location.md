# Restoring Sites to Original Location


To restore Microsoft SharePoint sites to the original location, do the following:

1. [Launch the Restore wizard](vesp_restore_site_launch_wizard.md).
2. [Select an authentication method](vesp_select_authentication_method_sites.md).
3. [Log in to Microsoft 365](vesp_login_sites.md).
4. [Specify a target site alias](specify_target_site.md).
5. [Specify restore options](vesp_specify_restore_options_sites.md).
6. [Review the operation summary](summary9.md).
