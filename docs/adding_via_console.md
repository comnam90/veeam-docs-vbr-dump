# Adding Veeam Infrastructure Appliance Using Console

In this article

This section describes how to add a host deployed from Veeam Infrastructure Appliance as a backup infrastructure component.

Before you add a Veeam Infrastructure Appliance, [check prerequisites](linux_infrastructure_appliance_byb.md) and configure the machine with Veeam Infrastructure Appliance. To add it as a backup infrastructure component using the Veeam Backup & Replication console, use the New Linux Server wizard for Veeam Infrastructure Appliances.

1. [Launch the New Linux Server wizard](linux_server_launch.md).

1. [Specify a server name or address](linux_server_name.md).
2. [Review components](linux_server_review.md).
3. [Apply settings](linux_server_apply.md).
4. [Finish working with the wizard](linux_server_summary.md).
