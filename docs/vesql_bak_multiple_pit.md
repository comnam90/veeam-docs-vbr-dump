# Exporting Point-in-Time State

In this article

To export data as of a point-in-time state to the default location, use the Export wizard.

1. [Launch the Export wizard](vesql_bak_multiple_pit_wizard.md).
2. [Select databases you want to export](vesql_bak_multiple_pit_select_databases.md).
3. [Specify a restore point](vesql_bak_multiple_pit_restore_point.md).
4. [Review the export summary](vesql_export_multiple_pit_bak_summary.md).
