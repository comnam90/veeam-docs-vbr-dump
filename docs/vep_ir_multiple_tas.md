# Instant Recovery to Another Server

In this article

To perform instant recovery of multiple instances to another server, use the Instant Recovery wizard.

1. [Launch the Instant Recovery wizard](vep_ir_multiple_tas_wizard.md).
2. [Specify restore point](vep_ir_multiple_tas_specify_restore_point.md).
3. [Specify target server](vep_ir_multiple_tas_specify_target_server.md).
4. [Specify switchover settings](vep_ir_multiple_tas_specify_switchover_settings.md).
5. [Finalize the Instant Recovery session](vep_ir_multiple_tas_finalize_ir_session.md).
