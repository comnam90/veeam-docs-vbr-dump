# Physical and Cloud Backup


Veeam Backup & Replication allows remote deployment and central administration of Veeam Agents that protect the following physical, virtual or cloud-based endpoints:

* Physical or virtual machines running Microsoft Windows, Linux, IBM AIX, Oracle Solaris or macOS operating systems.
* Cloud-based machines running Microsoft Windows or Linux.

For more information on OS-specific system requirements for Veeam Agent computers, see the following subsections:

* [System Requirements for Microsoft Windows Computers](platform_support_agent_management_windows.md)
* [System Requirements for Linux Computers (Veeam Agent with kernel module)](platform_support_agent_management_linux.md)
* [System Requirements for Linux Computers (nosnap Veeam Agent)](platform_support_agent_management_linux_nosnap.md)
* [System Requirements for IBM AIX Computers](platform_support_agent_management_aix.md)
* [System Requirements for Oracle Solaris Computers](platform_support_agent_management_solaris.md)
* [System Requirements for Mac Computers](platform_support_agent_management_mac.md)
