# Adding Deduplicating Storage Appliances

In this article

This section describes how to add a deduplicating storage appliance as a backup repository.

Before you add a backup repository, check [prerequisites](repo_before_you_begin.md). Then use the New Backup Repository wizard to add the backup repository.

1. [Launch the New Backup Repository wizard](dsa_repository_launch.md).
2. [Specify backup repository name and description](dsa_repository_name.md).
3. [Specify server settings](dsa_repository_server.md).
4. [Configure backup repository settings](dsa_repository_repository.md).
5. [Specify mount server settings](dsa_repository_mount_server.md).
6. [Review properties and components](dsa_repository_review.md).
7. [Apply backup repository settings](dsa_repository_apply.md).
8. [Finish working with the wizard](dsa_repository_finish.md).
