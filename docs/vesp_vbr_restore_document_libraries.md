# Restoring Document Libraries and Lists


You can restore Microsoft SharePoint document libraries and lists to the same or a different site.

To restore a Microsoft SharePoint library or list, take the following steps:

1. [Launch the Restore wizard](vesp_vbr_restore_document_libraries_wizard.md).
2. [Specify a target SharePoint site](vesp_vbr_restore_document_libraries_specify_target_site.md).
3. [Specify target list](vesp_vbr_restore_document_libraries_specify_target_list.md).
4. [Specify restore options](vesp_vbr_restore_document_libraries_specify_restore_options.md).
5. [Review the operation summary](summary19.md).
