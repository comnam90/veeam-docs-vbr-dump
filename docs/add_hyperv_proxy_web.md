# Adding Off-Host Backup Proxies Using Web UI

In this article

You can configure one or more off-host backup proxies in the backup infrastructure.

Before you add an off-host backup proxy using Veeam Backup & Replication web UI, [check prerequisites](offhost_backup_proxy.md#req_general). Then use the New Hyper-V Proxy wizard to add the off-host backup proxy.

1. [Launch the New Hyper-V Off-Host Backup Proxy wizard](hv_proxy_launch_web.md).
2. [Choose a server](hv_proxy_server_web.md).
3. [Finish working with the wizard](hv_proxy_summary_web.md).
