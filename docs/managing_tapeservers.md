# Modifying Tape Servers

In this article

To edit settings of an added tape server:

1. Open the Tape Infrastructure view.
2. Select the Servers node in the inventory pane.
3. Select the necessary server in the working area and click Edit Tape Server on the ribbon. You can also right-click the necessary server in working area and select Properties.
4. Then edit the tape server settings as required.
