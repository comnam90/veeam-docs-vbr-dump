# Adding NFS File Share


Before you add an NFS file share to the inventory of the virtual infrastructure, consider the following:

* The file share meets requirements listed in [Platform Support](https://helpcenter.veeam.com/docs/vbr/userguide/platform_support.html?ver=13#unstructured-data).
* If you plan to use a dedicated backup proxy server or cache repository, make sure these components are added in Backup Infrastructure.

To add an NFS file share as a source of unstructured data, do the following:

1. [Launch the New File Share wizard](file_share_backup_nfs_share_launch_wizard.md).
2. [Specify the path to the NFS file share](file_share_backup_nfs_share_path.md).
3. [Specify advanced NFS file share settings](file_share_backup_nfs_share_advanced_settings.md).
4. [Specify file share processing settings](file_share_backup_nfs_share_processing_settings.md).
5. [Apply file share settings](file_share_backup_nfs_share_apply.md).
6. [Finish working with the wizard](file_share_backup_nfs_share_summary.md).
