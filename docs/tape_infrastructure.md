# Tape Infrastructure Components


The tape infrastructure requires a set of components that you must configure before you start archiving data to tape. Some components are mandatory and provide core functionality; some components are optional and can be used to provide additional functionality for your business needs.

The following backup infrastructure components are required for tape jobs:

* Veeam backup server
* Tape server
* Tape device and tapes
* Source host
