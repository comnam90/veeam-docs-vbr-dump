# Performing Instant Recovery to VMware vSphere Using Console


With Instant Recovery, you can recover different workloads from backups and register them as VMware vSphere VMs. For the list of backups that you can use for recovery, see [Supported Backup Types](instant_recovery.md#backups).

To perform Instant Recovery to VMware vSphere, use the Instant Recovery to VMware wizard.

1. [Ð¡heck prerequisites](instant_recovery_before_you_begin_vm.md).
2. [Launch the Instant Recovery wizard](instant_recovery_launch_vm.md).
3. [Select workloads](instant_recovery_vms_vm.md).
4. [Select a restore point](instant_recovery_point_vm.md).
5. [Select a restore mode](instant_recovery_mode_vm.md).
6. [Specify a destination for restored VMs](instant_recovery_destination_vm.md).
7. [Specify a destination for virtual disk updates](instant_recovery_datastore_vm.md).
8. [Configure a helper appliance](instant_recovery_helper_appliance_vm.md).
9. [Specify secure restore settings](instant_recovery_av_vm.md).
10. [Specify a restore reason](instant_recovery_reason_vm.md).
11. [Verify Instant Recovery settings](instant_recovery_verify_vm.md).
