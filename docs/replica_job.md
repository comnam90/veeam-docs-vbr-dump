# Creating Replication Jobs

In this article

To create VM replicas, you must configure a replication job. The replication job defines how, where and when to replicate VM data. One job can be used to process one VM or multiple VMs.

To create a replication job, use the New Replication Job wizard.

1. [Check prerequisites](replication_job_before_you_begin.md).
2. [Launch the New Replication Job wizard](replica_launch_vm.md).
3. [Specify a job name and description](replica_name_vm.md).
4. [Select VMs to replicate](replica_vms_vm.md).
5. [Specify a data source](replica_data_source_vm.md).
6. [Exclude objects from being replicated](replica_exclude_vm.md).
7. [Define the VM processing order](replica_order_vm.md).
8. [Specify a replica destination](replica_destination_vm.md).
9. [Create a network map table](replica_network_vm.md).
10. [Configure Re-IP rules](replica_reip_vm.md).
11. [Specify replication job settings](replica_settings_vm.md).
12. [Specify advanced replica settings](replica_advanced_vm.md).
13. [Specify data transfer settings](replica_data_transfer_vm.md).
14. [Configure seeding and mapping](replica_seeding_vm.md).
15. [Specify guest processing settings](replica_vss_vm.md).
16. [Define a job schedule](replica_schedule_vm.md).
17. [Finish working with the wizard](replica_review_vm.md).
