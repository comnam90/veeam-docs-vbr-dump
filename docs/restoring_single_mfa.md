# Restoring Single Mailbox, Folder or Item


To restore a single mailbox, folder or item, do the following:

1. [Launch the Restore wizard](launch_restore_mailbox_mfa.md).
2. [Specify a target mailbox](specify_target_mailbox_mfa.md).
3. [Select an authentication method](select_authentication_method_mfa.md).
4. [Log in to Microsoft 365](login_on_to_office_365_mfa.md).
5. [Specify a user account to impersonate with a public folder mailbox](impersonation_account.md).
6. [Specify a CAS server](mfa_cas_server.md).
7. [Specify a target folder](target_folder_mfa.md).
8. [Specify restore options](specify_restore_options_mfa.md).
9. [Review the operation summary](summary3.md).
