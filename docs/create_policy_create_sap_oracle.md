# Creating SAP on Oracle Backup Policy

In this article

To back up databases protected with Veeam Plug-In for SAP on Oracle, you must configure an application backup policy in Veeam Backup & Replication.

Before configuring an application backup policy, [check prerequisites](policy_sap_oracle_before.md). Then use the New Application Backup Policy wizard to define settings for the backup policy.

1. [Launch the New Application Backup Policy wizard](policy_sap_oracle_launch.md).
2. [Specify policy name and description](policy_sap_oracle_name.md).
3. [Select databases to back up](policy_sap_oracle_databases.md).
4. [Specify backup storage settings](policy_sap_oracle_repository.md).
5. [Specify advanced backup settings](policy_sap_oracle_advanced.md).
6. [Specify database processing settings](policy_sap_oracle_database_processing.md).
7. [Specify the backup schedule](policy_sap_oracle_schedule.md).
8. [Review backup policy settings](policy_sap_oracle_summary.md).
