# Adding S3 Compatible Object Storage


Before you add an S3 compatible object storage and S3 compatible object storage with data archiving, check [prerequisites](s3_compatible_limitations.md) and [required permissions](required_permissions.md#s3compatible). If you plan to use immutability, verify that you have configured the [necessary settings](immutability_os_enable.md#S3) for your S3 bucket. After that, use the New Object Storage Repository wizard.
