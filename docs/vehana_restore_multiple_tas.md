# Restoring to Another Server


To restore multiple SAP HANA tenant databases to another server, use the Restore wizard.

|  |
| --- |
| Important |
| Before restoring tenant databases to another server, read the [Considerations and Limitations](vehana_considerations.md) section. |

1. [Launch the Restore wizard](vehana_restore_multiple_tas_wizard.md).
2. [Select databases to restore](vehana_restore_multiple_tas_select_databases.md).
3. [Specify server credentials](vehana_restore_multiple_tas_specify_server_credentials.md).
4. [Specify system credentials](vehana_restore_multiple_tas_specify_system_credentials.md).
5. [Specify a point in time](vehana_restore_multiple_tas_specify_pit.md).
6. [Specify SYSTEM user password](vehana_restore_multiple_tas_specify_password.md).
