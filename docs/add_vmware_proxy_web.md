# Adding VMware Backup Proxies Using Web UI

In this article

Before adding a VMware backup proxy using the Veeam Backup & Replication web UI, [check prerequisites](backup_proxy_requirements.md). Then use the New VMware Proxy wizard.

1. [Launch the New VMware Proxy wizard](vmware_proxy_launch_web.md).
2. [Choose a server](vmware_proxy_server_web.md).
3. [Finish working with the wizard](vmware_proxy_summary_web.md).
