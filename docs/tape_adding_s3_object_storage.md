# Adding Amazon S3 Object Storage


Before you add an Amazon S3 object storage to the inventory of the virtual infrastructure, consider the following:

* The object storage meets requirements and limitations listed in the [Platform Support](https://helpcenter.veeam.com/docs/vbr/userguide/platform_support.html?ver=13#unstructured-data) section.

* If you plan to use dedicated proxy servers, make sure these components are added in the [Backup Infrastructure](https://helpcenter.veeam.com/docs/vbr/userguide/unstructured_data_backup_infrastructure.html?ver=13).

To add Amazon S3 object storage:

1. [Launch the New Object Storage wizard](tape_add_amazon_s3_launch.md).
2. [Specify account settings](tape_add_amazon_s3_account.md).
3. [Specify object storage processing](tape_add_amazon_s3_processing.md).
4. [Apply object storage settings](tape_add_amazon_s3_apply.md).
5. [Finish working with the wizard](tape_add_amazon_s3_summary.md).
