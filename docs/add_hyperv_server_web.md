# Adding Microsoft Hyper‑V Servers Using Web UI

In this article

This section describes how to add a Microsoft Hyper-V server as a backup infrastructure component.

Before you add a Microsoft Hyper-V server using Veeam Backup & Replication web UI, [check prerequisites](hv_server_before_begin.md). Then use the New Hyper-V Server wizard to add the server.

1. [Launch the New Hyper-V Server wizard](hv_server_launch_web.md).
2. [Specify a server name or address](hv_server_name_web.md).
3. [Specify credentials](hv_server_credentials_web.md).
4. [Select hosts](hv_server_hosts_web.md).
5. [Review components](hv_server_apply_web.md).
6. [Finish working with the wizard](hv_server_results_web.md).
