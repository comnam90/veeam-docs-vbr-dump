# Log Shipping Servers

In this article

Log shipping servers are dedicated components that Veeam Backup & Replication uses for backup of Microsoft SQL Server transaction logs, PostgreSQL WAL files and Oracle archive logs. For more information, see [Microsoft SQL Server Log Backup](sql_backup.md), [PostgreSQL WAL Files Backup](postgresql_backup.md) and [Oracle Log Backup](oracle_backup.md).
