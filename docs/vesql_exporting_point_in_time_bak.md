# Exporting Point-in-Time State

In this article

To export data as of a point-in-time state to the default location, do the following:

1. [Launch the Export wizard](vesql_export_wizard_bak_pit.md).

3. [Fine-tune the restore point](vesql_exporting_bak_point_2.md).
4. [Review the export summary](vesql_export_single_pit_bak_summary.md).
