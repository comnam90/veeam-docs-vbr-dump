In this article

You can use this scenario to restore a Microsoft SQL Server database to a new location, or to any location but with different settings.

To restore an Oracle database with custom settings, use the SQL Restore wizard.

1. [Launch the SQL Restore wizard](restore_mssql_custom_settings_launch.md).
2. [Specify a target server](restore_mssql_custom_settings_target.md).
3. [Specify AlwaysOn restore settings](restore_mssql_custom_settings_alwayson.md).
4. [Specify files location](restore_mssql_custom_settings_files.md).
