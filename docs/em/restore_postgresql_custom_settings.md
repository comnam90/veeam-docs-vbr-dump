# Restore with Custom Settings


You can use this scenario to restore a PostgreSQL instance to a new location, or to any location but with different settings.

To restore a PostgreSQL instance with custom settings, use the PostgreSQL Restore wizard.

1. [Launch the PostgreSQL Restore wizard](restore_postgresql_custom_settings_launch.md).
2. [Specify a target server](restore_postgresql_custom_settings_target.md).
3. [Specify restore settings](restore_postgresql_custom_settings_restore_settings.md).
4. [Specify location for database tablespaces](restore_postgresql_custom_settings_tablespaces.md).
