In this article

You can migrate recovered SMB file shares to the production environment.

To migrate a recovered file share, use the Migrate to Production wizard.

1. [Launch the Migrate to Production wizard](nas_migration_launch.md).
2. [Specify file share destination](nas_migration_destination.md).
3. [Specify restore options](nas_migration_restore_options.md).
4. [Configure switchover](nas_migration_configure_switchover.md).
5. [Review the migration settings](nas_migration_summary.md).
