In this article

Users with Host Administrator permissions can perform the following operations within the network settings:

* [Change the server name](hmc_configure_hostname.md)
* [Manage domain settings](hmc_configure_domain.md)
* [Configure network interfaces](hmc_configure_interfaces.md)
* [Configure HTTP/HTTPS proxies](hmc_configure_proxies.md)

Users with Security Officer permissions cannot configure network settings.
