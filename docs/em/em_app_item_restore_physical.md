In this article

If your Veeam Backup & Replication is integrated with the Server edition of Veeam Agent, and other [prerequisites](em_support_physical.md) are met, you can use the backups of the physical application servers to restore the necessary application items.

To restore application items, take the steps described in the following sections:

* [Restoring Microsoft Exchange Items](restore_msexchange.md)
* [Restoring Microsoft SQL Server Databases](restore_mssql.md)
* [Restoring Oracle Databases](restore_oracle.md)
* [Restoring PostgreSQL Instances](restore_postgresql.md)
