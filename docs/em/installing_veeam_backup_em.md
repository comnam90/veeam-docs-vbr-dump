# Installing Enterprise Manager


Before you install Veeam Backup Enterprise Manager, [check prerequisites](em_installation_byb.md). Then use the setup wizard to install the product.

1. [Start the setup wizard](em_setup_launch_wizard.md).
2. [Select Enterprise Manager as a product to install](em_setup_select_product.md).
3. [Read and accept the license agreements](em_setup_accept_license_agreement.md).
4. [Provide a license file](em_setup_install_license.md).
5. [Install missing software](em_setup_system_configuration_check.md).
6. [Review the default installation settings](em_setup_default_configuration.md).
7. [Specify a service account](em_setup_specify_service_account.md).
8. [Specify a database server](setup_database.md).
9. [Specify data locations](em_setup_data_locations.md).
10. [Specify service ports](em_setup_port_configuration.md).
11. [Begin installation](em_setup_ready_to_install.md).
