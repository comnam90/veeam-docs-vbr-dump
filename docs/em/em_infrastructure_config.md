As part of the Veeam Backup Enterprise Manager configuration process, you can perform the following tasks:

* [Collect data from backup servers](collecting_data_from_backup_servers.md)
* [View vCenter Servers and install Veeam plug-in for vSphere Client on necessary servers](viewing_vc_info.md)
* [Configure retention settings for index and history](configuring_retention_settings.md)
* [Configure Enterprise Manager accounts and roles](veeam_backup_em_roles.md)
* [Configure SAML authentication settings](veeam_backup_em_saml.md)
* [Configure notification settings](configuring_notification_settings.md)
* [Install TLS Certificates](updating_security_certificate.md)
* [Manage display languages](em_lang_manage.md)

To start working with Veeam Backup Enterprise Manager, you must perform initial configuration. For more information, see [Initial Configuration](initial_configuration.md).
