In this article

Veeam Backup Enterprise Manager allows you to restore Microsoft Hyper-V VMs to Microsoft Hyper-V. You can restore VMs from backups to the original location or a new location included in your restore scope.

For more information on entire VM restore of Microsoft Hyper-V VMs, see the [Entire VM Restore](https://helpcenter.veeam.com/docs/vbr/userguide/full_recovery.html?ver=13) section of the Veeam Backup & Replication User Guide.

To restore an entire VM, use the Entire VM Restore wizard.

1. [Launch the Entire VM Restore wizard](entire_vm_restore_hv_launch.md).
2. [Select a restore point](entire_vm_restore_hv_restorepoints.md).
3. [Select a recovery mode](entire_vm_restore_hv_restoremode.md).
4. [Specify destination settings for the recovered VM](entire_vm_restore_hv_destination.md).
5. [Specify a target datastore](entire_vm_restore_hv_datastore.md).
6. [Configure network mapping](entire_vm_restore_hv_network.md).
7. [Review the recovery settings](entire_vm_restore_hv_summary.md).
