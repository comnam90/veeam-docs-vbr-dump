In this article

Veeam Plug-In for Kasten allows you to view statistics on data recovery operations. You can view the information on the restore reason, the parameters of the restored instance, the logs of the restore session, and so on. For more information on how to review statistics, see the [Viewing Real-Time Statistics](https://helpcenter.veeam.com/docs/backup/vsphere/realtime_statistics.html?ver=120) section in the Veeam Backup & Replication User Guide.
