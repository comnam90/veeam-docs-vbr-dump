# Adding SMB (CIFS) Repositories

In this article

This section describes how to add an SMB (CIFS) share as a backup repository.

To add a backup repository, use the New Backup Repository wizard.

1. [Launch the New Backup Repository wizard](smb_repository_launch.md).
2. [Specify backup repository name and description](smb_repository_name.md).
3. [Specify shared folder settings](smb_repository_server.md).
4. [Configure backup repository settings](smb_repository_repository.md).
5. [Specify mount server settings](smb_repository_mount_server.md).
6. [Review properties and components](smb_repository_review.md).
7. [Apply backup repository settings](smb_repository_apply.md).
8. [Finish working with the wizard](smb_repository_finish.md).
