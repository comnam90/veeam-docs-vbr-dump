# Performing Failover

In this article

For more information on failover, see [Failover and Failback for Cloud Director](vcd_failover_failback.md) and [Failover](vcd_replica_failover.md).

To perform failover, do the following.

1. [Check prerequisites](vcd_failover_byb.md).
2. [Launch the vCloud Failover Wizard](vcd_failover_launch.md).
3. [Select vApp to Fail Over](vcd_failover_vapps_select.md).
4. [Select a restore point](vcd_failover_rp.md).
5. [Specify the failover reason](vcd_failover_reason.md).
6. [Finish working with the wizard](vcd_failover_finish.md).
