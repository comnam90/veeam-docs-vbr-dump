# Configuration File

In this article

The SapMaxDBBackintConfigTool configures SAP MaxDB settings and saves the settings to the Veeam Plug-In configuration file (veeam\_config.xml). The file is located in the /opt/veeam/VeeamPluginforSAPMaxDB directory on the machine where Veeam Plug-In is installed.
