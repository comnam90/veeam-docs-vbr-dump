# Creating SureBackup Job


To create a new SureBackup job, use the New SureBackup Job wizard.

1. [Launch the New SureBackup Job wizard](surebackup_job_launch_hv.md).
2. [Specify a job name and description](surebackup_job_name_hv.md).
3. [Select a virtual lab](surebackup_job_vlab_hv.md).
4. [Select an application group](surebackup_job_appgroup_hv.md).
5. [Link backup jobs to the SureBackup job](surebackup_job_joblink_hv.md).
6. [Specify recovery verification options and tests](surebackup_job_tests_hv.md).
7. [Specify additional job settings](surebackup_job_settings_hv.md).
8. [Define a job schedule](surebackup_job_schedule_hv.md).
9. [Review the job summary and finish working with the wizard](surebackup_job_review_hv.md).
