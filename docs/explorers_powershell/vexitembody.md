Contains details about a Microsoft Exchange item body.

| Property | Type | Description |
| --- | --- | --- |
| BodySize | String | Body size in kilobytes. |
| BodyType | VEXItemBodyType | Item body format. Possible values:   * Plain * Html * Rtf |
| Body | String | If the item body is in the plain text format, it is printed in this field. |
