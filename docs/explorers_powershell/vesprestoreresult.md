In this article

Contains details about a Veeam Explorer for Microsoft SharePoint item restore process.

| Property | Type | Description |
| --- | --- | --- |
| Title | String | Title of the restored item. |
| Type | String | SharePoint item type. |
| Path | String[] | URL of the SharePoint item. |
| Warnings | String[] | Array of warnings that appeared during the restore operation. |
| Error | String | Error that occurred during the restore operation. |
| IsSkipped | Bool | If True, the item was skipped during the restore operation. |
