In this article

Veeam Explorers PowerShell allows you to perform all operations that are available in Veeam Explorers user interface.

See the topics below to get help with performing recovery operations from PowerShell command line. These topics contain instructions on carrying out tasks with PowerShell and reference for relevant cmdlets.

* [Veeam Explorer for Microsoft Active Directory](veeam_explorer_for_active_directory.md)
* [Veeam Explorer for Microsoft SQL Server](veeam_explorer_for_microsoft_sql.md)
* [Veeam Explorer for Oracle](veeam_explorer_for_oracle.md)
* [Veeam Explorer for PostgreSQL](veeam_explorer_for_postgresql.md)
* [Veeam Explorer for SAP HANA](veeam_explorer_for_sap_hana.md)
* [Veeam Explorer for MongoDB](veeam_explorer_for_mongodb.md)
* [Veeam Explorer for Microsoft Exchange](veeam_explorer_for_microsoft_exchange.md)
* [Veeam Explorer for Microsoft SharePoint](veeam_explorer_for_microsoft_sharepoint.md)
* [Veeam Explorer for Microsoft OneDrive for Business](veeam_explorer_for_microsoft_onedrive.md)
* [Veeam Explorer for Microsoft Teams](veeam_explorer_for_microsoft_teams.md)

To browse through the list of all cmdlets available in the Veeam Explorers PowerShell modules, see [A-Z Cmdlets List](cmdlets.md).

For more information on Veeam Explorers PowerShell types, see [Veeam Explorers PowerShell Types](veeam_explorer_powershell_types.md). Information about enumerations is provided in parameter or property descriptions.
