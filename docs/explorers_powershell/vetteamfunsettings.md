In this article

Contains Microsoft Teams team settings related to stickers and memes.

| Property | Type | Description |
| --- | --- | --- |
| AllowGiphy | Bool | If True, users can use Giphy. |
| AllowStickersAndMemes | Bool | If True, users can include stickers and memes. |
| AllowCustomMemes | Bool | If True, users can include custom memes. |
| GiphyContentRating | VETGiphyContentRating | Giphy content rating. Possible values:   * Moderate * Strict |
