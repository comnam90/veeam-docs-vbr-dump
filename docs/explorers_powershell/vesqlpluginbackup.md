# VESQLPluginBackup


Contains details about backups created with Veeam Plug-in for Microsoft SQL Server.

| Property | Type | Description |
| --- | --- | --- |
| Id | GUID | Backup ID. |
| Name | String | Backup name. |
| ServerName | String | DNS name or IP address of the source server. |
