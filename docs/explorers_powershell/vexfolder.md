# VEXFolder


Contains details about a Microsoft Exchange mailbox folder.

| Property | Type | Description |
| --- | --- | --- |
| Name | String | Folder name. |
| FolderClass | String | Folder class. |
| Description | String | Folder description. |
| Parent | VEXFolder | Parent folder. |
