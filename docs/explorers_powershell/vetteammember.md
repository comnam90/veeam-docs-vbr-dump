In this article

Contains details about a Microsoft Teams team member.

| Property | Type | Description |
| --- | --- | --- |
| Id | GUID | Team member ID. |
| DisplayName | String | Team member display name. |
| Mail | String | Team member email address. |
| Roles | TeamRoles | Team member roles. Possible values:   * Owner * Member * Guest |
