In this article

Contains details about a Microsoft Exchange mailbox.

| Property | Type | Description |
| --- | --- | --- |
| Id | GUID | Mailbox ID. |
| Name | String | Mailbox name. |
| Email | String | Mailbox email address. |
| IsDeleted | Bool | If True, the mailbox is deleted. |
| IsArchive | Bool | If True, the mailbox is archived. |
