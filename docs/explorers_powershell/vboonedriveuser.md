# VBOOneDriveUser


Contains details about Microsoft OneDrive user.

| Property | Type | Description |
| --- | --- | --- |
| Name | String | Microsoft OneDrive user name. |
| Organization | [VBOOneDriveOrganization](vboonedriveorganization.md) | Microsoft OneDrive organization to which the user is added. |
