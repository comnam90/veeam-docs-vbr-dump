In this article

You can use the cmdlets for Veeam Explorer for Microsoft SharePoint to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Get-VESPCommand](get-vespcommand.md) | Returns a set of cmdlets available for Veeam Explorer for Microsoft SharePoint and Veeam Explorer for Microsoft OneDrive for Business. |
| [Get-VBRSiteCollection](get-vbrsitecollection.md) | Returns site collections for Microsoft SharePoint. |
| [Get-VBRSiteCollectionRestorePoint](get-vbrsitecollectionrestorepoint.md) | Returns restore points of backed-up machines with Microsoft SQL databases for the specified Microsoft SharePoint site collection. |
| [Start-VBRSharePointItemRestoreSession](start-vbrsharepointitemrestoresession.md) | Starts restore sessions to explore backed-up SharePoint databases and to perform operations with these databases. |
| [Get-VBRSharePointItemRestoreSession](get-vbrsharepointitemrestoresession.md) | Returns active restore sessions started to perform operations with backed-up Microsoft SharePoint databases. |
| [Stop-VBRSharePointItemRestoreSession](stop-vbrsharepointitemrestoresession.md) | Stops active restore sessions initiated to perform operations with Microsoft SharePoint databases. |
| [Start-VBOSharePointItemRestoreSession](start-vbosharepointitemrestoresession.md) | Starts restore sessions to explore backed-up SharePoint items and to perform operations with these items. |
| [Get-VBOSharePointItemRestoreSession](get-vbosharepointitemrestoresession.md) | Returns active restore sessions started to perform operations with backed-up SharePoint items. |
| [Stop-VBOSharePointItemRestoreSession](stop-vbosharepointitemrestoresession.md) | Stops active restore sessions. |
| [Get-VESPDatabase](get-vespdatabase.md) | Returns backed-up Microsoft SharePoint databases. |
| [Get-VESPOrganization](get-vesporganization.md) | Returns SharePoint organizations added to Veeam Backup for Microsoft 365. |
| [Get-VESPDocument](get-vespdocument.md) | Returns SharePoint documents. |
| [Get-VESPDocumentVersion](get-vespdocumentversion.md) | Returns other versions of a SharePoint document. |
| [Get-VESPItem](get-vespitem.md) | Returns SharePoint items. |
| [Get-VESPItemAttachment](get-vespitemattachment.md) | Returns a specified SharePoint item attachment. |
| [Get-VESPItemVersion](get-vespitemversion.md) | Returns other versions of a SharePoint item. |
| [Get-VESPSite](get-vespsite.md) | Returns SharePoint sites. |
| [Get-VESPList](get-vesplist.md) | Returns SharePoint lists. |
| [Get-VESPDocumentLibrary](get-vespdocumentlibrary.md) | Returns SharePoint document libraries. |
| [Export-VESPItem](export-vespitem.md) | Exports SharePoint items. |
| [Restore-VESPItem](restore-vespitem.md) | Restores SharePoint items. |
| [Send-VESPItem](send-vespitem.md) | Sends SharePoint items as attachments to emails. |
| [Save-VESPItem](save-vespitem.md) | Saves SharePoint items. |
| [Get-VESPMailSettings](get-vespmailsettings.md) | Returns Veeam Explorer for Microsoft SharePoint email settings. |
| [Set-VESPMailSettings](set-vespmailsettings.md) | Modifies Veeam Explorer for Microsoft SharePoint email settings. |
| [Get-VESPSmtpSettings](get-vespsmtpsettings.md) | Returns SMTP settings. |
| [Set-VESPSmtpSettings](set-vespsmtpsettings.md) | Modifies SMTP settings. |
| [Get-VESPExtendedLogging](get-vespextendedlogging.md) | Returns a state of the extended logging mode for Veeam Explorer for Microsoft SharePoint. |
| [Enable-VESPExtendedLogging](enable-vespextendedlogging.md) | Enables the extended logging mode for Veeam Explorer for Microsoft SharePoint. |
| [Disable-VESPExtendedLogging](disable-vespextendedlogging.md) | Disables the extended logging mode for Veeam Explorer for Microsoft SharePoint. |
