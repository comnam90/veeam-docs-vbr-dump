# VEADDomain


Contains information about an Active Directory domain.

| Property | Type | Description |
| --- | --- | --- |
| Id | GUID | Domain ID. |
| Name | String | Domain name. |
| Session | [VEADRestoreSession](veadrestoresession.md) | Details about the restore session. |
