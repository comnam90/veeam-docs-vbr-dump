In this article

You can use the cmdlets for Veeam Explorer for Microsoft Exchange to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Get-VEXCommand](get-vexcommand.md) | Returns a set of cmdlets available for Veeam Explorer for Microsoft Exchange. |
| [Start-VBRExchangeItemRestoreSession](start-vbrexchangeitemrestoresession.md) | Starts restore sessions to explore backed-up Exchange databases and to perform operations with these databases. |
| [Get-VBRExchangeItemRestoreSession](get-vbrexchangeitemrestoresession.md) | Returns active restore sessions started to perform operations with backed-up Microsoft Exchange databases. |
| [Stop-VBRExchangeItemRestoreSession](stop-vbrexchangeitemrestoresession.md) | Stops active restore sessions initiated to perform operations with backed-up Microsoft Exchange databases. |
| [Start-VBOExchangeItemRestoreSession](start-vboexchangeitemrestoresession.md) | Starts restore sessions to explore backed-up Microsoft Exchange items and to perform operations with these items. |
| [Get-VBOExchangeItemRestoreSession](get-vboexchangeitemrestoresession.md) | Returns active restore sessions started to perform operations with backed-up Microsoft Exchange items. |
| [Stop-VBOExchangeItemRestoreSession](stop-vboexchangeitemrestoresession.md) | Stops active restore sessions initiated to perform operations with backed-up Microsoft Exchange items. |
| [Get-VEXDatabase](get-vexdatabase.md) | Returns Microsoft Exchange mailbox databases. |
| [Get-VEXMailbox](get-vexmailbox.md) | Returns Microsoft Exchange mailboxes. |
| [Test-VEXMailboxResolution](test-vexmailboxresolution.md) | Tests the availability of Microsoft Exchange mailboxes before a bulk mailbox restore. |
| [Get-VEXFolder](get-vexfolder.md) | Returns folders added to Microsoft Exchange mailboxes. |
| [Get-VEXItem](get-vexitem.md) | Returns items added to Microsoft Exchange mailboxes. |
| [Export-VEXItem](export-vexitem.md) | Exports Exchange organization backups. |
| [Restore-VEXItem](restore-vexitem.md) | Restores backups of Exchange organization mailboxes. |
| [Send-VEXItem](send-vexitem.md) | Sends Exchange mailbox items as attachments to emails. |
| [Get-VEXExtendedLogging](get-vexextendedlogging.md) | Returns a state of the extended logging mode for Veeam Explorer for Microsoft Exchange. |
| [Enable-VEXExtendedLogging](enable-vexextendedlogging.md) | Enables the extended logging mode for Veeam Explorer for Microsoft Exchange. |
| [Disable-VEXExtendedLogging](disable-vexextendedlogging.md) | Disables the extended logging mode for Veeam Explorer for Microsoft Exchange. |
| [Get-VEXMailSettings](get-vexmailsettings.md) | Returns Veeam Explorer for Microsoft Exchange email settings. |
| [Set-VEXMailSettings](set-vexmailsettings.md) | Modifies Veeam Explorer for Microsoft Exchange email settings. |
| [Get-VEXSmtpSettings](get-vexsmtpsettings.md) | Returns SMTP settings. |
| [Set-VEXSmtpSettings](set-vexsmtpsettings.md) | Modifies SMTP settings. |
| [Get-VEXUseSecureLdap](get-vexusesecureldap.md) | Returns the state of the secure LDAP mode for Veeam Explorer for Microsoft Exchange. |
| [Enable-VEXUseSecureLdap](enable-vexusesecureldap.md) | Enables the secure LDAP mode for Veeam Explorer for Microsoft Exchange. |
| [Disable-VEXUseSecureLdap](disable-vexusesecureldap.md) | Disables the secure LDAP mode for Veeam Explorer for Microsoft Exchange. |
