# VEORDatabase


Contains details about an Oracle database.

| Property | Type | Description |
| --- | --- | --- |
| GlobalName | String | Global database name, in the DB\_NAME.DB\_DOMAIN format. |
| Home | String | Oracle home path of the database on the target server. |
