# VEMDBDatabase


Contains details about a MongoDB database.

| Property | Type | Description |
| --- | --- | --- |
| Name | GUID | Database name. |
| Sharded | Bool | If True, the database belongs to a sharded cluster. |
| Size | Ulong | Database size in bytes. |
