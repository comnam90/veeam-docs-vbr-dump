In this article

Contains details about a Veeam Explorer for Microsoft Exchange item restore process.

| Property | Type | Description |
| --- | --- | --- |
| Title | String | Title of the restored item. |
| SourcePath | String | Source path of the restored item. |
| TargetPath | String | Target path of the restored item. |
| Type | String | Exchange item type. |
| Error | String | Error that occurred during the restore operation. |
