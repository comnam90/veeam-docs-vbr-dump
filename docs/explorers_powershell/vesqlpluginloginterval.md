In this article

Contains the available restore period for a Microsoft SQL Server database.

| Property | Type | Description |
| --- | --- | --- |
| From | DateTime | Date and time of the oldest available database state on the restore point. |
| To | DateTime | Date and time of the latest available database state on the restore point. |
