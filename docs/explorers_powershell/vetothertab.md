# VETOtherTab


Contains details about a Microsoft Teams channel tab.

| Property | Type | Description |
| --- | --- | --- |
| Id | String | Channel tab ID. |
| DisplayName | String | Channel tab display name. |
| ContentUrl | String | Path to the object published on the tab. |
