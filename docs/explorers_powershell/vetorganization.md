In this article

Contains details about a Microsoft Teams organization.

| Property | Type | Description |
| --- | --- | --- |
| Name | String | Microsoft Teams organization name, in the following format:  <organization\_name> (<backup\_job\_name>) (<date\_and\_time\_when\_restore\_session\_was\_created>)  For example:  qwbs.onmicrosoft.com (Backup job) (2/27/2025 10:41:17 AM)  Note that the backup job name is only displayed if you have specified the job name when starting the restore session. |
