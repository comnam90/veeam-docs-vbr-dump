# VESQLDatabaseFile


Contains details about Microsoft SQL Server database files.

| Property | Type | Description |
| --- | --- | --- |
| Path | String | Path to the database file. |
| Type | VESQLDatabaseFileType | Database file type. Possible values:   * Primary * Secondary |
