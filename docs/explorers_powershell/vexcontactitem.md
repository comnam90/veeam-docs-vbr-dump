In this article

Contains details about a Microsoft Exchange contact item.

| Property | Type | Description |
| --- | --- | --- |
| Address | String | Contact address. |
| BusinesPhone | String | Contact business phone. |
| Company | String | Company with whom the contact is affiliated. |
| DisplayAs | String | Contact alias. |
| Email | String | Contact email address. |
| Fax | String | Contact fax number. |
| FileAs | String | Default keyword String assigned to the contact when it is filed. |
| FullName | String | Full name of the contact. |
| HomePhone | String | Home phone number of the contact. |
| IMAddress | String | Microsoft Instant Messenger address of the contact. |
| JobTitle | String | Contact job title. |
| Mobile | String | Mobile phone number of the contact. |
| WebPage | String | Contact webpage. |
| ItemClass | String | Exchange item class. |
