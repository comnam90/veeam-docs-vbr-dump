# Veeam Explorer for Microsoft SQL Server


You can use the cmdlets for Veeam Explorer for Microsoft SQL Server to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Get-VESQLCommand](get-vesqlcommand.md) | Returns a set of cmdlets available for Veeam Explorer for Microsoft SQL Server. |
| [Start-VESQLRestoreSession](start-vesqlrestoresession.md) | Starts a restore session to explore backed-up Microsoft SQL databases and to perform operations with these databases. |
| [Get-VESQLRestoreSession](get-vesqlrestoresession.md) | Returns active restore sessions started to perform operations with backed-up Microsoft SQL Server databases. |
| [Stop-VESQLRestoreSession](stop-vesqlrestoresession.md) | Stops an active restore session initiated to perform operations with Microsoft SQL Server databases. |
| [Get-VESQLDatabase](get-vesqldatabase.md) | Returns backed-up Microsoft SQL Server databases. |
| [Get-VESQLDatabaseRestoreInterval](get-vesqldatabaserestoreinterval.md) | Returns details on the available restore period for a backed-up Microsoft SQL database. |
| [Get-VESQLDatabaseFile](get-vesqldatabasefile.md) | Returns full file names for a backed-up Microsoft SQL Server database. |
| [Restore-VESQLDatabase](restore-vesqldatabase.md) | Restores a backed-up Microsoft SQL Server database. |
| [New-VESQLIRSwitchOverOptions](new-vesqlirswitchoveroptions.md) | Defines the switchover option that you can apply to the instant recovery session of a Microsoft SQL Server database. |
| [Get-VESQLIRDatabase](get-vesqlirdatabase.md) | Returns Microsoft SQL Server databases that are published within an instant recovery session. |
| [Set-VESQLIRDatabase](set-vesqlirdatabase.md) | Modifies switchover settings for a specified Microsoft SQL Server database. |
| [Restore-VESQLIRDatabase](restore-vesqlirdatabase.md) | Performs instant recovery of a backed-up Microsoft SQL Server database. |
| [Switch-VESQLIRDatabase](switch-vesqlirdatabase.md) | Performs switchover of a Microsoft SQL Server database published within an instant recovery session. |
| [Stop-VESQLInstantRecovery](stop-vesqlinstantrecovery.md) | Stops an active instant recovery session for a Microsoft SQL Server database. |
| [Export-VESQLDatabase](export-vesqldatabase.md) | Exports a backed-up Microsoft SQL Server database. |
| [Publish-VESQLDatabase](publish-vesqldatabase.md) | Publishes a backed-up Microsoft SQL Server database. |
| [Get-VESQLPublishedDatabase](get-vesqlpublisheddatabase.md) | Returns published Microsoft SQL Server databases. |
| [Unpublish-VESQLDatabase](unpublish-vesqldatabase.md) | Unpublishes Microsoft SQL Server databases from the target server. |
| [Export-VESQLPublishedDatabase](export-vesqlpublisheddatabase.md) | Exports a published Microsoft SQL Server database. |
| [Get-VESQLPluginBackup](get-vesqlpluginbackup.md) | Returns backups created with Veeam Plug-in for Microsoft SQL Server. |
| [Start-VESQLPluginRestoreSession](start-vesqlpluginrestoresession.md) | Starts a restore session to explore and restore databases backed up with Veeam Plug-in for Microsoft SQL Server. |
| [Get-VESQLPluginRestoreSession](get-vesqlpluginrestoresession.md) | Returns active restore sessions started to explore and restore databases backed up with Veeam Plug-in for Microsoft SQL Server. |
| [Stop-VESQLPluginRestoreSession](stop-vesqlpluginrestoresession.md) | Stops an active restore session started to explore and restore databases backed up with Veeam Plug-in for Microsoft SQL Server. |
| [Get-VESQLPluginDatabase](get-vesqlplugindatabase.md) | Returns databases backed up with Veeam Plug-in for Microsoft SQL Server. |
| [Get-VESQLPluginRestorePoint](get-vesqlpluginrestorepoint.md) | Returns restore points for a database backed up with Veeam Plug-in for Microsoft SQL Server. |
| [Get-VESQLPluginLogInterval](get-vesqlpluginloginterval.md) | Returns details on the available restore period for a database backed up with Veeam Plug-in for Microsoft SQL Server. |
| [Get-VESQLPluginDatabaseFile](get-vesqlplugindatabasefile.md) | Returns database files for a database backed up with Veeam Plug-in for Microsoft SQL Server. |
| [Start-VESQLPluginDatabaseRestore](start-vesqlplugindatabaserestore.md) | Restores a database backed up with Veeam Plug-in for Microsoft SQL Server. |
| [Get-VESQLPluginDatabaseRestore](get-vesqlplugindatabaserestore.md) | Returns active restore jobs for a database backed up with Veeam Plug-in for Microsoft SQL Server. |
| [Stop-VESQLPluginDatabaseRestore](stop-vesqlplugindatabaserestore.md) | Stops a restore job for a database backed up with Veeam Plug-in for Microsoft SQL Server. |
| [Get-VESQLExtendedLogging](get-vesqlextendedlogging.md) | Returns the state of the extended logging mode for Veeam Explorer for Microsoft SQL Server. |
| [Enable-VESQLExtendedLogging](enable-vesqlextendedlogging.md) | Enables the extended logging mode for Veeam Explorer for Microsoft SQL Server. |
| [Disable-VESQLExtendedLogging](disable-vesqlextendedlogging.md) | Disables the extended logging mode for Veeam Explorer for Microsoft SQL Server. |
