# VESPDocumentLibrary


Contains details about a Microsoft SharePoint document library.

| Property | Type | Description |
| --- | --- | --- |
| Name | String | Document library name. |
| Description | String | Document library description. |
| Url | Url | Document library URL. |
| CreationTime | DateTime | Date and time when the document library was created. |
| ItemsCount | Ulong | Number of items in the document library. |
