# VEORRMANRestoreSession


Contains details about a Veeam Explorer for Oracle restore session started to explore and restore Oracle databases backed up with Veeam Plug-in for Oracle RMAN.

| Property | Type | Description |
| --- | --- | --- |
| ServerName | String | DNS name or IP address of the source server. |
| StartTime | DateTime | Date and time when the restore session was started. |
