# Veeam Explorer for Microsoft Teams


You can use the cmdlets for Veeam Explorer for Microsoft Teams to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Get-VETCommand](get-vetcommand.md) | Returns a set of cmdlets available for Veeam Explorer for Microsoft Teams. |
| [Start-VBOTeamsItemRestoreSession](start-vboteamsitemrestoresession.md) | Starts restore sessions to explore backed-up Microsoft Teams items and perform operations with these items. |
| [Get-VBOTeamsItemRestoreSession](get-vboteamsitemrestoresession.md) | Returns active restore sessions started to perform operations with backed-up Microsoft Teams items. |
| [Stop-VBOTeamsItemRestoreSession](stop-vboteamsitemrestoresession.md) | Stops active restore sessions initiated to perform operations with backed-up Microsoft Teams items. |
| [Get-VETOrganization](get-vetorganization.md) | Returns Microsoft Teams organizations added to Veeam Backup for Microsoft 365. |
| [Get-VETTeam](get-vetteam.md) | Returns Microsoft Teams teams. |
| [Get-VETTeamMember](get-vetteammember.md) | Returns Microsoft Teams team members. |
| [Get-VETChannel](get-vetchannel.md) | Returns Microsoft Teams team channels. |
| [Get-VETPost](get-vetpost.md) | Returns Microsoft Teams team channel posts. |
| [Get-VETFile](get-vetfile.md) | Returns files published in Microsoft Teams team channels. |
| [Get-VETOtherTab](get-vetothertab.md) | Returns Microsoft Teams team channel tabs. |
| [Export-VETPost](export-vetpost.md) | Exports Microsoft Teams team channel posts to a file in the HTML format. |
| [Restore-VETItem](restore-vetitem.md) | Restores Microsoft Teams items from backup. |
| [Send-VETItem](send-vetitem.md) | Sends Microsoft Teams items as email attachments. |
| [Save-VETItem](save-vetitem.md) | Saves Microsoft Teams items. |
| [Get-VETExtendedLogging](get-vetextendedlogging.md) | Returns a state of the extended logging mode for Veeam Explorer for Microsoft Teams. |
| [Enable-VETExtendedLogging](enable-vetextendedlogging.md) | Enables the extended logging mode for Veeam Explorer for Microsoft Teams. |
| [Disable-VETExtendedLogging](disable-vetextendedlogging.md) | Disables the extended logging mode for Veeam Explorer for Microsoft Teams. |
| [Get-VETMailSettings](get-vetmailsettings.md) | Returns Veeam Explorer for Microsoft Teams email settings. |
| [Set-VETMailSettings](set-vetmailsettings.md) | Modifies Veeam Explorer for Microsoft Teams email settings. |
| [Get-VETSmtpSettings](get-vetsmtpsettings.md) | Returns SMTP settings. |
| [Set-VETSmtpSettings](set-vetsmtpsettings.md) | Modifies SMTP settings. |
