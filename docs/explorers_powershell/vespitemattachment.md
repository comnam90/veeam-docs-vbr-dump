Contains details about a Microsoft SharePoint attachment.

| Property | Type | Description |
| --- | --- | --- |
| Item | [VESPItem](vespitem.md) | Details about the item to which the attachment is associated. |
| Name | String | Attachment name. |
| Size | Long | Attachment size in kilobytes. |
| URL | String | Attachment URL. |
