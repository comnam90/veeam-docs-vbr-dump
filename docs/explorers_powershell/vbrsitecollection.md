# VBRSiteCollection


Contains details about a Sharepoint site collection.

| Property | Type | Description |
| --- | --- | --- |
| URL | String | Site URL. |
| RestorePointID | GUID | Restore point ID. |
| Id | GUID | Site collection ID. |
