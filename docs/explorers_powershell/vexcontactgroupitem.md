# VEXContactGroupItem


Contains details about a Microsoft Exchange contact group item.

| Property | Type | Description |
| --- | --- | --- |
| Name | String | Contact group name. |
| Contacts | [VEXItemRecipient](vexitemrecipient.md)[] | Array of Microsoft Exchange recipients. |
