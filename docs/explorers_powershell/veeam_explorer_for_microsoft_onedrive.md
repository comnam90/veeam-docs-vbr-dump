In this article

You can use the cmdlets for Veeam Explorer for Microsoft OneDrive for Business to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Start-VEODRestoreSession](start-veodrestoresession.md) | Starts restore sessions to explore backed-up OneDrive items and to perform operations with these items. |
| [Get-VEODRestoreSession](get-veodrestoresession.md) | Returns restore sessions. |
| [Stop-VEODRestoreSession](stop-veodrestoresession.md) | Stops active restore sessions. |
| [Get-VEODOrganization](get-veodorganization.md) | Returns Microsoft OneDrive organizations added to Veeam Backup for Microsoft 365. |
| [Get-VEODUser](get-veoduser.md) | Returns OneDrive users. |
| [Get-VEODDocument](get-veoddocument.md) | Returns OneDrive documents. |
| [Get-VEODDocumentVersion](get-veoddocumentversion.md) | Returns other versions of a OneDrive document. |
| [Restore-VEODDocument](restore-veoddocument.md) | Restores OneDrive documents. |
| [Send-VEODDocument](send-veoddocument.md) | Sends OneDrive documents as attachments to emails. |
| [Save-VEODDocument](save-veoddocument.md) | Saves OneDrive documents. |
| [Get-VEODMailSettings](get-veodmailsettings.md) | Returns Veeam Explorer for Microsoft OneDrive for Business email settings. |
| [Set-VEODMailSettings](set-veodmailsettings.md) | Modifies Veeam Explorer for Microsoft OneDrive for Business email settings. |
| [Get-VEODSmtpSettings](get-veodsmtpsettings.md) | Returns SMTP settings. |
| [Set-VEODSmtpSettings](set-veodsmtpsettings.md) | Modifies SMTP settings. |
