In this article

Contains details about a Veeam Explorer for Microsoft SharePoint restore session started with a restore point created by Veeam Backup for Microsoft 365.

| Property | Type | Description |
| --- | --- | --- |
| Id | GUID | Restore session ID. |
| StartTime | DateTime | Date and time when the restore session was started. |
