# VEHANADatabase


Contains details about a backed-up SAP HANA database.

| Property | Type | Description |
| --- | --- | --- |
| System | [VEHANASystem](vehanasystem.md) | SAP HANA system. |
| Name | String | Database name. |
| Type | VEHANADatabaseType | Database type. Possible values:   * Tenant * System |
