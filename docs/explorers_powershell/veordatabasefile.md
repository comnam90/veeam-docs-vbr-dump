# VEORDatabaseFile


Contains details about an Oracle database file.

| Property | Type | Description |
| --- | --- | --- |
| Path | String | Path to the database file. |
| Type | VEORDatabaseFileType | Database file type. Possible values:   * Controlfile * Datafile * Logfile * Tempfile * FastRecoveryArea |
