Contains details about a Microsoft Exchange discussion item.

| Property | Type | Description |
| --- | --- | --- |
| Title | String | Discussion title. |
| From | String | Name of the discussion author. |
| PostedOn | DateTime | Date and time the discussion was started. |
| ItemClass | String | Exchange item class. |
