# Restoring Sites

In this article

You can restore Microsoft SharePoint sites to the same or a different location.

To restore a Microsoft SharePoint site, take the following steps:

1. [Launch the Restore wizard](vesp_vbr_restore_sites_wizard.md).
2. [Specify a target SharePoint site](vesp_vbr_restore_sites_specify_target_site.md).
3. [Specify restore options](vesp_vbr_restore_sites_specify_restore_options.md).
4. [Review the operation summary](summary18.md).
