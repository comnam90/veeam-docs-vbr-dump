# Managing Backups in Veeam Backup & Replication


In the Veeam Backup & Replication console, backups created by Veeam Plug-Ins are displayed in the Backups node of the Home view. In the working area, backups created by Veeam Plug-In for SAP on Oracle are listed under the SAP on Oracle node.

You can use the Veeam Backup & Replication console to perform the following operations with Veeam Plug-In backups:

* [Create a recovery token for a backup](plugins_sap_oracle_backup_management_token.md)
* [Remove a backup from configuration](plugins_sap_oracle_backup_management_remove.md)
* [Delete a backup from the backup repository](plugins_sap_oracle_backup_management_delete.md)
