# Updating Veeam Backup & Replication on Windows


Apart from major version releases of Veeam Backup & Replication (for example, 13.0.1 (build 13.0.1.180)), Veeam Software provides updates (for example, update 13.0.1 P1 (build 13.0.1.1071) and 13.0.1 P2 (build 13.0.1.2067) for v13.0.1). Updates contain bug fixes, performance enhancements and introduce new features.

|  |
| --- |
| Note |
| During the update procedure, Veeam Backup & Replication may recreate Windows Firewall rules with default settings. If you have manually modified Windows Firewall rules before installing a new cumulative patch, reapply those modifications after the update. |
