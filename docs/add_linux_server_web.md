# Adding Linux Servers Using Web UI

In this article

This section describes how to add a Linux server as a backup infrastructure component.

Before you add a Linux server, [check prerequisites](linux_server_before_begin.md). To add a Linux server using the Veeam Backup & Replication web UI, use the New Linux Server wizard.

1. [Launch the New Linux Server wizard](linux_server_launch_web.md).
2. [Specify a server name or address](linux_server_name_web.md).
3. [Specify credentials and SSH settings](linux_server_ssh_web.md).
4. [Review components](linux_server_review_web.md).
5. [Finish working with the wizard](linux_server_summary_web.md).
