# Creating SureBackup Job

In this article

To create a new SureBackup job, use the New SureBackup Job wizard.

1. [Launch the New Sure Backup Job wizard](surebackup_job_launch_vm.md).
2. [Specify a job name and description](surebackup_job_name_vm.md).
3. [Select a virtual lab](surebackup_job_vlab_vm.md).
4. [Select an application group](surebackup_job_appgroup_vm.md).
5. [Link a backup or replica job to the SureBackup job](surebackup_job_joblink_vm.md).
6. [Specify recovery verification options and tests](surebackup_job_tests_vm.md).
7. [Specify additional job settings](surebackup_job_settings_vm.md).
8. [Define a job schedule](surebackup_job_schedule_vm.md).
9. [Review the job summary and finish working with wizard](surebackup_job_review_vm.md).
