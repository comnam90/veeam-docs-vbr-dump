# Veeam Explorer for SAP HANA


Veeam Explorer for SAP HANA allows you to restore system or tenant SAP HANA databases from backups created by application backup policies managed by Veeam Backup & Replication and backup jobs managed by a standalone Veeam Plug-In for SAP HANA.
