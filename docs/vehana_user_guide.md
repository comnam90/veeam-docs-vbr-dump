# Veeam Explorer for SAP HANA


Veeam Explorer for SAP HANA allows you to restore system or tenant SAP HANA databases from database-level backups — created by application backup policies managed by Veeam Backup & Replication or backup jobs managed by a standalone Veeam Plug-In for SAP HANA.
