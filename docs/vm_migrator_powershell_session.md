# Step 1. Start Veeam PowerShell Session


To use Veeam VM Migrator Utility, start Veeam PowerShell session in one of the following ways:

* For Windows: in the main menu of Veeam Backup & Replication, select Console > PowerShell.
* For Linux: start a PowerShell session on your machine using the pwsh command in shell.
