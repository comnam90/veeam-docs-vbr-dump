# Managing Veeam Plug-In

In this article

You can use the backup console to manage the following Veeam Plug-Ins on a specific computer in the inventory:

* Veeam Plug-In for Oracle RMAN
* Veeam Plug-In for SAP HANA
* Veeam Plug-In for SAP on Oracle

You can perform the following management tasks for Veeam Plug-In:

* [Installing Veeam Plug-In](protected_computers_install.md).
* [Upgrading Veeam Plug-In](protected_computers_upgrade.md).
* [Uninstalling Veeam Plug-In](protected_computers_uninstall.md).
