# Creating Backup to Tape Jobs

In this article

Before you create a backup to tape job, [check the prerequisites](tape_before_you_begin.md).

To archive backups to tape, you should create a backup to tape job using the New Backup to Tape Job wizard. With this wizard, you can create a backup to tape job or a GFS job.

To create a backup to tape job, follow the next steps:

1. [Launch the New Backup To Tape Job wizard](backup_to_tape_launch.md).
2. [Specify job name and description](backup_to_tape_name.md).
3. [Choose backups to archive](backup_to_tape_backups.md).
4. [Choose a media pool for full backups](backup_to_tape_pools.md).
5. [Choose a media pool for incremental backups](backup_to_tape_incremental_pool.md).
6. [Specify archiving options](backup_to_tape_options.md).
7. [Specify advanced job settings](backup_to_tape_advanced.md).
8. [Define the job schedule](backup_to_tape_schedule.md).
9. [Finish working with the wizard](backup_to_tape_review.md).
