# Connecting HPE Morpheus VM Essentials Server


An HPE Morpheus VM Essentials manager allows the backup server to access HPE Morpheus VM Essentials resources such as VMs, datastores and networks. After you add an HPE Morpheus VM Essentials manager to the backup infrastructure, you will be able to deploy workers and to manage data protection tasks for HPE Morpheus VM Essentials VMs.
