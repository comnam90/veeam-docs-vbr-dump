# Adding Linux Servers Using Console


This section describes how to add a Linux server as a backup infrastructure component.

Before you add a Linux server, [check prerequisites](linux_server_before_begin.md). To add a Linux server using the Veeam Backup & Replication console, use the New Linux Server wizard.

1. [Launch the New Linux Server wizard](linux_server_launch.md).

1. [Specify a server name or address](linux_server_name.md).
2. [Specify credentials and SSH settings](linux_server_ssh.md).
3. [Review components](linux_server_review.md).
4. [Apply settings](linux_server_apply.md).
5. [Finish working with the wizard](linux_server_summary.md).
