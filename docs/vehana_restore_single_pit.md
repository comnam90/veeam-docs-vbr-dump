# Restoring Point-in-Time State

In this article

To restore a point-in-time state of an SAP HANA database, use the Restore wizard.

1. [Launch the Restore wizard](vehana_restore_single_pit_wizard.md).
2. [Specify server credentials](vehana_restore_single_pit_specify_server_credentials.md).
3. [Specify system credentials](vehana_restore_single_pit_specify_system_credentials.md).
4. [Specify a recovery type](vehana_restore_single_pit_specify_recovery_type.md).
5. [Specify recovery catalog options](vehana_restore_single_pit_specify_recovery_catalog.md).
6. [Select a backup](vehana_restore_single_pit_select_backup.md).
7. [Specify SYSTEM user password](vehana_restore_single_pit_specify_password.md).
