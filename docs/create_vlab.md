# Creating Virtual Lab

In this article

Before you create a new virtual lab, [check prerequisites](vlab_before_you_begin_vm.md). Then use the New Virtual Lab wizard to create a virtual lab.

1. [Launch the New Virtual Lab wizard](launch_new_virtual_lab_wizard.md).
2. [Specify a virtual lab name and description](vlab_name_vm.md).
3. [Select a host](vlab_host_vm.md).
4. [Select a datastore](vlab_datastore_vm.md).
5. [Set up a proxy appliance](vlab_proxy_vm.md).
6. [Select the networking mode](vlab_network_vm.md).
7. [Create isolated networks](vlab_isolated_network_vm.md).
8. [Specify network settings](vlab_net_settings_vm.md).
9. [Specify static IP mapping rules](vlab_ip_vm.md).
10. [Apply parameters](vlab_apply_vm.md).
