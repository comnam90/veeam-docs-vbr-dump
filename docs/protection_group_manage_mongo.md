# Managing Protection Group for MongoDB

In this article

In Veeam Backup & Replication, computers with MongoDB are organized into a protection group for applications. You can perform the following operations with this type of protection groups:

* [Add protection group to the existing application backup policy](mongo_protection_group_job.md).
* [Add protection group to the existing Veeam Agent backup job](mongo_protection_group_add_veeam_agent_job.md).
* [Edit protection group settings](mongo_protection_group_edit.md).
* [Rescan a protection group](mongo_protection_group_rescan.md).
* [Assign location to a protection group](mongo_protection_group_location.md).
* [Disable a protection group](mongo_protection_group_disable.md).
* [Remove a protection group](mongo_protection_group_remove.md).
