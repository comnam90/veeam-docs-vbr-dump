# Adding HPE Alletra Storage MP B10000, 9000, Primera, 3PAR

In this article

Before you add an HPE storage system to the backup infrastructure, check [Planning and Preparation](storage_limitations.md). To add an HPE Alletra Storage MP B10000, 9000, Primera, 3PAR storage system to the backup infrastructure, use the New Alletra/Primera Storage wizard.

1. [Enable HPE Web Services API Server](storage_configure_enable_web.md).
2. [Launch the wizard](hpe_add_select_type.md).
3. [Specify HPE Web Services API address and storage role](hp_3par_add_name.md).
4. [Specify credentials](hp_3par_add_credentials.md).
5. [Specify VMware access options](hp_3par_add_options.md).
6. [Specify Veeam Agent access options](hp_3par_agents_access.md).
7. [Apply settings](hp_3par_add_apply.md).
8. [Finish working with the wizard](hp_3par_add_summary.md).
