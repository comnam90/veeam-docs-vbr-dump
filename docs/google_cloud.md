# Google Cloud

In this article

|  |
| --- |
| Note |
| Google Cloud Plug-In for Veeam Backup & Replication is available for Microsoft Windows-based backup servers. |

Veeam Backup & Replication with Google Cloud Plug-In for Veeam Backup & Replication allows you to add Google Cloud backup appliances to Veeam Backup & Replication and manage data protection and recovery operations for all these appliances from a single Veeam Backup & Replication console. You can use Veeam Backup & Replication with the installed plug-in to perform the following operations:

* Create image-level backups and cloud-native snapshots of VM instances.
* Create image-level backups and cloud-native snapshots of Cloud SQL instances.
* Create image-level backups and cloud-native snapshots of Cloud Spanner instances.
* Keep the backed-up data in cost-effective, long-term Google storage buckets.
* Restore entire Cloud SQL instances and specific Cloud SQL databases.
* Restore entire VM instances, individual persistent disks, and guest OS files and folders.
* Restore entire VM instances to Microsoft Azure, AWS and Nutanix AHV.
* Perform Instant Recovery of VM instances to VMware vSphere and Hyper-V environments, and to Nutanix AHV clusters.
* Restore entire Cloud Spanner instances and specific Cloud Spanner databases.

For more information, see the [Veeam Backup for Google Cloud User Guide](https://helpcenter.veeam.com/docs/vbgc/guide/welcome.html?ver=7).
