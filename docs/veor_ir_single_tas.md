# Instant Recovery to Another Server

In this article

This section explains how to instantly recover a standalone database or a Data Guard database to another server.

To perform instant recovery to another server, perform the following steps:

1. [Launch the Instant Recovery wizard](veor_ir_single_tas_wizard.md).
2. [Specify a restore point](veor_ir_single_tas_specify_restore_point.md).
3. [Fine-tune the restore point](veor_ir_single_tas_fine_tune.md).
4. [Specify the target Oracle server](veor_ir_single_tas_specify_target_server.md).
5. [Specify Oracle settings](veor_ir_single_tas_specify_oracle_settings.md).
6. [Specify database files location](veor_ir_single_tas_specify_db_file_location.md).
7. [Specify switchover settings](veor_ir_single_tas_specify_switchover_settings.md).
8. [Finalize the Instant Recovery session](veor_ir_single_tas_finalize_ir_session.md).
