# Performing Instant Recovery to VMware vSphere Using Web UI


With Instant Recovery, you can recover workloads from backups and register them as VMware vSphere VMs.

To perform Instant Recovery, use the Instant Recovery to VMware vSphere wizard.

1. [Launch the wizard](instant_recovery_launch_vm_web.md).
2. [Select workloads and restore points](instant_recovery_vms_vm_web.md).
3. [Select a restore mode](instant_recovery_mode_vm_web.md).
4. [Select target hosts and resource pools](instant_recovery_destination_vm_web.md).
5. [Configure settings and folders](instant_recovery_settings_vm_web.md).
6. [Select the destination for virtual disk updates](instant_recovery_datastore_vm_web.md).
7. [Specify a restore reason](instant_recovery_reason_vm_web.md).
8. [Verify settings](instant_recovery_verify_vm_web.md).
9. [Finalize recovery](instant_recovery_review_vm_web.md).

|  |
| --- |
| Note |
| At the moment, web UI has a limited functionality. It allows recovering only VMware vSphere VMs to VMware vSphere. |
