# System Requirements


Before installing or upgrading Veeam Backup & Replication, review the system requirements to ensure compatibility and optimal performance.

This section includes:

* [Workloads](platform_support.md): Describes the operating systems, hypervisors, applications, and workloads that can be protected with Veeam Backup & Replication.
* [Backup Infrastructure](system_requirements.md): Lists the hardware and software requirements for Veeam Backup & Replication components to ensure smooth deployment and reliable operation.

Review both sub-sections carefully to prepare your environment for successful installation and ongoing use.
