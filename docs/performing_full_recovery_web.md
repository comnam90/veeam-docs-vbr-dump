# Restoring Entire VMs Using Web UI


To restore an entire VM using the web UI, use the Entire VM Restore wizard.

1. [Launch the wizard](full_restore_launch_vm_web.md).
2. [Select VMs and restore points](full_restore_vms_vm_web.md).
3. [Select a restore mode](full_restore_mode_vm_web.md).
4. [Select target hosts and resource pools](full_restore_host_vm_web.md).
5. [Select target datastores and disk types](full_restore_datastore_vm_web.md).
6. [Select target folders and change VM settings](full_restore_folder_vm_web.md).
7. [Configure network mapping](full_restore_network_vm_web.md).
8. [Specify a restore reason](full_restore_reason_vm_web.md).
9. [Verify restore settings](full_restore_review_vm_web.md).
