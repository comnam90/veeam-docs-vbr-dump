# Creating Policy for Linux Computers

In this article

To back up data of a computer protected with Veeam Agent for Linux, you can configure a Veeam Agent backup policy in Veeam Backup & Replication.

Before configuring a Veeam Agent backup policy, [check prerequisites](agent_policy_before_linux.md). Then use the New Agent Backup Job wizard to define settings for the backup policy.

1. [Launch the New Agent Backup Job wizard](agent_policy_launch_linux.md).
2. [Select the type of protected computers and the job mode](agent_policy_protection_linux.md).
3. [Specify job name and description](agent_policy_name_linux.md).
4. [Select computers to back up](agent_policy_comp_linux.md).
5. [Select backup mode](agent_policy_mode_linux.md).
6. [Specify backup scope](agent_policy_scope_linux.md).
7. [Select backup destination](agent_policy_target_linux.md).
8. [Specify backup storage settings](agent_policy_target_set_linux.md).
9. [Specify advanced backup settings](agent_policy_advanced_linux.md).
10. [Specify secondary backup target](agent_policy_linux_second_target.md).
11. [Specify guest processing settings](agent_policy_guest_linux.md).
12. [Specify the backup schedule](agent_policy_schedule_linux.md).
13. [Review backup job settings](agent_policy_review_linux.md).
