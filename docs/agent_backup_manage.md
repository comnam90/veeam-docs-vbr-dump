# Managing Veeam Agent Backups

In this article

You can perform administration tasks with backups created by Veeam Agent backup jobs and stored in the Veeam backup repository. For such Veeam Agent backups, Veeam Backup & Replication allows you to perform the following tasks:

* [Create a SureBackup job](recovery_verification_surebackup_job.md).
* [Move a Veeam Agent backup](agent_backup_move.md).
* [Copy a Veeam Agent backup](agent_backup_copy.md).
* [Perform a backup copy for Veeam Agent backup](agent_backup_backup_copy.md).
* [Archive Veeam Agent Backups to Tape.](agent_tape.md)
* [Create a recovery token](agent_backup_recovery_token.md).
* [Create Veeam Recovery Media for a computer](agent_backup_create_image.md).
* [Detach a Veeam Agent backup from a backup job](agent_backup_detach.md).
* [Delete a Veeam Agent backup from disk](agent_backup_delete.md).
* [Remove a Veeam Agent backup from configuration](agent_backup_remove.md).
* [View properties of a Veeam Agent backup](agent_backup_properties.md).
* [Managing Backup Files and Restore Points](agent_backup_points.md)
* [Scanning Veeam Agent backups](agent_backup_scan.md).
