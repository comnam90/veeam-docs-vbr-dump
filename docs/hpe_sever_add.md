# Adding HPE Morpheus VM Essentials Server to Backup Infrastructure


To add a HPE Morpheus VM Essentials manager to the backup infrastructure, do the following:

1. [Launch the HPE Morpheus VM Essentials wizard](hpe_server_add_launch.md).
2. [Specify the HPE Morpheus VM Essentials manager domain name or IP address](hpe_server_add_name.md).
3. [Enter credentials to access the HPE Morpheus VM Essentials manager](hpe_server_add_credentials.md).
4. [Select storage for snapshots](hpe_server_add_storage.md).
5. [Apply HPE Morpheus VM Essentials manager settings](hpe_server_add_apply.md).
6. [Finish working with the wizard](hpe_server_add_summary.md).
