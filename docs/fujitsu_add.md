# Adding Fujitsu ETERNUS HX/AX


Before you add a Fujitsu ETERNUS HX/AX storage system to the backup infrastructure, check [Planning and Preparation](storage_limitations.md).

The Fujitsu ETERNUS HX/AX is based on the ONTAP OS. That is why you can use the NetApp ONTAP wizard. For more information on the steps of the wizard, see [Adding NetApp ONTAP](netapp_add.md).
