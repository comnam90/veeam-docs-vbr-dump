# Performing Failover


For more information on failover, see [Failover](cdp_failover.md) and [Failover and Failback for CDP](cdp_failover_failback.md).

To perform failover, use the VMware Failover wizard.

1. [Check prerequisites](cdp_failover_byb.md).
2. [Launch the VMware Failover wizard](cdp_failover_launch.md).
3. [Select VMs](cdp_failover_workloads.md).
4. [Select a restore point](cdp_failover_point.md).
5. [Specify the failover reason](cdp_failover_reason.md).
6. [Finish working with the wizard](cdp_failover_summary.md).
