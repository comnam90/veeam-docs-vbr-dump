# Creating Backup Jobs


To create a backup job, do the following:

1. [Check prerequisites and limitations](sch_backup_job_create_prerequisites.md).
2. [Launch the New Backup Job wizard](sch_backup_job_create_launch.md).
3. [Specify a job name and description](sch_backup_job_create_general_settings.md).
4. [Selects VMs to back up](sch_backup_job_create_assign_vms.md).
5. [Specify a backup repository and configure backup settings](sch_backup_job_create_destination.md).
6. [Enable guest processing](sch_backup_job_create_guest_processing.md) (only in Veeam Plug-in for Scale Computing HyperCore v3)
7. [Create a schedule for the backup job](sch_backup_job_create_schedule.md).
8. [Finish working with the wizard](sch_backup_job_create_summary.md).
