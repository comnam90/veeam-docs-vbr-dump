# Data Recovery (All Platforms)

In this article

Veeam Backup & Replication provides platform-specific data recovery methods as well as platform-independent methods:

* [VM Recovery](vm_recovery_all.md) — to restore entire VMs to different data protection environments: to VMware vSphere, Hyper-V, Amazon EC2 and so on.
* [Disk Recovery](disk_recovery_all.md)— to publish and export disks.
* [Item Recovery](item_recovery_all.md) — to recover guest OS files and folders, and application items.
