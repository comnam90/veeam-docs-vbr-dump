# Adding Linux Repositories Using Console


This section describes how to add a Linux server as a backup repository.

To add a backup repository using the Veeam Backup & Replication console, use the New Backup Repository wizard.

1. [Launch the New Backup Repository wizard](linux_repository_launch.md).
2. [Specify backup repository name and description](linux_repository_name.md).
3. [Specify server settings](linux_repository_server.md).
4. [Configure backup repository settings](linux_repository_repository.md).
5. [Specify mount server settings](linux_repository_mount_server.md).
6. [Review properties and components](linux_repository_review.md).
7. [Apply backup repository settings](linux_repository_apply.md).
8. [Finish working with the wizard](linux_repository_finish.md).
