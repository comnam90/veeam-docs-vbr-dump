# Performing Planned Failover


For more information on planned failover, see [Planned Failover](planned_failover.md) and [Failover and Failback for Replication](failover_failback.md).

To perform planned failover, use the Planned Failover wizard.

1. [Check prerequisites](planned_failover_before_you_begin.md).
2. [Launch the Planned Failover wizard](planned_failover_launch.md).
3. [Select VMs](planned_failover_vms.md).
4. [Specify a failover reason](planned_failover_reason.md).
5. [Review summary and finish working with the wizard](planned_failover_summary.md).
