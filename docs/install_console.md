# Installing Veeam Backup & Replication Console

In this article

Before you install the Veeam Backup & Replication console, [check prerequisites](console_install_before_you_begin.md). Then use the Veeam Backup & Replication Console Setup wizard to install the console.

1. [Start the setup wizard](console_install_launch.md).
2. [Select the component to install](console_install_select_component.md).
3. [Read and accept the License Agreement](console_install_eula.md).
4. [Review the default installation settings](console_install_default.md).
5. [Specify the installation path](console_install_path.md).
6. [Begin the installation](console_install_summary.md).
