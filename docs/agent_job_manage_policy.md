# Managing Veeam Agent Backup Policies


You can use the Veeam Backup & Replication console to perform the following operations with a Veeam Agent backup job managed by Veeam Agent (a Veeam Agent backup policy):

* [Apply backup policy to Veeam Agent computers](agent_policy_apply.md).
* [Start and stop Veeam Agent backup jobs on computers added to the backup policy](agent_policy_start_stop.md).
* [Perform active full backup on computers added to the backup policy](agent_policy_active_full.md).
* [Clear the backup cache on computers added to the backup policy](agent_policy_cache.md).
* [Edit backup policy settings](agent_policy_edit.md).
* [Enable and disable a backup policy](agent_policy_disable.md).
* [Clone a backup policy](agent_policy_clone.md).
* [Remove a backup policy](agent_policy_delete.md).
