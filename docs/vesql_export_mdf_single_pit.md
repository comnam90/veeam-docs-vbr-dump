# Exporting Point-in-Time State


To export data as of a point-in-time state to the default location, use the Export wizard.

1. [Launch the Export wizard](vesql_export_mdf_single_pit_wizard.md).
2. [Specify a restore point](vesql_export_mdf_single_pit_restore_point.md).
3. [Fine-tune the restore point](vesql_export_mdf_single_pit_fine_tune.md).
4. [Review the export summary](vesql_export_single_pit_summary.md).

|  |
| --- |
| Note |
| Point-in-time restore is available only if transaction log backups exist. For more information, see [Required Job Settings](vesql_bu_job_settings.md). |
