# Performing Failback


For more information on failback, see [Failover and Failback for Cloud Director CDP](vcd_cdp_failover_failback.md) and [Failback](vcd_cdp_failback.md).

To perform failback, do the following:

1. [Launch the Failback wizard](vcd_cdp_failback_launch.md).
2. [Select replicas](vcd_cdp_failback_replicas.md).
3. [Select the failback destination](vcd_cdp_failback_dest.md).
4. [Select organization VDCs](vcd_cdp_failback_vdc.md).
5. [Specify storage policies](vcd_cdp_failback_storage.md).
6. [Configure network mapping](vcd_cdp_failback_network.md).
7. [Select the target vApp](vcd_cdp_failback_vapp.md).
8. [Schedule switching to production vApps](vcd_cdp_failback_schedule.md).
9. [Finish working with the wizard](vcd_cdp_failback_summary.md).
