# Adding NetApp ONTAP

In this article

Before you add to the backup infrastructure a NetApp storage system running ONTAP, check [Planning and Preparation](storage_limitations.md). Then use the New NetApp Data ONTAP Storage wizard to add the storage system.

1. [Launch the New NetApp Data ONTAP Storage wizard](netapp_add_launch.md).
2. [Specify NetApp server name or address and storage role](netapp_add_name.md).
3. [Specify credentials and protocol type](netapp_add_creds.md).
4. [[For Microsoft Windows-based backup server]](specify_vmware_access.md) [Specify VMware access options](netapp_add_options.md).
5. [[For Microsoft Windows-based backup server]](specify_vmware_access.md) [Specify Veeam Agent access options](netapp_agents_access.md).
6. [Specify NAS access options](netapp_nas_access.md).
7. [Apply settings](netapp_add_apply.md).
8. [Finish working with wizard](netapp_add_rescan.md).
