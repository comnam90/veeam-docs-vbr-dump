# Veeam Backup PowerShell Module


Veeam Backup PowerShell is an extension for Microsoft Windows PowerShell that adds a set of cmdlets to allow users to perform backup, replication and recovery tasks through the command-line interface of PowerShell or run custom scripts to fully automate operation of Veeam Backup & Replication. For more information on the Veeam Backup PowerShell module, see the [Getting Started](https://helpcenter.veeam.com/docs/vbr/powershell/getting_started.html?ver=13) section of Veeam PowerShell Reference.
