# Creating Replication Jobs


To create VM replicas, you must configure a replication job. The replication job defines how, where and when to replicate VM data. One job can be used to process one VM or multiple VMs.

To create a replication job, use the New Replication Job wizard.

1. [Check prerequisites](replication_byb_hv.md).
2. [Launch the New Replication Job wizard](replica_launch_hv.md).
3. [Specify a job name and description](replica_name_hv.md).
4. [Select VMs to replicate](replica_vms_hv.md).
5. [Specify a data source](replica_data_source_hv.md).
6. [Exclude objects from being replicated](replica_exclude_hv.md).
7. [Define the VM processing order](replica_order_hv.md).
8. [Specify a replica destination](replica_destination_hv.md).
9. [Create a network map table](replica_network_hv.md).
10. [Configure Re-IP rules](replica_reip_hv.md).
11. [Specify replication job settings](replica_settings_hv.md).
12. [Specify advanced replica settings](replica_advanced_hv.md).
13. [Specify data transfer settings](replica_data_transfer_hv.md).
14. [Configure seeding and mapping](replica_seeding_hv.md).
15. [Specify guest processing settings](replica_vss_hv.md).
16. [Define a job schedule](replica_schedule_hv.md).
17. [Finish working with the wizard](replica_review_hv.md).
