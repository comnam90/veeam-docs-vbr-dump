# Exporting to Different Location

In this article

To export a single Microsoft SQL Server database to a custom location, use the Export wizard.

1. [Launch the Export wizard](vesql_mdf_single_tdl_wizard.md).
2. [Specify a restore point](vesql_mdf_single_tdl_restore_point.md).
3. [Fine-tune the restore point](vesql_mdf_single_tdl_fine_tune.md).
4. [Specify export location](vesql_mdf_single_tdl_files.md).
5. [Review the export summary](vesql_export_single_tal_summary.md).
