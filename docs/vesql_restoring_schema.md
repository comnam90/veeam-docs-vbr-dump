# Restoring Database Schema and Data


To restore database schema and data, do the following:

1. [Launch the Restore wizard](vesql_restore_wizard_schema.md).
2. [Specify a restore point](vesql_restore_point_schema.md).
3. [Fine-tune the restore point](vesql_fine_tune_schema.md).
4. [Specify target SQL server settings](vesql_target_server_schema.md).
5. [Select database objects](vesql_database_obj_schema.md).
6. [Specify names for restored objects](vesql_names_obj_schema.md).
7. [Specify directory names for file tables](vesql_dir_names_schema.md).
8. [Specify additional restore options](vesql_specify_additional_schema.md).
9. [Review the restore summary](vesql_restore_schema_summary.md).
