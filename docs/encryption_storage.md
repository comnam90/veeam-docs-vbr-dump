# Storage Encryption


Storage-level encryption is supported for the following backups:

* Backup files stored to capacity tier. For more information, see [Encryption for Capacity Tier](encryption_for_capacity_tier.md).
* Backup files stored to archive tier. For more information, see [Encryption for Archive Tier](encryption_for_archive_tier.md).
* Backup files created by standalone applications. For more information, see [Encrypting Standalone Application Backups in Backup Repositories](encrypting_backups.md).
* Backup files stored to external repositories. For more information, see [Encryption](external_repository_encryption.md).
* Backup files stored to tape media pools. For more information, see [Tape Encryption](encryption_tape.md).
