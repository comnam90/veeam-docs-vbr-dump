# Scale Computing HyperCore

In this article

Veeam Backup & Replication with Veeam Plug-In for Scale Computing HyperCore is a solution developed for protection and disaster recovery tasks for the Scale Computing HyperCore environment. With Veeam Plug-In for Scale Computing HyperCore, you can perform the following operations:

* Create backups of Scale Computing HyperCore VMs and store them in backup repositories.

* Create VeeamZIP backups of Scale Computing HyperCore VMs.

* Create several instances (copies) of the same backup data in different locations.
* Restore VMs from Scale Computing HyperCore VM backups to the Scale Computing HyperCore environment.

* Restore VMs from VMware ESXi and Microsoft Hyper-V to the Scale Computing HyperCore environment.
* Restore VMs from Nutanix AHV, Proxmox VE and oVirt KVM backups to the Scale Computing HyperCore environment.

* Restore VMs from Microsoft Azure, Amazon Web Services (AWS) and Google Cloud backups to the Scale Computing HyperCore environment.
* Restore physical machines from backups created by Veeam Agents to the Scale Computing HyperCore environment.

* Restore VMs from Scale Computing HyperCore backups to Microsoft Azure, Amazon Web Services (AWS) and Google Cloud environments.
* Perform Instant Recovery of Scale Computing HyperCore VMs to VMware vSphere, Microsoft Hyper-V and Nutanix AHV environments.
* Restore application items (such as Microsoft Active Directory, Microsoft Exchange, Microsoft SharePoint, Oracle Database and Microsoft SQL Server).
* Restore files and folders of Scale Computing HyperCore VM guest OSes.
* Export disks of backed-up Scale Computing HyperCore VMs to VMDK, VHD and VHDX formats.
* Mount disks of backed-up Scale Computing HyperCore VMs to any server and access data in the read-only mode.

For more information, see the [Veeam Plug-In for Scale Computing HyperCore User Guide](https://helpcenter.veeam.com/docs/vpsch/userguide/overview.html?ver=2).
