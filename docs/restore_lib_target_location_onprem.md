# Step 2. Select Target Site Location


At this step of the wizard, select On-premises Microsoft SharePoint Server to restore the document library or list to an on-premises Microsoft SharePoint organization. For more information, see the [Adding On-Premises Microsoft Organizations](https://helpcenter.veeam.com/docs/vbo365/guide/vbo_add_onpremises_org.html?ver=80) section of the Veeam Backup for Microsoft 365 User Guide.
