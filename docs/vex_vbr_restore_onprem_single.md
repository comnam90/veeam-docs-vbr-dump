# Restoring Single Mailbox, Folder or Item


To restore a single mailbox, folder or item, do the following:

1. [Launch the Restore wizard](vex_vbr_restore_onprem_single_launch_wizard.md).
2. [Specify a target mailbox](vex_vbr_restore_onprem_single_target_mailbox.md).
3. [Specify a user account](vex_vbr_restore_onprem_single_specify_user_account.md).
4. [Specify a CAS server and target folder](vex_vbr_restore_onprem_single_specify_cas_and_folder.md).
5. [Specify restore options](vex_vbr_restore_onprem_single_restore_options.md).
6. [Review the operation summary](summary6.md).
