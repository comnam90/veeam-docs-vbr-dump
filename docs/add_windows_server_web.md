# Adding Microsoft Windows Servers Using Web UI


This section describes how to add a Microsoft Windows server as a backup infrastructure component.

To add a Microsoft Windows server using the Veeam Backup & Replication web UI, use the New Windows Server wizard.

1. [Launch the New Windows Server wizard](windows_server_launch_web.md).
2. [Specify a server name or address](windows_server_name_web.md).
3. [Specify credentials](windows_server_credentials_web.md).
4. [Review components](windows_server_review_web.md).
5. [Finish working with the wizard](windows_server_summary_web.md).
