# Adding oVirt KVM Manager to Backup Infrastructure


To add the oVirt KVM Manager to the backup infrastructure, do the following:

1. [Launch the New oVirt KVM Manager wizard](ovirt_add_rhv_launch.md).
2. [Specify the oVirt KVM Manager domain name or IP address](ovirt_add_rhv_name.md).
3. [Enter credentials to access the oVirt KVM Manager](ovirt_add_rhv_credentials.md).
4. [Apply oVirt KVM Manager settings](ovirt_add_rhv_apply_settings.md).
5. [Finish working with the wizard](ovirt_add_rhv_finish.md).
