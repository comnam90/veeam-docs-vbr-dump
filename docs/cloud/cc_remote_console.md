# Connecting to Tenant with Remote Access Console


To connect to the tenant backup server, the SP must run the Remote Access Console on the SP backup server or dedicated machine.

Before connecting to the tenant backup server with the Remote Access Console, [check prerequisites](cc_remote_console_byb.md). Then perform the following operations:

1. [Open the Remote Access Console](cc_remote_console_launch.md).
2. [Connect to the SP backup server](cc_remote_console_sp.md).
3. [Connect to the tenant backup server](cc_remote_console_tenant.md).
