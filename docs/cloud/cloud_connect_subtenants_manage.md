In this article

To provide subtenants with individual storage quotas on the cloud repository, the SP or tenant must register a subtenant account for each subtenant. The SP can perform the following operations with subtenant accounts:

* [Add a subtenant account for a standalone tenant account](cloud_connect_subtenant_create_sp.md).
* [Add a subtenant account for a VMware Cloud Director tenant account](subtenant_vcd_create_sp.md).
* [Edit a subtenant account](cloud_connect_subtenants_edit_sp.md).
* [Remove a subtenant account](cloud_connect_subtenants_remove_sp.md).
