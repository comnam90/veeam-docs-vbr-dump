# Performing Backup


You can perform the following data protection tasks with the cloud repository in Veeam Backup & Replication:

* [VM backup](cloud_connect_backup.md)
* [VMware Cloud Director backup](cloud_connect_vcloud.md) (for VMware vSphere platform)
* [Veeam Agent backup](cloud_connect_backup_agents.md)
* [Backup copy](cloud_connect_backup_copy.md) (To a cloud repository only. Backup copy from a cloud repository is not supported.)
