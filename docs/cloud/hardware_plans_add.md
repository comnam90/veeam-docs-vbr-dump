In this article

You can configure one or several hardware plans in your Veeam Cloud Connect infrastructure.

Before configuring hardware plans, [check prerequisites](hardware_plan_before_you_begin.md). Then use the New Hardware Plan wizard to configure a hardware plan.

1. [Launch the New Hardware Plan wizard](hardware_plan_launch.md).
2. [Specify hardware plan name and description](hardware_plan_name.md).
3. [Specify a host or a cluster for allocating resources](hardware_plan_host.md).
4. [Specify storage settings](hardware_plan_storage.md).
5. [Specify network settings](hardware_plan_network.md).
6. [Assess results](hardware_plan_assess_results.md).
7. [Finish working with the wizard](cloud_connect_user_finish.md).
