In this article

The Guide for Service Providers is intended for SPs who expose cloud repository resources and provide disaster recovery as a service to their customers using the Veeam Cloud Connect functionality in Veeam Backup & Replication. The guide describes main tasks that the SP must take to set up the necessary infrastructure and manage it.
