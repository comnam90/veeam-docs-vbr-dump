# Configuration File


The DB2ConfigTool configures IBM Db2 settings and saves the settings to the Veeam Plug-In configuration file (veeam\_config.xml). The file is located in the /opt/veeam/VeeamPluginforDB2 directory on the machine where Veeam Plug-In is installed.
