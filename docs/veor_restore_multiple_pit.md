# Restoring Point-in-Time State

In this article

To restore multiple Oracle databases to the original server as of the selected point-in-time state, do the following:

1. [Launch the Restore wizard](veor_restore_multiple_pit_wizard.md).
2. [Specify a restore point](veor_restore_multiple_pit_specify_restore_point.md).
3. [Review the restore summary](veor_restore_multiple_pit_summary.md).
