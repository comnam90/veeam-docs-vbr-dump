# Publishing to Specified Server


This section explains how to publish a standalone database or Data Guard database to the specified Oracle server.

To publish data, take the following steps:

1. [Launch the Publish wizard](veor_ptsr_launch.md).
2. [Specify a restore point](veor_ptsr_restore_point.md).
3. [Fine-tune the restore point](veor_ptsr_fine_tune.md).
4. [Specify the target Oracle server](veor_ptsr_target_server.md).
5. [Specify Oracle settings](veor_oracle_settings.md).
6. [Review the publishing summary](veor_publish_tas_summary.md).
