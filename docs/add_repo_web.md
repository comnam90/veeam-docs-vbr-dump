# Adding Microsoft Windows Repositories Using Web UI

In this article

This section describes how to add a Microsoft Windows server as a backup repository.

To add a backup repository using the Veeam Backup & Replication web UI, use the New Backup Repository wizard.

1. [Launch the New Backup Repository wizard](repository_launch_web.md).
2. [Specify backup repository name and description](repository_name_web.md).
3. [Specify server or shared folder settings](repository_server_web.md).
4. [Configure path and load control settings](repository_repository_web.md).
5. [Specify mount server settings](repository_mount_server_web.md).
6. [Review properties and components](repository_review_web.md).
7. [Finish working with wizard](repository_finish_web.md).
