# Adding Hardened Repositories Using Web UI

In this article

This section describes how to add a hardened repository.

To add a hardened repository using the Veeam Backup & Replication web UI, use the New Backup Repository wizard.

1. [Launch the new backup repository wizard](linux_repository_launch_web.md).
2. [Specify hardened repository name and description](linux_repository_name_web.md).
3. [Specify server settings](linux_repository_server_web.md).
4. [Configure hardened repository settings](linux_repository_repository_web.md).
5. [Specify mount server settings](linux_repository_mount_server_web.md).
6. [Review properties and components](linux_repository_review_web.md).
7. [Finish working with the wizard](linux_repository_finish_web.md).
