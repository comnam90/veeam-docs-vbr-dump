# Instant Recovery of Point-in-Time State


This section explains how to instantly recover a standalone database or a Data Guard database to a selected point-in-time state on the backup file.

To perform instant recovery to a selected point-in-time state, perform the following steps:

1. [Launch the Instant Recovery wizard](veor_ir_single_pit_wizard.md).
2. [Specify a restore point](veor_ir_single_pit_specify_restore_point.md).
3. [Fine-tune the restore point](veor_ir_single_pit_fine_tune.md).
4. [Specify switchover settings](veor_ir_single_pit_specify_switchover_settings.md).
5. [Finalize the Instant Recovery session](veor_ir_single_pit_finalize_ir_session.md).

|  |
| --- |
| Note |
| Point-in-time restore is available only if archived log backups exist. For more information, see [Required Job Settings](veo_bu_job_settings.md). |
