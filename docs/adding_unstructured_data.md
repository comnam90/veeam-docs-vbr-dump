# Adding Unstructured Data Source

In this article

You must add to the backup infrastructure sources of unstructured data that you plan to protect with the file to tape and object to tape jobs.

You can add the following types to the inventory of your virtual infrastructure:

* [Windows-managed or Linux-managed file server](tape_adding_file_server.md)
* [File shares](tape_adding_file_share.md)

* [SMB file share](tape_adding_smb_share.md)
* [NFS file share](tape_adding_nfs_share.md)

* [Object storage](tape_adding_object_storage.md)
