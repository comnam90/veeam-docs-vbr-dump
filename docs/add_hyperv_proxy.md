# Adding Off-Host Backup Proxies Using Console

In this article

You can configure one or more off-host backup proxies in the backup infrastructure.

Before you add an off-host backup proxy using Veeam Backup & Replication console, [check prerequisites](offhost_backup_proxy.md#req_general). Then use the New Hyper-V Off-Host Backup Proxy wizard to add the off-host backup proxy.

1. [Launch the New Hyper-V Off-Host Backup Proxy wizard](hv_proxy_launch.md).
2. [Choose a server](hv_proxy_server.md).
3. [Configure traffic throttling rules](hv_proxy_traffic.md).
4. [Review components](hv_proxy_review.md).
5. [Finish working with the wizard](hv_proxy_summary.md).
