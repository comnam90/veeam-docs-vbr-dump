# Configuring Veeam Plug-In for HPE Morpheus VM Essentials


To start working with Veeam Plug-in for HPE Morpheus VM Essentials, perform a number of steps for its configuration:

1. [Configure backup repositories](hpe_configure_repository.md) where Veeam Backup & Replication will store backups of HPE Morpheus VM Essentials VMs.
2. [Connect to HPE Morpheus VM Essentials managers](hpe_sever_add.md) that administer HPE Morpheus VM Essentials resources you want to protect.
3. [Deploy workers](hpe_workers.md) that will transfer backup traffic.
4. [[Optional] Configure email settings and notifications](hpe_general_settings.md).
