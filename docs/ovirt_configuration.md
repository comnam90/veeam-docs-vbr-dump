# Configuring Veeam Plug-In for oVirt KVM


To start working with Veeam Plug-in for oVirt KVM, perform a number of steps for its configuration:

1. [Configure backup repositories where Veeam Backup & Replication will store backups of oVirt VMs.](ovirt_configure_repository.md)
2. [Connect to oVirt KVM Manager that administer oVirt resources you want to protect.](ovirt_connecting_manager.md)
3. [Deploy workers that will transfer backup traffic.](ovirt_workers.md)
4. [[Optional] Configure email settings and notifications.](ovirt_general_settings.md)
