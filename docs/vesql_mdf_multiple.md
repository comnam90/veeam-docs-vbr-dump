# Exporting Multiple Databases


You can export multiple Microsoft SQL Server databases to the machine where Veeam Explorer for Microsoft SQL Server is running by following one of these scenarios:

* [Exporting Latest State](vesql_mdf_multiple_latest.md)
* [Exporting Point-in-Time State](vesql_mdf_multiple_pit.md)
* [Exporting to Different Location](vesql_mdf_multiple_tdl.md)
