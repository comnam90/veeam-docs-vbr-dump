# Creating Policy for Windows Computers


To back up data of a computer protected with Veeam Agent for Microsoft Windows, you can configure a Veeam Agent backup policy in Veeam Backup & Replication.

Before configuring a Veeam Agent backup policy, [check prerequisites](agent_policy_win_before.md). Then use the New Agent Backup Job wizard to define settings for the backup policy.

1. [Launch the New Agent Backup Job wizard](agent_policy_win_launch.md).
2. [Select the type of protected computers and the job mode](agent_policy_win_protection_mode.md).
3. [Specify job name and description](agent_policy_win_name.md).
4. [Select computers to back up](agent_policy_win_scope.md).
5. [Select backup mode](agent_policy_win_mode.md).
6. [Specify backup scope](agent_policy_win_backup_scope.md).
7. [Select backup destination](agent_policy_win_destination.md).
8. [Specify backup storage settings](agent_policy_win_target_settings.md).
9. [Specify advanced backup settings](agent_policy_win_advanced.md).
10. [Specify secondary backup target](agent_policy_win_secondary_target.md).
11. [Specify backup cache settings](agent_policy_win_cache.md).
12. [Specify guest processing settings](agent_policy_win_vss.md).
13. [Specify the backup schedule](agent_policy_win_schedule.md).
14. [Review backup job settings](agent_policy_win_review.md).
