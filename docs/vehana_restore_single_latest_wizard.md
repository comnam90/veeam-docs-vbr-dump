# Step 1. Launch Restore Wizard

In this article

To restore the latest available state of an SAP HANA database, do the following:

1. In the navigation pane, select a database.
2. On the Database tab, select Restore Database > Restore the latest state to <original\_location>.

Alternatively, you can right-click a database and select Restore latest state to <original\_location>.
