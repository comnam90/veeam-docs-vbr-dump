# Entra ID Log Restore


Entra ID log restore allows restoring folders containing log files or individual log files.

To restore folders with audit and sign-in logs or individual log files, use the Microsoft Entra ID Audit Restore wizard.

1. [Launch the wizard](entra_id_restore_log_launch.md).
2. [Select files and folders to restore](entra_id_restore_log_file.md).
3. [Select a restore mode](entra_id_restore_log_mode.md).
4. [Select a restore point](entra_id_restore_log_point.md).
5. [Specify the destination](entra_id_restore_log_dest.md).
6. [Finish working with the wizard](entra_id_restore_log_summary.md).
