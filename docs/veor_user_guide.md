# Veeam Explorer for Oracle


Veeam Explorer for Oracle allows you to restore, publish, instantly recover and export standalone Oracle databases and Oracle Data Guard databases from restore points created by Veeam Backup & Replication. Veeam Explorer for Oracle supports data recovery from backups, snapshot replicas, CDP replicas, VeeamZIP backups and storage snapshots. Veeam Explorer for Oracle also allows you to restore your data from backups created with Veeam Plug-In for Oracle RMAN (in managed or standalone mode).
