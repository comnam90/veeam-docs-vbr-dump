# Performing Failover

In this article

For more information on failover, see [Failover](uni_cdp_failover.md) and [Failover and Failback for CDP](uni_cdp_failover_failback.md).

To perform failover, use the Failover wizard.

1. [Check prerequisites](uni_cdp_failover_byb.md).
2. [Launch the Wizard](uni_cdp_failover_launch.md).
3. [Select VMs](uni_cdp_failover_workloads.md).
4. [Select restore points](uni_cdp_failover_point.md).
5. [Specify failover reason](uni_cdp_failover_reason.md).
6. [Finish working with the wizard](uni_cdp_failover_summary.md).
