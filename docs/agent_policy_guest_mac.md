# Step 11. Specify Guest Processing Settings


At the Guest Processing step of the wizard, you can enable guest processing and specify pre-job and post-job scripts.

To activate guest processing settings, do the following:

1. Select Enable application-aware processing.
2. Click Applications to specify application-aware processing settings for individual items.

![Step 11. Specify Guest Processing Settings](images/agent_policy_guest_general_mac.webp)

Veeam Agent will display the Application-Aware Processing Options window.
