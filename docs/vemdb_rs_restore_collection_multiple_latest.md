# Restoring Latest State

In this article

To restore the latest available state of multiple MongoDB collections, use the Restore wizard.

1. [Launch the Restore wizard](vemdb_rs_restore_collection_multiple_wizard.md).
2. [Specify restore settings](vemdb_rs_restore_collection_multiple_specify_restore_settings.md).
3. [Specify the replica set credentials](vemdb_rs_restore_collection_multiple_specify_rs_credentials.md).
4. [Specify primary node credentials](vemdb_rs_restore_collection_multiple_specify_primary_node_credentials.md).
5. [Select databases](vemdb_rs_restore_collection_multiple_select_databases.md).
6. [Specify collections](vemdb_rs_restore_collection_multiple_select_collections.md).
