# Data Restore


This section contains information about the available restore operations in Veeam Explorer for SAP HANA. You can restore SAP HANA databases from database-level backups — created by application backup policies managed by Veeam Backup & Replication or backup jobs managed by a standalone Veeam Plug-In for SAP HANA.

Before restoring data, read the [Considerations and Limitations](vehana_considerations.md) section.
