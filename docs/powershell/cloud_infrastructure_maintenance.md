# Veeam Cloud Connect Infrastructure Maintenance


You can use the cmdlets from this section to enable or disable Maintenance mode for the cloud infrastructure:

| Cmdlet | Operation |
| --- | --- |
| [Enable-VBRCloudMaintenanceMode](enable-vbrcloudmaintenancemode.md) | Enables Maintenance mode for the cloud infrastructure |
| [Disable-VBRCloudMaintenanceMode](disable-vbrcloudmaintenancemode.md) | Disables Maintenance mode for the cloud infrastructure |
| [Get-VBRCloudInfrastructureState](get-vbrcloudinfrastructurestate.md) | Checks the cloud infrastructure status |
