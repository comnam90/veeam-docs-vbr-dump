# VBRAzureDiskConfiguration


Contains Azure VM disk types.

Properties

| Property | Type | Description |
| --- | --- | --- |
| DiskName | String | Disk name. |
| DiskType | [VBRAzureDiskType](enums.md#VBRAzureDiskType) | Azure VM disk type. |
| Include | Boolean | Defines that the cmdlet will include (TRUE) or exclude (FALSE) Azure VM disks. |
