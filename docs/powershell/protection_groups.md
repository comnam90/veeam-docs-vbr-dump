# Veeam Agent Management Protection Groups


You can use the cmdlet in this topic to perform the following operations.

| Cmlet | Operation |
| --- | --- |
| [Add-VBRProtectionGroup](add-vbrprotectiongroup.md) | Adds protection groups. |
| [Get-VBRProtectionGroup](get-vbrprotectiongroup.md) | Returns protection groups. |
| [Set-VBRProtectionGroup](set-vbrprotectiongroup.md) | Modifies protection groups. |
| [Get-VBRProtectionGroupLinuxPackage](get-vbrprotectiongrouplinuxpackage.md) | Returns Veeam Agent for Linux packages. |
| [Get-VBRProtectionGroupUnixPackage](get-vbrprotectiongroupunixpackage.md) | Returns Veeam Agents for Unix packages. |
| [New-VBRPeriodicallyOptions](new-vbrperiodicallyoptions.md) | Creates periodical schedule settings. |
| [New-VBRProtectionGroupScheduleOptions](new-vbrprotectiongroupscheduleoptions.md) | Creates a discovery schedule. |
| [New-VBRProtectionGroupDeploymentOptions](new-vbrprotectiongroupdeploymentoptions.md) | Creates agent deployment settings. |
| [New-VBRProtectionGroupNotificationOptions](new-vbrprotectiongroupnotificationoptions.md) | Creates notification options for protection groups. |
| [Set-VBRProtectionGroupNotificationOptions](set-vbrprotectiongroupnotificationoptions.md) | Modifies notification options for protection groups. |
| [New-VBRProtectionGroupAdvancedWindowsOptions](new-vbrprotectiongroupadvancedwindowsoptions.md) | Creates advanced settings for Veeam Agent for Microsoft Windows. |
| [Disable-VBRProtectionGroup](disable-vbrprotectiongroup.md) | Disables automatic discovery. |
| [Enable-VBRProtectionGroup](enable-vbrprotectiongroup.md) | Enables automatic discovery. |
| [Remove-VBRProtectionGroup](remove-vbrprotectiongroup.md) | Removes protection groups. |
| [Generate-VBRBackupServerDeployerKit](generate-vbrbackupserverdeployerkit.md) | Generates Veeam Installer Service and Veeam Deployer Service certificates and installation packages for Microsoft Windows and Linux computers. |
| [Get-VBRBackupServerDeployerCertificate](get-vbrbackupserverdeployercertificate.md) | Downloads Veeam Installer Service and Veeam Deployer Service certificates to the specified location. |
| [Remove-VBRBackupServerDeployerCertificate](remove-vbrbackupserverdeployercertificate.md) | Removes Veeam Deployer Service certificates from the Veeam Backup & Replication database. |
