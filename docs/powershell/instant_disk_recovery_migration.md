# Instant Disk Recovery Migration


You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [New-VBRViDiskMigrationMappingRule](new-vbrvidiskmigrationmappingrule.md) | Defines mapping settings of recovered VM virtual disks and target datastores |
| [Start-VBRViInstantRecoveryDiskMigration](start-vbrviinstantrecoverydiskmigration.md) | Starts publish VM virtual disks to the production environment. |
