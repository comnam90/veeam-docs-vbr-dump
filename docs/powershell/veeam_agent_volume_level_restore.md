# Restoring Volumes


You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRComputerDisk](get-vbrcomputerdisk.md) | Returns disks of computers backed-up with Veeam Agent for Microsoft Windows. |
| [Get-VBRComputerDiskPartition](get-vbrcomputerdiskpartition.md) | Returns volumes of computers backed-up with Veeam Agent for Microsoft Windows. |
| [Start-VBRComputerVolumeRestore](start-vbrcomputervolumerestore.md) | Starts restore to original location of computer volumes backed-up with Veeam Agent for Microsoft Windows. |
