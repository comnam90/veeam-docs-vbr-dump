# Application Backup Policies


You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRApplicationBackupJob](add-vbrapplicationbackupjob.md) | Creates application backup policies. |
| [Get-VBRApplicationBackupJob](get-vbrapplicationbackupjob.md) | Returns application backup policies. |
| [Set-VBRApplicationBackupJob](set-vbrapplicationbackupjob.md) | Modifies application backup policies. |
| [Get-VBRApplicationBackupJobSession](get-vbrapplicationbackupjobsession.md) | Returns application backup policy sessions. |
| [Start-VBRApplicationBackupJob](start-vbrapplicationbackupjob.md) | Starts application backup policies. |
| [Stop-VBRApplicationBackupJob](stop-vbrapplicationbackupjob.md) | Stops application backup policies. |
| [Remove-VBRApplicationBackupJob](remove-vbrapplicationbackupjob.md) | Removes application backup policies. |
| [Enable-VBRApplicationBackupJob](enable-vbrapplicationbackupjob.md) | Enables application backup policies. |
| [Disable-VBRApplicationBackupJob](disable-vbrapplicationbackupjob.md) | Disables application backup policies. |
