# VBRCloudTenantBackup


This object contains backups of AD tenants and tenants managed by the Service Provider.

Properties

| Property | Type | Description |
| --- | --- | --- |
| TenantId | GUID | Tenant ID. |
| RepositoryId | GUID | Repository ID. |
| CreationTime | DateTime | Creation time of the tenant backup. |
