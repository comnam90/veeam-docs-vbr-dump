# VBRAmazonS3ExternalConnection


Contains connection settings to Amazon S3 object storage.

Properties

| Property | Type | Description |
| --- | --- | --- |
| HostId | Guid | Gateway server. |
| CredentialsId | Guid | Amazon S3 credentials record. |
| RegionType | VBRAmazonRegionType | Amazon S3 region. |
