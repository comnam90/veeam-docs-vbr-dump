# Global Changes


This section provides essential information about global changes introduced in Veeam PowerShell version 13.

* PowerShell 7 Required

Veeam PowerShell version 13 requires PowerShell 7 as the minimum version to operate.

* New Standalone Veeam PowerShell Module for Linux Distributions

A new standalone Veeam PowerShell module for Veeam Backup & Replication on the Linux platform has been introduced. The module supports Rocky Linux 9 latest version and RHEL 9.6. For more information on how to install the Veeam PowerShell module, see [Running Veeam PowerShell Session from Linux Machines](running_ps_sessions_linux.md).

* Windows Requirements

To utilize Veeam PowerShell on Windows, you must use the Veeam Backup & Replication Console. For more information, see [Running Veeam Backup PowerShell on Windows Machines](running_ps_sessions_windows.md).

* Veeam Explorer PowerShell

To utilize the Veeam Explorer PowerShell functionality, you must install the necessary Veeam Explorer PowerShell module. For more information, see [Starting PowerShell Sessions](https://helpcenter.veeam.com/docs/vbr/explorers_powershell/ps_sessions.html?ver=13) in the Veeam Explorers PowerShell Reference.
