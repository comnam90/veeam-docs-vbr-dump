# VBRUnstructuredBackup

In this article

Contains details on backup of unstructured data.

Properties

| Property | Type | Description |
| --- | --- | --- |
| Id | Guid | Backup ID |
| Name | String | Backup name |
| JobId | Guid | Backup job information. |
| CreationTime | Guid | Date and time when backup was created |
| LastRestorePointCreationTime | bool | Last restore point details |
| IsEncrypted | Boolean | Defines that the backup is encrypted |
