# Managing Amazon Connection Service


The cmdlets in this section allow you to start a session with the Amazon S3 object storage.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Connect-VBRAmazonS3Service](connect-vbramazons3service.md) | Starts a new session with the AmazonS3 object storage. |
| [Connect-VBRAmazonS3CompatibleService](connect-vbramazons3compatibleservice.md) | Starts a new session with S3 compatible object storage. |
| [Disconnect-VBRAmazonS3Service](disconnect-vbramazons3service.md) | Stops active sessions with Amazon S3 and S3 compatible object storage. |
