# CDP Proxy

In this article

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRCDPProxy](add-vbrcdpproxy.md) | Adds CDP proxies to the backup infrastructure. |
| [Set-VBRCDPProxy](set-vbrcdpproxy.md) | Modifies settings of CDP proxies added to the backup infrastructure. |
| [Get-VBRCDPProxy](get-vbrcdpproxy.md) | Returns CDP proxies added to the backup infrastructure |
| [Enable-VBRCDPProxy](enable-vbrcdpproxy.md) | Enables CDP proxies that were disabled. |
| [Disable-VBRCDPProxy](disable-vbrcdpproxy.md) | Disables CDP proxies. |
| [Remove-VBRCDPProxy](remove-vbrcdpproxy.md) | Removes CDP proxies from the backup infrastructure. |
