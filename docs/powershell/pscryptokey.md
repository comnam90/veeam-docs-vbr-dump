# PSCryptoKey


Contains encryption key.

Transforms into VBREncryptionKey.

Properties

| Property | Type | Description |
| --- | --- | --- |
| Id | GUID | Encryption key ID. |
| Description | string | Encryption key description. |
| EncryptedPassword | string | Password in the SecureString format. |
| ModificationDateUtc | DateTime | Last modification date and time. |
