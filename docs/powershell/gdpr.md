# Locations

In this article

Manage geographic locations of Veeam Backup & Replication infrastructure objects for both VMware and Hyper-V environments.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRLocation](get-vbrlocation.md) | Returns geographic locations |
| [Add-VBRLocation](add-vbrlocation.md) | Creates a geographic location |
| [Set-VBRLocation](set-vbrlocation.md) | Modifies geographic locations |
| [Remove-VBRLocation](remove-vbrlocation.md) | Removes geographic locations |
| [Apply-VBRLocation](apply-vbrlocation.md) | Assigns geographic locations to Veeam Backup & Replication infrastructure objects |
| [Export-VBRLocation](export-vbrlocation.md) | Exports geographic locations to an XML file |
| [Import-VBRLocation](import-vbrlocation.md) | Imports geographic locations from an XML file |
