# VBRCloudCertificate

In this article

Contains cloud SSL certificate. The type is a Microsoft X509Certificate2 class.

For more information about the Microsoft X509Certificate2 class, see [Microsoft Docs](https://msdn.microsoft.com/en-us/library/system.security.cryptography.x509certificates.x509certificate2).
