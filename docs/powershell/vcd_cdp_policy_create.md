# Cloud Director CDP Policy


You can use the cmdlet in this topic to perform the following operations.

Cloud Director CDP Policy

| Cmdlet | Operation |
| [Add-VBRvCDCDPPolicy](add-vbrvcdcdppolicy.md) | Creates a Cloud Director CDP policy. |
| [Get-VBRCDPPolicy](get-vbrcdppolicy.md) | Returns Cloud Director CDP policies. |
| [Set-VBRvCDCDPPolicy](set-vbrvcdcdppolicy.md) | Modifies a Cloud Director CDP policy. |
| [Enable-VBRCDPPolicy](enable-vbrcdppolicy.md) | Enables Cloud Director CDP policies. |
| [Disable-VBRCDPPolicy](disable-vbrcdppolicy.md) | Disables Cloud Director CDP policies. |
| [Remove-VBRCDPPolicy](remove-vbrcdppolicy.md) | Removes Cloud Director CDP policies. |
