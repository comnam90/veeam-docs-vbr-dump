# HPE 3PAR StoreServ Storage Systems

In this article

Add HPE 3PAR StoreServ storage systems to your Veeam Backup & Replication console and administrate them.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Add-HP3Storage](add-hp3storage.md) | Adds HPE 3PAR StoreServ storage systems to Veeam Backup & Replication. |
| [Get-HP3Storage](get-hp3storage.md) | Returns HPE 3PAR StoreServ storage systems added to Veeam Backup & Replication. |
| [Sync-HP3Storage](sync-hp3storage.md) | Starts a rescan session of HPE 3PAR StoreServ storage systems. |
| [Set-HP3Storage](set-hp3storage.md) | Modifies HPE 3PAR StoreServ storage settings. |
| [Remove-HP3Storage](remove-hp3storage.md) | Removes HPE 3PAR StoreServ storage systems. |
| [Get-HP3Volume](get-hp3volume.md) | Returns HPE 3PAR StoreServ storage volumes from the Veeam Backup & Replication infrastructure. |
| [Get-HP3InfrastructureVolume](get-hp3infrastructurevolume.md) | Returns storage volumes from the HPE 3PAR StoreServ infrastructure. |
| [Sync-HP3Volume](sync-hp3volume.md) | Starts a rescan session of HPE 3PAR StoreServ volumes. |
| [Add-HP3Snapshot](add-hp3snapshot.md) | Creates HPE 3PAR StoreServ storage snapshots. |
| [Get-HP3Snapshot](get-hp3snapshot.md) | Returns HPE 3PAR StoreServ storage snapshots. |
| [Remove-HP3Snapshot](remove-hp3snapshot.md) | Removes HPE 3PAR StoreServ storage snapshots. |
| [New-VBRStorageProtocolPolicy](new-vbrstorageprotocolpolicy.md) | Creates protocol policies for storage systems. |
