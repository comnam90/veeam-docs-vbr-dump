# Adding 11:11 Cloud Object Storage


You can use the cmdlets in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBR1111SystemsS3Repository](add-vbr1111systemss3repository.md) | Adds 11:11 Cloud Object Storage to the backup infrastructure. |
| [Set-VBR1111SystemsS3Repository](set-vbr1111systemss3repository.md) | Modifies settings of 11:11 Cloud Object Storage added to the backup infrastructure. |
