# Adding Veeam Data Cloud Vault

In this article

You can use the cmdlets in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRDataCloudVaultRepository](add-vbrdatacloudvaultrepository.md) | Adds a Veeam Data Cloud Vault to the backup infrastructure. |
| [Set-VBRDataCloudVaultRepository](set-vbrdatacloudvaultrepository.md) | Modifies a Veeam Data Cloud Vault added to the backup infrastructure. |
