# CDP Policy Management

In this article

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRCDPPolicy](get-vbrcdppolicy.md) | Returns CDP policies including Cloud Director CDP policies, CDP policies, cloud CDP policies and so on. |
| [Enable-VBRCDPPolicy](enable-vbrcdppolicy.md) | Enables CDP policies including Cloud Director CDP policies, CDP policies, cloud CDP policies and so on. |
| [Disable-VBRCDPPolicy](disable-vbrcdppolicy.md) | Disables CDP policies including Cloud Director CDP policies, CDP policies, cloud CDP policies and so on. |
| [Remove-VBRCDPPolicy](remove-vbrcdppolicy.md) | Removes CDP policies including Cloud Director CDP policies, CDP policies, cloud CDP policies and so on. |
