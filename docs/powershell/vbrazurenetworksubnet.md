# VBRAzureNetworkSubnet

In this article

Contains Microsoft Azure virtual network subnet.

Properties

| Property | Type | Description |
| --- | --- | --- |
| NetworkName | string | Virtual network name. |
| SubscriptionId | GUID | ID of subscription with which the virtual network is associated. |
| Name | string | Virtual network subnet name. |
| Id | string | Subnet ID. |
