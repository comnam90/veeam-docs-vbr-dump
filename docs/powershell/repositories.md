# Backup Repositories

In this article

You can use the cmdlets in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRBackupRepository](add-vbrbackuprepository.md) | Adds a new backup repository. |
| [Get-VBRBackupRepository](get-vbrbackuprepository.md) | Looks for connected backup repositories. |
| [Set-VBRBackupRepository](set-vbrbackuprepository.md) | Modifies a backup repository. |
| [Remove-VBRBackupRepository](remove-vbrbackuprepository.md) | Removes a backup repository. |
| [Sync-VBRBackupRepository](sync-vbrbackuprepository.md) | Rescans a backup repository. |
| [Set-VBRImmutabilityLockExpirationDate](set-vbrimmutabilitylockexpirationdate.md) | Modifies immutability period of restore points located in a hardened repository. |
