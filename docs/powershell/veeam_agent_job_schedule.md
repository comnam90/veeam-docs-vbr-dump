# Veeam Agent Job Schedule


You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [New-VBRWindowsWorkstationScheduleOptions](new-vbrwindowsworkstationscheduleoptions.md) | Creates a schedule of the backup policy for Windows workstations. |
| [Set-VBRWindowsWorkstationScheduleOptions](set-vbrwindowsworkstationscheduleoptions.md) | Modifies the schedule of the backup policy for Windows workstations. |
| [New-VBRLinuxScheduleOptions](new-vbrlinuxscheduleoptions.md) | Creates a schedule for Veeam Agent jobs that back up Linux workstations. |
| [Set-VBRLinuxScheduleOptions](set-vbrlinuxscheduleoptions.md) | Modifies the schedule for Veeam Agent backup jobs that back up Linux workstations. |
| [New-VBRServerScheduleOptions](new-vbrserverscheduleoptions.md) | Creates a schedule for Veeam Agent backup jobs. |
| [Set-VBRServerScheduleOptions](set-vbrserverscheduleoptions.md) | Modifies the schedule for Veeam Agent backup jobs. |
| [New-VBRMacScheduleOptions](new-vbrmacscheduleoptions.md) | Defines the schedule for macOS jobs. |
| [Set-VBRMacScheduleOptions](set-vbrmacscheduleoptions.md) | Modifies schedule for macOS jobs. |
| [New-VBRUnixScheduleOptions](new-vbrunixscheduleoptions.md) | Defines the schedule for Unix jobs. |
