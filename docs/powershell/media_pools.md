# Tape Media Pools


You can use the cmdlets from this section to perform the following operations with media pools:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRTapeMediaPool](add-vbrtapemediapool.md) | Creates standard media pools. |
| [New-VBRTapeMediaPoolRetentionPolicy](new-vbrtapemediapoolretentionpolicy.md) | Creates retention settings for standard media pools. |
| [New-VBRTapeMediaSetCreationPolicy](new-vbrtapemediasetcreationpolicy.md) | Sets rules for creating media sets for standard media pools. |
| [Set-VBRTapeMediaPool](set-vbrtapemediapool.md) | Modifies standard media pool. |
| [Add-VBRTapeGFSMediaPool](add-vbrtapegfsmediapool.md) | Creates GFS media pool. |
| [New-VBRTapeGFSMediaSetPolicy](new-vbrtapegfsmediasetpolicy.md) | Creates GFS media set options. |
| [Set-VBRTapeGFSMediaPool](set-vbrtapegfsmediapool.md) | Modifies GFS media pool. |
| [Get-VBRTapeMediaPool](get-vbrtapemediapool.md) | Returns media pools. |
| [Remove-VBRTapeMediaPool](remove-vbrtapemediapool.md) | Removes media pools. |
