# Adding File Shares


You can use the cmdlets in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRNASSMBServer](add-vbrnassmbserver.md) | Adds SMB network shared folders to the inventory. |
| [Set-VBRNASSMBServer](set-vbrnassmbserver.md) | Modifies settings of SMB network shared folders added to the inventory. |
| [Add-VBRNASNFSServer](add-vbrnasnfsserver.md) | Adds NFS network shared folders to the inventory. |
| [Set-VBRNASNFSServer](set-vbrnasnfsserver.md) | Modifies settings of NFS shared folders added to the inventory. |
| [Get-VBRNASServerPath](get-vbrnasserverpath.md) | Returns files paths to NFS network shared folders. |
