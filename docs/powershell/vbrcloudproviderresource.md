# VBRCloudProviderResource

In this article

Contains cloud provider settings.

Properties

| Property | Type | Description |
| --- | --- | --- |
| RepositoryName | string | Name of the cloud repository presented to the user. |
| RepositoryAllocatedSpace | int | Repository quota. |
| WanAcceleratorEnabled | bool | Indicates if the WAN accelerator in enabled (TRUE) or disabled (FALSE). |
