# Veeam Cloud Connect Replication


You can use the cmdlets from this section to perform the following operations with cloud replication jobs:

| VMware | Hyper-V | Operation |
| --- | --- | --- |
| [Add-VBRViCloudReplicaJob](add-vbrvicloudreplicajob.md) | [Add-VBRHvCloudReplicaJob](add-vbrhvcloudreplicajob.md) | Creates a replication job |
| [Set-VBRViCloudReplicaJob](set-vbrvicloudreplicajob.md) | [Set-VBRHvCloudReplicaJob](set-vbrhvcloudreplicajob.md) | Modifies a replication job |
| [Get-VBRCloudDatastore](get-vbrclouddatastore.md) |  | Returns datastores or disk volumes available by your cloud resources |
| [Get-VBRCloudServer](get-vbrcloudserver.md) |  | Returns cloud hosts available by your cloud resources |
| [Get-VBRCloudServerNetworkInfo](get-vbrcloudservernetworkinfo.md) |  | Returns networks available in your cloud resources |
