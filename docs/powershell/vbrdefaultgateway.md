# VBRDefaultGateway


Contains default gateway.

Properties

| Property | Type | Description |
| --- | --- | --- |
| IpAddress | string | Default gateway IP address. |
| NetworkMask | string | Default gateway network mask. |
| Ipv6Address | string | The IPv6 address of the default gateway. |
| Ipv6SubnetAddress | string | The production IPv6 network. |
| Ipv6PrefixLength | byte | The length of an IPv6 prefix. |
| Name | string | Default gateway name. |
| Id | GUID | Default gateway ID. |
