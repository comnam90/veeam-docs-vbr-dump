# Network Traffic Management


You can use the cmdlets in this topic to perform the following operations:

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRNetworkTrafficRule](add-vbrnetworktrafficrule.md) | Creates network traffic rules. |
| [Set-VBRNetworkTrafficRule](set-vbrnetworktrafficrule.md) | Modifies network traffic rules. |
| [Get-VBRNetworkTrafficRule](get-vbrnetworktrafficrule.md) | Returns the array of network traffic rules. |
| [Get-VBRNetworkTrafficRuleOptions](get-vbrnetworktrafficruleoptions.md) | Returns settings of global network traffic rules. |
| [Validate-VBRInfrastructure](validate-vbrinfrastructure.md) | Validates the network infrastructure of a backup server. |
| [Set-VBRNetworkTrafficRuleOptions](set-vbrnetworktrafficruleoptions.md) | Modifies settings of global network traffic rules. |
| [Remove-VBRNetworkTrafficRule](remove-vbrnetworktrafficrule.md) | Removes network traffic rules. |
| [Add-VBRPreferredNetwork](add-vbrpreferrednetwork.md) | Creates the settings for the preferred network. |
| [Get-VBRPreferredNetwork](get-vbrpreferrednetwork.md) | Returns the array of the preferred networks. |
| [Remove-VBRPreferredNetwork](remove-vbrpreferrednetwork.md) | Removes preferred networks. |
| [Enable-VBRPreferredNetworkDataTransfer](enable-vbrpreferrednetworkdatatransfer.md) | Enables the option to use the preferred network for the data transfer. |
| [Disable-VBRPreferredNetworkDataTransfer](disable-vbrpreferrednetworkdatatransfer.md) | Disables the option to use the preferred network for the data transfer. |
| [Get-VBRUploadStreamsConfiguration](get-vbruploadstreamsconfiguration.md) | Returns information on data transfer settings for job sessions. |
| [Set-VBRUploadStreamsConfiguration](set-vbruploadstreamsconfiguration.md) | Modifies data transfer settings for job sessions. |
