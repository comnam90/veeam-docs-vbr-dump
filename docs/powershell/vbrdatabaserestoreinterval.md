# VBRDatabaseRestoreInterval


Contains date and time of the first and the last available restore point of database.

Properties

| Property | Type | Description |
| --- | --- | --- |
| From | DateTime | Date and time (UTC) of creation of the first restore point for this database. |
| To | DateTime | Date and time (UTC) of creation of the last restore point for this database. |
