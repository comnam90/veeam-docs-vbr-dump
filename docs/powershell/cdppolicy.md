# VMware CDP Policy

In this article

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [New-VBRCDPPolicyRetentionOptions](new-vbrcdppolicyretentionoptions.md) | Defines retention setting of CDP policies. |
| [Set-VBRCDPPolicyRetentionOptions](set-vbrcdppolicyretentionoptions.md) | Modifies retention setting of CDP policies. |
| [New-VBRIndividualGuestOSCredentials](new-vbrindividualguestoscredentials.md) | Defines a user account that will be explicitly used to connect to specific machine guest OSes |
| [New-VBRReplicaApplicationProcessingOptions](new-vbrreplicaapplicationprocessingoptions.md) | Defines application processing settings for a CDP policy. |
| [Set-VBRReplicaApplicationProcessingOptions](set-vbrreplicaapplicationprocessingoptions.md) | Modifies application processing settings for a CDP policy. |
| [Add-VBRCDPPolicy](add-vbrcdppolicy.md) | Creates CDP policies for VMware vSphere VMs. |
| [Set-VBRCDPPolicy](set-vbrcdppolicy.md) | Modifies CDP policies for VMware vSphere VMs. |
