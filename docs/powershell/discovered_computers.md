# Veeam Agent Management Discovered Computers

In this article

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRDiscoveredComputer](get-vbrdiscoveredcomputer.md) | Returns discovered computers. |
| [Reset-VBRDiscoveredComputer](reset-vbrdiscoveredcomputer.md) | Reboots discovered computers. |
| [Install-VBRDiscoveredComputerAgent](install-vbrdiscoveredcomputeragent.md) | Installs backup agents on discovered computers. |
| [Uninstall-VBRDiscoveredComputerAgent](uninstall-vbrdiscoveredcomputeragent.md) | Removes Veeam Agent from a specific protected computer. |
| [Reset-VBRAgentCertificate](reset-vbragentcertificate.md) | Resets backup agents' SSL certificates. |
| [Get-VBRDiscoveredComputerUpdate](get-vbrdiscoveredcomputerupdate.md) | Returns available agent private fixes. |
| [Set-VBRDiscoveredComputerUpdate](set-vbrdiscoveredcomputerupdate.md) | Creates an association between private fixes and discovered computers. |
| [New-VBRDiscoveredComputerConfigurationOption](new-vbrdiscoveredcomputerconfigurationoption.md) | Creates new configuration options for Veeam Agent settings. |
| [Add-VBRDiscoveredComputerConfigurationPolicy](add-vbrdiscoveredcomputerconfigurationpolicy.md) | Updates Veeam Agent settings on discovered computers using available configuration options. |
| [Get-VBRDiscoveredComputerConfigurationPolicy](get-vbrdiscoveredcomputerconfigurationpolicy.md) | Returns configuration options applied to protected computers. |
| [Remove-VBRDiscoveredComputerConfigurationPolicy](remove-vbrdiscoveredcomputerconfigurationpolicy.md) | Removes policies that contain configuration options for Veeam Agent settings. |
