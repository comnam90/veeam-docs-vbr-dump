# NetApp Storage Systems


Add NetApp storage systems to your  Veeam Backup & Replication console and administrate them.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Add-NetAppHost](add-netapphost.md) | Adds NetApp storage systems to Veeam Backup & Replication. |
| [Get-NetAppHost](get-netapphost.md) | Returns NetApp storage systems added to Veeam Backup & Replication. |
| [Set-NetAppHost](set-netapphost.md) | Modifies NetApp storage settings. |
| [Sync-NetAppHost](sync-netapphost.md) | Starts a rescan session of NetApp storage systems. |
| [Remove-NetAppHost](remove-netapphost.md) | Removes NetApp storage systems. |
| [Get-NetAppVolume](get-netappvolume.md) | Returns NetApp storage volumes from the Veeam Backup & Replication infrastructure. |
| [Get-NetAppInfrastructureVolume](get-netappinfrastructurevolume.md) | Returns storage volumes from the NetApp storage infrastructure. |
| [Sync-NetAppVolume](sync-netappvolume.md) | Starts a rescan session of NetApp storage volumes from the Veeam Backup & Replication infrastructure. |
| [Add-NetAppSnapshot](add-netappsnapshot.md) | Creates NetApp storage snapshots. |
| [Get-NetAppSnapshot](get-netappsnapshot.md) | Looks for NetApp storage snapshots. |
| [Remove-NetAppSnapshot](remove-netappsnapshot.md) | Removes NetApp storage snapshots. |
| [New-VBRStorageProtocolPolicy](new-vbrstorageprotocolpolicy.md) | Creates protocol policies for storage systems. |
