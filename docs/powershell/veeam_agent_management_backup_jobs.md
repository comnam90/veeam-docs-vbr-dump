# Veeam Agent Management Backup Jobs


You can use the cmdlet in this topic to perform the following operations.

|  |
| --- |
| Tip |
| To create and manage backup copy jobs for Veeam Agent backups, use cmdlets described in [Backup Copy Jobs for VMs and Physical Machines](backup_copy.md). |

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRComputerBackupJob](add-vbrcomputerbackupjob.md) | Creates Veeam Agent backup jobs and Veeam Agent backup policies. |
| [Set-VBRComputerBackupJob](set-vbrcomputerbackupjob.md) | Modifies Veeam Agent backup jobs and Veeam Agent backup policies. |
| [Get-VBRComputerBackupJob](get-vbrcomputerbackupjob.md) | Returns Veeam Agent backup jobs and Veeam Agent backup policies. |
| [Get-VBRComputerBackupJobSession](get-vbrcomputerbackupjobsession.md) | Returns jobs sessions. |
| [Start-VBRComputerBackupJob](start-vbrcomputerbackupjob.md) | Starts Veeam Agent backup jobs and Veeam Agent backup policies. |
| [Stop-VBRComputerBackupJob](stop-vbrcomputerbackupjob.md) | Stops Veeam Agent backup jobs and Veeam Agent backup policies. |
| [Copy-VBRComputerBackupJob](copy-vbrcomputerbackupjob.md) | Clones Veeam Agent backup jobs and Veeam Agent backup policies. |
| [Remove-VBRComputerBackupJob](remove-vbrcomputerbackupjob.md) | Removes Veeam Agent backup jobs and Veeam Agent backup policies. |
| [Apply-VBRManagedByAgentPolicy](apply-vbrmanagedbyagentpolicy.md) | Assigns Veeam Agent backup policies to protected computers. |
| [Clear-VBRManagedByAgentPolicyCache](clear-vbrmanagedbyagentpolicycache.md) | Removes cache from the protected computers. |
| [Enable-VBRComputerBackupJob](enable-vbrcomputerbackupjob.md) | Enables Veeam Agent backup jobs and Veeam Agent backup policies. |
| [Disable-VBRComputerBackupJob](disable-vbrcomputerbackupjob.md) | Disables Veeam Agent backup jobs and Veeam Agent backup policies. |
