# Veeam Plug-In Job Sessions

In this article

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRPluginBackupSession](get-vbrpluginbackupsession_standalone.md) | Returns sessions of application backup policies and backup jobs for standalone Veeam Plug-Ins. |
| [Get-VBRPluginRestoreSession](get-vbrpluginrestoresession_standalone.md) | Returns active restore sessions for standalone and managed Veeam Plug-Ins. |
