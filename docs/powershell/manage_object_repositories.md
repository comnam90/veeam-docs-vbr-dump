# Managing Object Storage Repositories


You can use the cmdlets in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRObjectStorageRepository](get-vbrobjectstoragerepository.md) | Returns object storage added as backup repositories to Veeam Backup & Replication. |
| [Get-VBRArchiveObjectStorageRepository](get-vbrarchiveobjectstoragerepository.md) | Returns archive storage added as backup repositories to Veeam Backup & Replication. |
| [Remove-VBRObjectStorageRepository](remove-vbrobjectstoragerepository.md) | Removes object storage added as backup repositories to Veeam Backup & Replication. |
| [Remove-VBRArchiveObjectStorageRepository](remove-vbrarchiveobjectstoragerepository.md) | Removes archive storage added as backup repositories to Veeam Backup & Replication. |
| [Mount-VBRObjectStorageRepository](mount-vbrobjectstoragerepository.md) | Mounts object storage and archive repositories. |
| [Unmount-VBRObjectStorageRepository](unmount-vbrobjectstoragerepository.md) | Unmounts object storage and archive repositories. |
| [Get-VBRS3CompatibleRepositorySecurityOptions](get-vbrs3compatiblerepositorysecurityoptions.md) | Returns settings that Veeam Agent uses to access S3 compatible Compatible repositories. |
| [Set-VBRS3CompatibleRepositorySecurityOptions](set-vbrs3compatiblerepositorysecurityoptions.md) | Modifies settings that Veeam Agent uses to access S3 compatible Compatible repositories. |
