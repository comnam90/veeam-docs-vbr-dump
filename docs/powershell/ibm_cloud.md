# Adding IBM Cloud Object Storage


You can use the cmdlets in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRIBMCloudRepository](add-vbribmcloudrepository.md) | Adds the IBM Cloud object storage repository to the backup infrastructure. |
| [Set-VBRIBMCloudRepository](set-vbribmcloudrepository.md) | Modifies settings of the IBM Cloud object storage repository added to the backup infrastructure. |
