# Advanced Job Options


Configure advanced job options. The cmdlets in this section match the tabs of the Advanced settings in the job wizards.

| VMware Cmdlet | Hyper-V Cmdlet | Operation |
| --- | --- | --- |
| [Set-VBRJobAdvancedBackupOptions](set-vbrjobadvancedbackupoptions.md) |  | Sets advanced backup options for a selected job |
| [Set-VBRJobAdvancedViOptions](set-vbrjobadvancedvioptions.md) | [Set-VBRJobAdvancedHvOptions](set-vbrjobadvancedhvoptions.md) | Sets special options for a selected job |
| [Set-VBRJobAdvancedNotificationOptions](set-vbrjobadvancednotificationoptions.md) |  | Sets advanced notification options for a selected job |
| [Set-VBRJobAdvancedOptions](set-vbrjobadvancedoptions.md) |  | Sets advanced options for a selected job |
| [Set-VBRJobAdvancedStorageOptions](set-vbrjobadvancedstorageoptions.md) |  | Sets advanced storage options for a selected job |
| [New-VBRHealthCheckOptions](new-vbrhealthcheckoptions.md) |  | Defines the health check schedule options. |
| [Set-VBRHealthCheckOptions](set-vbrhealthcheckoptions.md) |  | Modifies the health check schedule options. |
