# Veeam Data Movers for Guest Processing


You can use the cmdlets in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Install-VBRLinuxTransportService](install-vbrlinuxtransportservice.md) | Installs Veeam Data Mover for guest processing on Linux VMs. |
| [Uninstall-VBRLinuxTransportService](uninstall-vbrlinuxtransportservice.md) | Uninstalls guest processing Veeam Data Mover from Linux VMs. |
