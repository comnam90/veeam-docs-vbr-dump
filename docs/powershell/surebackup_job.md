# SureBackup Jobs

In this article

|  |
| --- |
| Note |
| SureBackup requires Enterprise or Enterprise Plus, Veeam Universal License product edition. |

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRSureBackupJob](add-vbrsurebackupjob.md) | Creates a SureBackup job. |
| [Get-VBRSureBackupJob](get-vbrsurebackupjob.md) | Returns SureBackup jobs. |
| [Set-VBRSureBackupJob](set-vbrsurebackupjob.md) | Modifies settings of SureBackup jobs. |
| [Disable-VBRSureBackupJob](disable-vbrsurebackupjob.md) | Disables running SureBackup jobs. |
| [Enable-VBRSureBackupJob](enable-vbrsurebackupjob.md) | Enables stopped SureBackup jobs. |
| [Start-VBRSureBackupJob](start-vbrsurebackupjob.md) | Starts the SureBackup job. |
| [Stop-VBRSureBackupJob](stop-vbrsurebackupjob.md) | Stops the SureBackup job. |
| [Remove-VBRSureBackupJob](remove-vbrsurebackupjob.md) | Removes SureBackup jobs. |
| [New-VBRSureBackupJobScheduleOptions](new-vbrsurebackupjobscheduleoptions.md) | Defines the SureBackup job schedule. |
| [Set-VBRSureBackupJobScheduleOptions](set-vbrsurebackupjobscheduleoptions.md) | Modifies settings of the SureBackup job schedule. |
| [New-VBRSureBackupJobVerificationOptions](new-vbrsurebackupjobverificationoptions.md) | Defines additional settings for the SureBackup job. |
| [Set-VBRSureBackupJobVerificationOptions](set-vbrsurebackupjobverificationoptions.md) | Modifies additional settings for the SureBackup job. |
| [New-VBRSureBackupLinkedJob](new-vbrsurebackuplinkedjob.md) | Defines jobs linked with the SureBackup job. |
| [Set-VBRSureBackupLinkedJob](set-vbrsurebackuplinkedjob.md) | Modifies settings of jobs linked with the SureBackup job. |
| [Get-VBRSureBackupSession](get-vbrsurebackupsession.md) | Returns SureBackup sessions that have been run. |
| [Get-VBRSureBackupTaskSession](get-vbrsurebackuptasksession.md) | Returns tasks performed during the specified SureBackup session. |
