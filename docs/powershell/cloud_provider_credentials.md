# Cloud Provider Credentials

In this article

You can use the cmdlets from this section to add the cloud provider credentials into your Veeam Backup & Replication infrastructure and modify these credentials.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRCloudProviderCredentials](add-vbrcloudprovidercredentials.md) | Adds cloud provider credentials records |
| [Get-VBRCloudProviderCredentials](get-vbrcloudprovidercredentials.md) | Returns an array of cloud provider credential records |
| [Set-VBRCloudProviderCredentials](set-vbrcloudprovidercredentials.md) | Modifies cloud provider credential records |
| [Remove-VBRCloudProviderCredentials](remove-vbrcloudprovidercredentials.md) | Removes cloud provider credential records |

Page updated 9/4/2023
