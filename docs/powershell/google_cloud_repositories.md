# Adding Google Cloud Object Storage

In this article

You can use the cmdlets in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRGoogleCloudRepository](add-vbrgooglecloudrepository.md) | Adds Google Cloud object storage to the backup infrastructure. |
| [Set-VBRGoogleCloudRepository](set-vbrgooglecloudrepository.md) | Modifies settings for Google Cloud object storage repository added to the backup infrastructure. |
