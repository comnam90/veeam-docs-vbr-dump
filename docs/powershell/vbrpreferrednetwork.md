# VBRPreferredNetwork


Contains details of a preferred network for the backup infrastructure.

Properties

| Property | Type | Description |
| --- | --- | --- |
| IpAddress | string | The network IP address. |
| SubnetMask | string | The network subnet mask. |
| CIDRNotation | string | The network CIDR notation. |
