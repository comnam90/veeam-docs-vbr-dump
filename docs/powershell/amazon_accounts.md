# Managing Amazon Accounts


The cmdlets in this section allow you to get and manage the Amazon S3 credential records.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRAmazonAccount](add-vbramazonaccount.md) | Creates Amazon credentials records. |
| [Get-VBRAmazonAccount](get-vbramazonaccount.md) | Returns the array of Amazon S3 credentials records. |
| [Set-VBRAmazonAccount](set-vbramazonaccount.md) | Modifies Amazon S3 credentials records. |
| [Remove-VBRAmazonAccount](remove-vbramazonaccount.md) | Removes Amazon S3 credentials records. |
