# Virtualization Servers and Hosts

In this article

You can use the cmdlets in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRvCenter](add-vbrvcenter.md) | Adds vCenter Servers to the backup infrastructure. |
| [Set-VBRvCenter](set-vbrvcenter.md) | Modifies VMware vCenter servers added to the backup infrastructure. |
| [Add-VBRESXi](add-vbresxi.md) | Adds VMware ESXi hosts to the backup infrastructure. |
| [Set-VBRESXi](set-vbresxi.md) | Modifies ESXi hosts added to the backup infrastructure. |
| [Add-VBRHvScvmm](add-vbrhvscvmm.md) | Adds a Hyper-V SCVMM server to the backup infrastructure. |
| [Set-VBRHvScvmm](set-vbrhvscvmm.md) | Modifies settings of a Hyper-V SCVMM server. |
| [Add-VBRHvCluster](add-vbrhvcluster.md) | Adds Hyper-V clusters to the backup infrastructure. |
| [Set-VBRHvCluster](set-vbrhvcluster.md) | Modifies settings of a Hyper-V cluster. |
| [Add-VBRSmbV3Cluster](add-vbrsmbv3cluster.md) | Adds an SMB3 cluster to the backup infrastructure. |
| [Add-VBRSmbV3Host](add-vbrsmbv3host.md) | Adds an SMB3 host to the backup infrastructure. |
| [Add-VBRHvHost](add-vbrhvhost.md) | Adds Hyper-V hosts to the backup infrastructure. |
| [Set-VBRHvHost](set-vbrhvhost.md) | Modifies settings of Hyper-V hosts. |
| [Add-VBRWinServer](add-vbrwinserver.md) | Adds Windows server to the backup infrastructure. |
| [Set-VBRWinServer](set-vbrwinserver.md) | Modifies Windows server added to the backup infrastructure. |
| [Add-VBRLinux](add-vbrlinux.md) | Adds a Linux server to the backup infrastructure. |
| [Set-VBRLinux](set-vbrlinux.md) | Modifies settings of a Linux server added to the backup infrastructure. |
| [Get-VBRLocalhost](get-vbrlocalhost.md) | Returns the local server on which Veeam Backup & Replication is installed. |
| [Get-VBRServer](get-vbrserver.md) | Returns hosts connected to Veeam Backup & Replication. |
| [Get-VBRBackupServerInfo](get-vbrbackupserverinfo.md) | Returns the backup server information. |
| [Remove-VBRServer](remove-vbrserver.md) | Removes hosts from the backup infrastructure. |
| [Get-VBRHvServerConfiguration](get-vbrhvserverconfiguration.md) | Returns settings of Microsoft Hyper-V hosts added to the backup infrastructure. |
| [Set-VBRHvServerConfiguration](set-vbrhvserverconfiguration.md) | Modifies settings of Microsoft Hyper-V hosts added to the backup infrastructure. |
| [Get-VBRHvServerVolume](get-vbrhvservervolume.md) | Returns volume-specific settings for Microsoft Hyper-V hosts. |
| [Get-VBRHvVssProvider](get-vbrhvvssprovider.md) | Returns VSS providers. |
| [Set-VBRHvServerVolume](set-vbrhvservervolume.md) | Modifies volume-specific settings for Microsoft Hyper-V hosts. |
| [Uninstall-VBRServerComponent](uninstall-vbrservercomponent.md) | Removes a backup infrastructure component from the backup infrastructure. |
| [Get-VBRLinuxPackage](get-vbrlinuxpackage.md) | Returns information about Linux components installed on the backup server. |
