# Server Components Upgrade


You can use the cmdlet in this topic to perform the following operations.

Server Components Upgrade

| Cmdlet | Operation |
| [Get-VBRPhysicalHost](get-vbrphysicalhost.md) | Returns managed servers added to the backup infrastructure. |
| [Update-VBRServerComponent](update-vbrservercomponent.md) | Upgrades all components on managed servers. |
