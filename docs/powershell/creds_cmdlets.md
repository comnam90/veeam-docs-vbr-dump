# Credentials


You can use the cmdlet in this topic to perform the following operations:

Credentials

| Cmdlet | Operation |
| [Add-VBRCredentials](add-vbrcredentials.md) | Creates credentials records. |
| [Get-VBRCredentials](get-vbrcredentials.md) | Returns credentials records. |
| [Set-VBRCredentials](set-vbrcredentials.md) | Edits credentials records properties. |
| [Remove-VBRCredentials](remove-vbrcredentials.md) | Removes a credentials record from Veeam Backup & Replication. |
