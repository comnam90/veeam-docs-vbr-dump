# Veeam Agent Job Guest Processing Options


You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [New-VBRApplicationProcessingOptions](new-vbrapplicationprocessingoptions.md) | Creates application-aware processing settings for Veeam Agent backup jobs. |
| [New-VBROracleProcessingOptions](new-vbroracleprocessingoptions.md) | Creates settings for processing Oracle database archived logs. |
| [New-VBRSQLProcessingOptions](new-vbrsqlprocessingoptions.md) | Creates settings for processing Microsoft SQL database transaction logs. |
| [New-VBRPostgreSQLProcessingOptions](new-vbrpostgresqlprocessingoptions.md) | Defines settings for processing backed-up PostgreSQL databases on Linux computers. |
| [New-VBRMySQLProcessingOptions](new-vbrmysqlprocessingoptions.md) | Defines settings for processing backed-up MySQL databases. |
| [New-VBRScriptProcessingOptions](new-vbrscriptprocessingoptions.md) | Creates pre-freeze and post-thaw scripts settings for Veeam Agent backup jobs. |
| [New-VBRComputerIndexingOptions](new-vbrcomputerindexingoptions.md) | Creates indexing settings for Veeam Agent backup job. |
| [Set-VBRApplicationProcessingOptions](set-vbrapplicationprocessingoptions.md) | Modifies application-aware processing settings for Veeam Agent backup jobs. |
| [Set-VBROracleProcessingOptions](set-vbroracleprocessingoptions.md) | Modifies settings for processing Oracle database archived logs. |
| [Set-VBRSQLProcessingOptions](set-vbrsqlprocessingoptions.md) | Modifies settings for processing Microsoft SQL database transaction logs. |
| [Set-VBRPostgreSQLProcessingOptions](set-vbrpostgresqlprocessingoptions.md) | Modifies settings for processing backed-up PostgreSQL databases. |
| [Set-VBRMySQLProcessingOptions](set-vbrmysqlprocessingoptions.md) | Modifies settings for processing backed-up MySQL databases. |
| [Set-VBRScriptProcessingOptions](set-vbrscriptprocessingoptions.md) | Modifies pre-freeze and post-thaw scripts settings for Veeam Agent backup jobs. |
| [Set-VBRComputerIndexingOptions](set-vbrcomputerindexingoptions.md) | Modifies indexing settings for Veeam Agent backup job. |
