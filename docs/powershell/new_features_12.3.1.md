# New Features


This section contains information on new features that were introduced in Veeam PowerShell v12.3.1

Backup Infrastructure

Object Storage Repositories

In this version, you can run the [Get-VBRVeeamDataCloudAssignedVault](get-vbrveeamdatacloudassignedvault.md) cmdlet to add Veeam Data Cloud Vault.

Backup

Managing Backups

In this version, you can run the [Remove-VBREncryptedBackup](remove-vbrencryptedbackup.md) cmdlet to remove encrypted backups.
