In this article

You can use the cmdlets in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |

| [Set-VBRNASFileServer](set-vbrnasfileserver.md) | Modifies managed Windows or Linux file serves added to the inventory. |
