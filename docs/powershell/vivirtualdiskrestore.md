# Virtual Disks Restore

In this article

You can use the cmdlets in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Start-VBRViVirtualDiskRestore](start-vbrvivirtualdiskrestore.md) | Starts restore of backed-up virtual disks. |
| [New-VBRViVirtualDeviceMappingRule](new-vbrvivirtualdevicemappingrule.md) | Defines backed-up virtual disk mapping settings. |
| [Set-VBRViVirtualDeviceMappingRule](set-vbrvivirtualdevicemappingrule.md) | Modifies backed-up virtual disk mapping settings. |
| [Start-VBRRestoreVirtualDisks](start-vbrrestorevirtualdisks_vm.md) | Restores disks from backups of physical or virtual machines and converts them to the VMDK, VHD or VHDX formats. |
| [New-VBRHvVirtualDeviceMappingRule](new-vbrhvvirtualdevicemappingrule.md) | Defines backed-up virtual disk mapping settings. |
