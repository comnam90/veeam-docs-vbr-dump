# Licenses

In this article

You can use the cmdlets in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Install-VBRLicense](install-vbrlicense.md) | Installs licenses on a backup server. |
| [Get-VBRInstalledLicense](get-vbrinstalledlicense.md) | Returns details about licenses installed on a backup server. |
| [Uninstall-VBRLicense](uninstall-vbrlicense.md) | Removes a license from a backup server. |
| [Update-VBRLicense](update-vbrlicense.md) | Updates licenses. |
| [Generate-VBRLicenseUsageReport](generate-vbrlicenseusagereport.md) | Creates reports on license usage. |
| [Get-VBRSocketLicenseSummary](get-vbrsocketlicensesummary.md) | Returns details on the per-socket license usage. |
| [Get-VBRInstanceLicenseSummary](get-vbrinstancelicensesummary.md) | Returns details on the per-instance license usage. |
| [Get-VBRLicensedSocketWorkload](get-vbrlicensedsocketworkload.md) | Returns details on licensed hosts. |
| [Get-VBRLicensedInstanceWorkload](get-vbrlicensedinstanceworkload.md) | Returns details on protected workloads. |
| [Enable-VBRFreeAgentInstanceConsumption](enable-vbrfreeagentinstanceconsumption.md) | Enables instance consumption by unlicensed Veeam Agents. |
| [Get-VBRFreeAgentInstanceConsumptionStatus](get-vbrfreeagentinstanceconsumptionstatus.md) | Returns a state of the instance consumption by non-licensed Veeam Agents. |
| [Disable-VBRFreeAgentInstanceConsumption](disable-vbrfreeagentinstanceconsumption.md) | Disables instance consumption by non-licensed Veeam Agents. |
| [Enable-VBRLicenseAutoUpdate](enable-vbrlicenseautoupdate.md) | Enables the automatic license update option. |
| [Get-VBRLicenseAutoUpdateStatus](get-vbrlicenseautoupdatestatus.md) | Returns a state of the automatic license update option. |
| [Disable-VBRLicenseAutoUpdate](disable-vbrlicenseautoupdate.md) | Disables the the automatic license update option. |
| [Assign-VBRInstanceWorkload](assign-vbrinstanceworkload.md) | Sets the product edition for standalone Veeam Agents. |
