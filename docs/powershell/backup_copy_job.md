# Backup Copy Job of Amazon EC2 Instances


You can use cmdlets in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBREC2Backup](get-vbrec2backup.md) | Returns EC2 instance backups. |
| [Set-VBREC2BackupCopyJob](set-vbrec2backupcopyjob.md) | Modifies backup copy jobs for backups of Amazon EC2 instances. |
| [New-VBRRecoveryPointObjective](new-vbrrecoverypointobjective.md) | Creates an interval for backup copy jobs that process backups of Amazon EC2 instances. |
| [Set-VBRRecoveryPointObjective](set-vbrrecoverypointobjective.md) | Modifies an interval for backup copy jobs that process backups of Amazon EC2 instances. |
| [Get-VBRRecoveryPointObjective](get-vbrrecoverypointobjective.md) | Returns a schedule for backup copy jobs. |
| [New-VBRSimpleRetentionPolicy](new-vbrsimpleretentionpolicy.md) | Creates a retention policy for backup copy jobs that process backups of Amazon EC2 instances. |
| [Get-VBRRetentionPolicy](get-vbrretentionpolicy.md) | Returns a retention policy for backup copy jobs that process backups of Amazon EC2 instances. |
| [Set-VBRSimpleRetentionPolicy](set-vbrsimpleretentionpolicy.md) | Modifies a retention policy for backup copy jobs. |
| [New-VBRGFSRetentionPolicy](new-vbrgfsretentionpolicy.md) | Creates a GFS retention policy for backup copy jobs that process backups of Amazon EC2 instances. |
| [Set-VBRGFSRetentionPolicy](set-vbrgfsretentionpolicy.md) | Modifies a GFS retention policy for backup copy jobs that process backups of Amazon EC2 instances. |
