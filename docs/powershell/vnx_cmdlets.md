# Dell VNX Storage Systems

In this article

Add Dell VNX storage systems to your Veeam Backup & Replication console and administrate them.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Add-VNXHost](add-vnxhost.md) | Adds Dell VNX storage systems to Veeam Backup & Replication. |
| [Get-VNXHost](get-vnxhost.md) | Returns Dell VNX storage systems. |
| [Set-VNXHost](set-vnxhost.md) | Modifies Dell VNX storage settings. |
| [Sync-VNXHost](sync-vnxhost.md) | Starts a rescan session of Dell VNX storage system. |
| [Remove-VNXHost](remove-vnxhost.md) | Removes Dell VNX storage systems from Veeam Backup & Replication. |
| [Get-VNXVolume](get-vnxvolume.md) | Returns Dell VNX storage volumes from the Veeam Backup & Replication infrastructure. |
| [Get-VNXInfrastructureVolume](get-vnxinfrastructurevolume.md) | Returns storage volumes from the Dell VNX storage infrastructure. |
| [Sync-VNXVolume](sync-vnxvolume.md) | Starts a rescan session of Dell VNX storage volumes from the Veeam Backup & Replication infrastructure. |
| [Add-VNXSnapshot](add-vnxsnapshot.md) | Creates Dell VNX storage snapshots. |
| [Get-VNXSnapshot](get-vnxsnapshot.md) | Returns Dell VNX storage snapshots. |
| [Remove-VNXSnapshot](remove-vnxsnapshot.md) | Removes Dell VNX storage snapshots. |
| [New-VBRStorageProtocolPolicy](new-vbrstorageprotocolpolicy.md) | Creates protocol policies for storage systems. |
