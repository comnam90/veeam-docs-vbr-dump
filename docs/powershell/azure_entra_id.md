# Managing Azure Entra ID Account


You can use the cmdlets in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRAzureADAccount](add-vbrazureadaccount.md) | Adds Azure Entra ID-based storage accounts to Veeam Backup & Replication. |
| [Get-VBRAzureADAccount](get-vbrazureadaccount.md) | Returns Azure Entra ID-based storage accounts. |
| [Set-VBRAzureADAccount](set-vbrazureadaccount.md) | Modifies settings of Azure Entra ID-based storage accounts. |
| [Remove-VBRAzureADAccount](remove-vbrazureadaccount.md) | Removes Azure Entra ID-based storage accounts from Veeam Backup & Replication. |
