# VBRDefaultGatewayConfiguration


Contains default gateway configuration.

Properties

| Property | Type | Description |

| ProviderId | GUID | Service provider ID. |
| RoutingEnabled | bool | Indicates if the routing is enabled for the default gateway. |
| DefaultGateway | [VBRDefaultGateway](vbrdefaultgateway.md)[] | Default gateway. |
