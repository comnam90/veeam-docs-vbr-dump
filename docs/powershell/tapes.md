# Managing Tapes


You can use the cmdlets from this section to perform the following operations with tapes:

| Cmdlet | Operation |
| --- | --- |
| [Disable-VBRTapeProtection](disable-vbrtapeprotection.md) | Disables overwrite protection. |
| [Eject-VBRTapeMedium](eject-vbrtapemedium.md) | Ejects a tape from a drive. |
| [Enable-VBRTapeProtection](enable-vbrtapeprotection.md) | Enables overwrite protection. |
| [Erase-VBRTapeMedium](erase-vbrtapemedium.md) | Erases tapes. |
| [Export-VBRTapeMedium](export-vbrtapemedium.md) | Exports tapes to mail slot. |
| [Get-VBRTapeMedium](get-vbrtapemedium.md) | Returns tapes. |
| [Import-VBRTapeMedium](import-vbrtapemedium.md) | Imports new tapes. |
| [Move-VBRTapeMedium](move-vbrtapemedium.md) | Moves tapes to media pool or vault. |
| [Remove-VBRTapeMedium](remove-vbrtapemedium.md) | Removes tapes. |
| [Set-VBRTapeMedium](set-vbrtapemedium.md) | Modifies tapes properties. |
| [Get-VBRTapeCopyDependentMedium](get-vbrtapecopydependentmedium.md) | Returns dependent tapes. |
| [Start-VBRTapeVerification](start-vbrtapeverification.md) | Starts tape verification jobs. |
| [Start-VBRTapeCopy](start-vbrtapecopy.md) | Starts tape copy jobs. |
