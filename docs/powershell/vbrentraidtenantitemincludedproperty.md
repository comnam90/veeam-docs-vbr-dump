# VBREntraIDTenantItemIncludedProperty

In this article

Contains item properties or references that you plan to restore.

Properties

| Property | Type | Description |
| --- | --- | --- |
| PropertyId | String | Specifies different values depending on what you plan to restore:   * [For property restore] Specifies a property name. A property can be a surname, city or other. * [For reference restore] Specifies a reference ID. A reference can be a manager, group membership or other. |
| GroupId | String | Reference type. |
