# Managing Infrastructure

In this article

You can use the cmdlet in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Find-VBRViEntity](find-vbrvientity.md) | Looks for VMware entities created on a selected host. |
| [Find-VBRHvEntity](find-vbrhventity.md) | Returns Hyper-V objects added to Veeam Backup & Replication. |
| [Rescan-VBREntity](rescan-vbrentity.md) | Rescans backup infrastructure components. |
