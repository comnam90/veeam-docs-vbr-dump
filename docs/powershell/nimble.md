# HPE Nimble Storage Systems


Add HPE Nimble storage systems to your Veeam Backup & Replication console and administrate them.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Add-NimbleHost](add-nimblehost.md) | Adds HPE Nimble storage systems to Veeam Backup & Replication. |
| [Get-NimbleHost](get-nimblehost.md) | Returns HPE Nimble storage systems. |
| [Set-NimbleHost](set-nimblehost.md) | Modifies HPE Nimble storage systems. |
| [Sync-NimbleHost](sync-nimblehost.md) | Rescans HPE Nimble storage systems. |
| [Remove-NimbleHost](remove-nimblehost.md) | Removes HPE Nimble storage systems. |
| [Get-NimbleVolume](get-nimblevolume.md) | Returns HPE Nimble storage volumes from the Veeam Backup & Replication infrastructure. |
| [Get-NimbleInfrastructureVolume](get-nimbleinfrastructurevolume.md) | Returns HPE Nimble storage volumes from the storage infrastructure. |
| [Sync-NimbleVolume](sync-nimblevolume.md) | Starts a rescan session of HPE Nimble storage volumes from the Veeam Backup & Replication infrastructure. |
| [Add-NimbleSnapshot](add-nimblesnapshot.md) | Creates HPE Nimble storage snapshots. |
| [Get-NimbleSnapshot](get-nimblesnapshot.md) | Looks for HPE Nimble storage snapshots. |
| [Remove-NimbleSnapshot](remove-nimblesnapshot.md) | Returns HPE Nimble storage snapshots. |
| [New-VBRStorageProtocolPolicy](new-vbrstorageprotocolpolicy.md) | Creates protocol policies for storage systems. |
