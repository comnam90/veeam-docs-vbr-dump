# VBRNASProxyServer

In this article

Contains settings of the file backup proxy server.

Properties

| Property | Type | Description |
| --- | --- | --- |
| Id | GUID | ID of the file backup proxy server. |
| Description | String | Description of the file backup proxy server. |
| Server | CHost | Proxy server. |
| ConcurrentTaskNumber | Int32 | Number of concurrent tasks that can be assigned to the file backup proxy simultaneously. |
