# NDMP Volumes Restore


You can use the cmdlets from this section to add NDMP servers to the backup infrastructure and perform restore operations with NDMP servers:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRNDMPServer](add-vbrndmpserver.md) | Adds an NDMP server to Veeam Backup & Replication. |
| [Set-VBRNDMPServer](set-vbrndmpserver.md) | Modifies NDMP server settings. |
| [Get-VBRNDMPServer](get-vbrndmpserver.md) | Returns NDMP servers. |
| [Get-VBRNDMPVolume](get-vbrndmpvolume.md) | Returns NDMP server volumes. |
| [Remove-VBRNDMPServer](remove-vbrndmpserver.md) | Removes the NDMP server from the Veeam Backup & Replication infrastructure. |
| [Get-VBRNDMPVolumeRestorePoint](get-vbrndmpvolumerestorepoint.md) | Returns restore points of NDMP server volumes. |
| [Start-VBRNDMPVolumeRestore](start-vbrndmpvolumerestore.md) | Restores data from tape to an NDMP server. |
