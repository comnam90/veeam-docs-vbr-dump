# Cloud Director Replica Failback


You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Start-VBRvCDReplicaFailback](start-vbrvcdreplicafailback.md) | Starts to perform failback from Cloud Director replica to the production vApp. |
| [Stop-VBRvCDReplicaFailback](stop-vbrvcdreplicafailback.md) | Stops a Cloud Director replica failback. |
