# Backup Copy Jobs for VMs and Physical Machines

In this article

You can use cmdlets in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRBackupCopyJob](add-vbrbackupcopyjob.md) | Creates a backup copy job. |
| [Copy-VBRBackupCopyJob](copy-vbrbackupcopyjob.md) | Copies a backup copy job. |
| [Disable-VBRBackupCopyJob](disable-vbrbackupcopyjob.md) | Disables a backup copy job. |
| [Get-VBRBackupCopyJob](get-vbrbackupcopyjob.md) | Returns a backup copy job. |
| [Remove-VBRBackupCopyJob](remove-vbrbackupcopyjob.md) | Removes a backup copy job. |
| [Set-VBRBackupCopyJob](set-vbrbackupcopyjob.md) | Modifies a backup copy job. |
| [Stop-VBRBackupCopyJob](stop-vbrbackupcopyjob.md) | Stops backup copy jobs. |
| [Enable-VBRBackupCopyJob](enable-vbrbackupcopyjob.md) | Enables a backup copy job. |
| [Sync-VBRBackupCopyJob](sync-vbrbackupcopyjob.md) | Synchronizes backup copy job data. |
| [Get-VBRLegacyBackupCopySourceJob](get-vbrlegacybackupcopysourcejob.md) | Returns legacy periodic backup copy job source objects. |
| [Convert-VBRLegacyCopyBackup](convert-vbrlegacycopybackup.md) | Converts legacy backup file of legacy periodic backup copy job to the per-machine backup with separate metadata files using mapping. |
