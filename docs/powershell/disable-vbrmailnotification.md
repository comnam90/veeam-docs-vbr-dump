# Disable-VBRMailNotification


Short Description

Disables global email notification settings.

Applies to

Product Edition: Standard, Enterprise, Enterprise Plus, Veeam Universal License

Syntax

|  |
| --- |
| Disable-VBRMailNotification Â [<CommonParameters>] |

Detailed Description

This cmdlet disables global email notification settings.

Run the [Enable-VBRMailNotification](enable-vbrmailnotification.md) cmdlet to enable global email notification settings.

<CommonParameters>

This cmdlet supports Microsoft PowerShell common parameters. For more information on common parameters, see  [Microsoft Docs](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/about/about_commonparameters?view=powershell-7).

Output Object

None.

Examples

Disabling Global Email Notification Settings

This command disables global email notification settings.
