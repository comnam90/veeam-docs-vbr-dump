In this article

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |

| [New-VBRvCDIndividualObjectStoragePolicy](new-vbrvcdindividualobjectstoragepolicy.md) | Defines Cloud Director storage policies for vApps or VMs. |
| [Set-VBRvCDIndividualObjectStoragePolicy](set-vbrvcdindividualobjectstoragepolicy.md) | Modifying the Cloud Director storage policies for vApps or VMs. |
