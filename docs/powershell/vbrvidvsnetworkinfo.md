# VBRViDVSNetworkInfo


Contains virtual network connected to DVS on VMware host.

Extends [VBRViNetworkInfo](vbrvinetworkinfo.md).

Properties

| Property | Type | Description |
| --- | --- | --- |
| Uuid | string | The network UUID. |
| SwitchName | string | The name of the switch that is connected to the network. |
