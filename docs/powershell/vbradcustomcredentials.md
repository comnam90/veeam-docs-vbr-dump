# VBRADCustomCredentials


Contains custom credentials for authenticating with Active Directory objects.

Properties

| Property | Type | Description |
| --- | --- | --- |
| Entity | [VBRADEntity](vbradentity.md) | The array of Active Directory objects. |
| Credentials | CCredentials | Credentials for authenticating with Active Directory objects. |
