# VBRCopyBackupSession

In this article

This object contains settings of the copy session.

Properties

| Property | Type | Description |
| --- | --- | --- |
| TargetRepositoryID | GUID | ID of the target repository. |
| TargetPath | string | Path to the target folder. |
| Backup | GUID[] | ID of the backup that will be copied. |
| Name | string | Name of the backup session. |

Related Commands
