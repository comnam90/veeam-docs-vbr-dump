# VBRBackupFile


Contains missing backup files.

Properties

| Property | Type | Description |
| --- | --- | --- |
| Path | string | Path to the location where missing backup is located. |
| ID | GUID | ID of the missing backup file. |
| IsAvailable | bool | Defines if the backup file is available (TRUE) or not available (FALSE). |

Related Commands
