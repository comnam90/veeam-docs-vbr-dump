# Working with Jobs


You can use the cmdlets in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRViBackupJob](add-vbrvibackupjob.md) | Creates VMware backup jobs. |
| [Add-VBRHvBackupJob](add-vbrhvbackupjob.md) | Creates Hyper-V backup jobs. |
| [Get-VBRJob](get-vbrjob.md) | Returns existing jobs. |
| [Start-VBRJob](start-vbrjob.md) | Starts backup, replication or copy job. |
| [Stop-VBRJob](stop-vbrjob.md) | Stops a running backup, replication or copy job. |
| [Copy-VBRJob](copy-vbrjob.md) | Clones existing jobs. |
| [Disable-VBRJob](disable-vbrjob.md) | Puts a selected job on hold. |
| [Enable-VBRJob](enable-vbrjob.md) | Enables a disabled job. |
| [Remove-VBRJob](remove-vbrjob.md) | Removes a selected backup, replication or backup copy job. |
