# SureBackup Application Groups


|  |
| --- |
| Note |
| SureBackup requires Enterprise or Enterprise Plus, Veeam Universal License product edition. |

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRApplicationGroup](add-vbrapplicationgroup.md) | Creates application groups for SureBackup jobs. |
| [Get-VBRApplicationGroup](get-vbrapplicationgroup.md) | Returns VMware and Hyper-V application groups. |
| [Set-VBRApplicationGroup](set-vbrapplicationgroup.md) | Modifies settings of application groups. |
| [Remove-VBRApplicationGroup](remove-vbrapplicationgroup.md) | Removes application groups from the Veeam Backup & Replication infrastructure. |
| [New-VBRSureBackupVM](new-vbrsurebackupvm.md) | Defines VMs to add to application groups. |
| [Set-VBRSureBackupVM](set-vbrsurebackupvm.md) | Modifies settings of VMs to add to application groups. |
| [New-VBRSureBackupStartupOptions](new-vbrsurebackupstartupoptions.md) | Defines startup settings for VMs. |
| [Set-VBRSureBackupStartupOptions](set-vbrsurebackupstartupoptions.md) | Modifies startup settings for VMs that are added to application groups and to jobs that are linked to SureBackup jobs. |
| [New-VBRSureBackupTestScript](new-vbrsurebackuptestscript.md) | Defines recovery verification scripts for VMs. |
| [Set-VBRSureBackupTestScript](set-vbrsurebackuptestscript.md) | Modifies settings of custom recovery verification scripts for VMs. |
| [New-VBRApplicationGroupStartupOptions](new-vbrapplicationgroupstartupoptions_surebackup.md) | Creates VM startup settings for staged restore. |
| [Set-VBRApplicationGroupStartupOptions](set-vbrapplicationgroupstartupoptions_surebackup.md) | Modifies VM startup settings for staged restore. |
