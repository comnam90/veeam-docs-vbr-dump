# Veeam Cloud Connect Cloud Gateway Pools

In this article

You can use the cmdlets from this section to perform the following operations with cloud Gateway pools:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRCloudGatewayPool](add-vbrcloudgatewaypool.md) | Creates cloud gateway pools. |
| [Get-VBRCloudGatewayPool](get-vbrcloudgatewaypool.md) | Returns cloud gateway pools. |
| [Set-VBRCloudGatewayPool](set-vbrcloudgatewaypool.md) | Modifies cloud gateway pools. |
| [Remove-VBRCloudGatewayPool](remove-vbrcloudgatewaypool.md) | Removes cloud gateway pools. |
| [Disable-VBRCloudGatewayPool](disable-vbrcloudgatewaypool.md) | Disables cloud gateway pools. |
| [Enable-VBRCloudGatewayPool](enable-vbrcloudgatewaypool.md) | Enables disabled cloud gateway pools. |
