# Cloud Credentials

In this article

This section contains instructions on setting up and managing a list of credentials records that you plan to use to connect to cloud services by means of Veeam PowerShell.

You can set up and manage credentials for the following cloud services:

* [Microsoft Azure Accounts](azure.md)
* [Amazon S3 Accounts](amazon.md)
* [Google Cloud Accounts](google_cloud.md)
