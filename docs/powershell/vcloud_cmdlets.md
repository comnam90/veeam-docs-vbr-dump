# VMware Cloud Director

In this article

Veeam Backup & Replication allows you to back up VMs managed by VMware Cloud Director.

Veeam PowerShell provides all means to add the Cloud Director server to your console and manage back up and recovery of the VMs.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRvCloud](add-vbrvcloud.md) | Adds a Cloud Director server |
| [Add-VBRvCloudJob](add-vbrvcloudjob.md) | Creates a Cloud Director backup job |
| [Add-VBRvCloudVC](add-vbrvcloudvc.md) | Adds vCenter Server managed by Cloud Director |
| [Remove-VBRServer](remove-vbrserver.md) | Removes a Cloud Director server |
| [Find-VBRvCloudEntity](find-vbrvcloudentity.md) | Looks for any kinds of Cloud Director objects |
| [New-VBRvCloudVAppRestoreSettings](new-vbrvcloudvapprestoresettings.md) | Creates an object with a set of restore parameters for a vApp restore (run the [Start-VBRvCloudRestoreVApp](start-vbrvcloudrestorevapp.md) cmdlet to apply) |
| [Start-VBRvCloudInstantRecovery](start-vbrvcloudinstantrecovery.md) | Starts an instant recovery of Cloud Director VMs |
| [Start-VBRvCloudQuickMigration](start-vbrvcloudquickmigration.md) | Starts a quick migration of Cloud Director VMs |
| [Start-VBRvCloudRestoreVApp](start-vbrvcloudrestorevapp.md) | Restores Cloud Director vApp |
| [Start-VBRvCloudRestoreVm](start-vbrvcloudrestorevm.md) | Restores a Cloud Director VM |
| [Set-VBRvCloudOrganizationJobMapping](set-vbrvcloudorganizationjobmapping.md) | Adds Cloud Director backup jobs and backups to the Cloud Director Organization or removes them from it. |
