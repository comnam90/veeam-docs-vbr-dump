# Managing Unstructured Data Source


You can use the cmdlets in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRUnstructuredServer](get-vbrunstructuredserver.md) | Returns unstructured data sources added to the inventory. |
| [Remove-VBRUnstructuredServer](remove-vbrunstructuredserver.md) | Removes unstructured data sources from the inventory. |
| [Update-VBRUnstructuredBackupPath](update-vbrunstructuredbackuppath.md) | Updates the data source for file backups and object storage backups. |
