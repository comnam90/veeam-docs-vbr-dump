# Minor Non-Breaking Changes


The following minor non-breaking changes were introduced in Veeam PowerShell 12.3.1

Backup Copy

Backup Copy Jobs for HPE StoreOnce Repositories

In this version, the EnableHealthCheck set to the False value by default for these cmdlets:

* [Add-CatalystCopyJob](add-vbrcatalystcopyjob.md)
* [Set-CatalystCopyJob](set-vbrcatalystcopyjob.md)
