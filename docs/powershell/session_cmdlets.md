# Sessions

In this article

Get sessions of a selected job and look which tasks it contains.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRBackupSession](get-vbrbackupsession.md) | Returns VM backup sessions |
| [Get-VBRSession](get-vbrsession.md) | Returns jobs sessions |
| [Get-VBRTaskSession](get-vbrtasksession.md) | Returns tasks performed during a specific session |
| [Get-VBRRestoreSession](get-vbrrestoresession.md) | Returns restore sessions |
| [Stop-VBRSession](stop-vbrsession.md) | Stops tape jobs sessions. |
| [Get-VBRCDPSession](get-vbrcdpsession.md) | Returns CDP policy sessions. |
| [Get-VBRCDPTaskSession](get-vbrcdptasksession.md) | Returns VMs processed during CDP policy sessions. |
| [Get-VBRCopyBackupSession](get-vbrcopybackupsession.md) | Returns copy sessions. |
| [Get-VBRMoveBackupSession](get-vbrmovebackupsession.md) | Returns move sessions. |
| [Retry-VBRCopyBackupSession](retry-vbrcopybackupsession.md) | Retries the copy operation for failed workloads. |
| [Retry-VBRMoveBackupSession](retry-vbrmovebackupsession.md) | Retries the move operation for failed workloads. |
| [Stop-VBRCopyBackupSession](stop-vbrcopybackupsession.md) | Specifies how to process backups for which a copy sessions failed. |
| [Stop-VBRMoveBackupSession](stop-vbrmovebackupsession.md) | Specifies how to process backups for which a move sessions failed. |

For SureBackup sessions, see [SureBackup Jobs](surebackup_job.md).
