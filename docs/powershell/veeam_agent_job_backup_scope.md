# Veeam Agent Job Backup Scope


You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [New-VBRSelectedFilesBackupOptions](new-vbrselectedfilesbackupoptions.md) | Creates backup scope settings for Veeam Agent backup jobs. |
| [New-VBRLinuxSelectedVolume](new-vbrlinuxselectedvolume.md) | Creates the scope of volumes for Veeam Agent backup jobs that back up Linux computers. |
| [Set-VBRSelectedFilesBackupOptions](set-vbrselectedfilesbackupoptions.md) | Modifies the backup scope settings for Veeam Agent backup jobs. |
| [Set-VBRLinuxSelectedVolume](set-vbrlinuxselectedvolume.md) | Modifies the scope of volumes for Veeam Agent backup jobs that back up Linux computers. |
| [New-VBRSelectedPersonalFolders](new-vbrselectedpersonalfolders.md) | Defines the scope of personal data for Agent Backup jobs that back up Microsoft Windows and Mac computers. |
| [Set-VBRSelectedPersonalFolders](set-vbrselectedpersonalfolders.md) | Modifies the scope of personal data for Agent Backup jobs that back up Microsoft Windows and Mac computers. |
