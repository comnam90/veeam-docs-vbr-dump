# VBRLinuxTrustedHostPolicy


Contains a trust policy for Linux hosts connecting to the Veeam backup server.

Properties

| Property | Type | Description |
| --- | --- | --- |
| Type | VBRLinuxTrustedHostPolicyType | The trust policy type:   * All * KnownHosts |
