# Job Options


You can use the cmdlets in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRJobOptions](get-vbrjoboptions.md) | Returns job settings. |
| [New-VBRJobOptions](new-vbrjoboptions.md) | Creates new job options. |
| [New-VBRJobScriptOptions](new-vbrjobscriptoptions.md) | Creates new job script options. |
| [New-VBRNotificationOptions](new-vbrnotificationoptions.md) | Creates new job notification options. |
| [New-VBRRPONotificationOption](new-vbrrponotificationoption.md) | Creates RPO notification options. |
| [Set-VBRRPONotificationOption](set-vbrrponotificationoption.md) | Modifies RPO notification settings. |
| [Reset-VBRJobOptions](reset-vbrjoboptions.md) | Applies default settings to jobs. |
| [Set-VBRJobOptions](set-vbrjoboptions.md) | Modifies job settings. |
| [Set-VBRJobScriptOptions](set-vbrjobscriptoptions.md) | Modifies job script options. |
| [Set-VBRNotificationOptions](set-vbrnotificationoptions.md) | Modifies job notification settings. |
