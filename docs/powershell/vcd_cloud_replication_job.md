# Cloud Director Cloud Replication Job

In this article

You can use the cmdlets from this section to perform the following operations with Cloud Director cloud replication jobs:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRvCDCloudReplicaJob](add-vbrvcdcloudreplicajob.md) | Creates Cloud Director cloud replication jobs |
| [Set-VBRvCDCloudReplicaJob](set-vbrvcdcloudreplicajob.md) | Modifies Cloud Director cloud replication jobs |
| [Get-VBRvCDCloudOrganizationvDC](get-vbrvcdcloudorganizationvdc.md) | Returns Cloud Director Organizations |
| [Get-VBRvCDCloudStoragePolicy](get-vbrvcdcloudstoragepolicy.md) | Returns Cloud Director storage policies |
| [Get-VBRvCDCloudNetworkInfo](get-vbrvcdcloudnetworkinfo.md) | Returns the target Cloud Director network for replica mapping |
| [Get-VBRvCDCloudvApp](get-vbrvcdcloudvapp.md) | Returns Cloud Director vApps |
| [Get-VBRvCDCloudVM](get-vbrvcdcloudvm.md) | Returns cloud VMs from organization VCDs. |
