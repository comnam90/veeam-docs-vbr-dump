# VBRInternetProxyUpdateOptions

In this article

Contains proxy update settings.

Properties

| Property | Type | Description |
| --- | --- | --- |
| Credentials | CCredentials | The credentials used to log in to the proxy server. |
| Enabled | Boolean | Indicates if a proxy server is used. |
| Port | Int32 | The port used to connect to the proxy server. |
| ProxyServerName | String | The DNS or IP address of the proxy server. |
| RequireAuthentication | Boolean | Indicates if the proxy server requires authentication. |
