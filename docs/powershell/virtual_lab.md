# SureBackup Virtual Labs


|  |
| --- |
| Note |
| SureBackup requires Enterprise or Enterprise Plus, Veeam Universal License product edition. |

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRViAdvancedVirtualLab](add-vbrviadvancedvirtuallab.md) | Creates a VMware advanced virtual lab. |
| [Add-VBRViSimpleVirtualLab](add-vbrvisimplevirtuallab.md) | Creates a VMware basic virtual lab. |
| [Get-VBRViVirtualLabConfiguration](get-vbrvivirtuallabconfiguration.md) | Returns virtual labs and their settings. |
| [Set-VBRViVirtualLab](set-vbrvivirtuallab.md) | Modifies settings of VMware virtual labs. |
| [Connect-VBRViVirtualLab](connect-vbrvivirtuallab.md) | Adds VMs to Veeam Backup & Replication. |
| [Add-VBRHvAdvancedVirtualLab](add-vbrhvadvancedvirtuallab.md) | Creates a Hyper-V advanced virtual lab. |
| [Add-VBRHvSimpleVirtualLab](add-vbrhvsimplevirtuallab.md) | Creates a Hyper-V basic virtual lab. |
| [Get-VBRHvVirtualLabConfiguration](get-vbrhvvirtuallabconfiguration.md) | Returns virtual labs and their settings. |
| [Set-VBRHvVirtualLab](set-vbrhvvirtuallab.md) | Modifies settings of Hyper-V virtual labs. |
| [Connect-VBRHvVirtualLab](connect-vbrhvvirtuallab.md) | Adds VMs to Veeam Backup & Replication. |
| [Remove-VSBHvVirtualLab](remove-vsbhvvirtuallab.md) | Removes a Hyper-V virtual lab. |
| [Get-VBRVirtualLab](get-vbrvirtuallab.md) | Returns virtual labs with main settings. |
| [Remove-VBRVirtualLab](remove-vbrvirtuallab.md) | Removes virtual labs from Veeam Backup & Replication. |
| [New-VBRViNetworkMappingRule](new-vbrvinetworkmappingrule.md) | Defines network mapping rules of isolated networks. |
| [Set-VBRViNetworkMappingRule](set-vbrvinetworkmappingrule.md) | Modifies network mapping rules of isolated networks. |
| [New-VBRHvProductionNetworkOptions](new-vbrhvproductionnetworkoptions.md) | Defines production networks options for Hyper-V virtual lab. |
| [Set-VBRHvProductionNetworkOptions](set-vbrhvproductionnetworkoptions.md) | Modifies production network options for Hyper-V virtual lab. |
| [New-VBRViVirtualLabNetworkOptions](new-vbrvivirtuallabnetworkoptions.md) | Defines network settings of isolated networks. |
| [Set-VBRViVirtualLabNetworkOptions](set-vbrvivirtuallabnetworkoptions.md) | Modifies network settings of isolated networks. |
| [New-VBRHvIsolatedNetworkOptions](new-vbrhvisolatednetworkoptions.md) | Defines network settings of isolated networks. |
| [Set-VBRHvIsolatedNetworkOptions](set-vbrhvisolatednetworkoptions.md) | Modifies network settings of isolated networks for Hyper-V virtual lab. |
| [New-VBRViVirtualLabIPMappingRule](new-vbrvivirtuallabipmappingrule.md) | Defines static IP address mapping rules. |
| [Set-VBRViVirtualLabIPMappingRule](set-vbrvivirtuallabipmappingrule.md) | Modifies static IP address mapping rules. |
| [New-VBRHvVirtualLabStaticIPMappingRule](new-vbrhvvirtuallabstaticipmappingrule.md) | Defines static IP address mapping rules. |
| [Set-VBRHvVirtualLabStaticIPMappingRule](set-vbrhvvirtuallabstaticipmappingrule.md) | Modifies static IP address mapping rules for Hyper-V virtual lab. |
| [New-VBRViVirtualLabProxyAppliance](new-vbrvivirtuallabproxyappliance.md) | Defines settings of proxy appliances. |
| [Set-VBRViVirtualLabProxyAppliance](set-vbrvivirtuallabproxyappliance.md) | Modifies settings of proxy appliances. |
