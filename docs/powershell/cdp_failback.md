# CDP Failback

In this article

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Start-VBRCDPReplicaFailback](start-vbrcdpreplicafailback.md) | Starts to perform failback from CDP replica to the production VM. |
| [Stop-VBRCDPReplicaFailback](stop-vbrcdpreplicafailback.md) | Undoes a CDP replica failback. |
