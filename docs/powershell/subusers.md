# Veeam Cloud Connect Subusers


You can use the cmdlets from this section to perform the following operations with cloud subuser accounts:

| Cmdlet | Operation |
| --- | --- |
| [New-VBRCloudSubUserResource](new-vbrcloudsubuserresource.md) | Creates quotas of cloud subuser resources |
| [Set-VBRCloudSubUserResource](set-vbrcloudsubuserresource.md) | Modifies settings of cloud subuser resources |
| [Add-VBRCloudSubUser](add-vbrcloudsubuser.md) | Creates new cloud subuser accounts |
| [Add-VBRvCDCloudSubUser](add-vbrvcdcloudsubuser.md) | Creates Cloud Director subuser accounts |
| [Get-VBRCloudSubUser](get-vbrcloudsubuser.md) | Looks for cloud subuser accounts |
| [Get-VBRvCDCloudOrganizationUser](get-vbrvcdcloudorganizationuser.md) | Returns Cloud Director Â user accounts. |
| [Set-VBRCloudSubUser](set-vbrcloudsubuser.md) | Modifies cloud subuser accounts settings |
| [Remove-VBRCloudSubUser](remove-vbrcloudsubuser.md) | Removes cloud subuser accounts |
| [Disable-VBRCloudSubUser](disable-vbrcloudsubuser.md) | Disables cloud subuser accounts |
| [Enable-VBRCloudSubUser](enable-vbrcloudsubuser.md) | Enables disabled cloud subuser accounts |
