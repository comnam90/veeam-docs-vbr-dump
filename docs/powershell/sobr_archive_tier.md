# Archive Tier

In this article

You can use the cmdlets in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRArchiveExtent](get-vbrarchiveextent.md) | Returns the archive extent of a scale-out backup repository. |
| [Enable-VBRArchiveExtentMaintenanceMode](enable-vbrarchiveextentmaintenancemode.md) | Enables the Maintenance mode for the archive extent of a scale-out backup repository. |
| [Disable-VBRArchiveExtentMaintenanceMode](disable-vbrarchiveextentmaintenancemode.md) | Disables the Maintenance mode for the archive extent of a scale-out backup repository. |
| [Enable-VBRArchiveExtentSealedMode](enable-vbrarchiveextentsealedmode.md) | Enables the Sealed mode for the archive extent Â of a scale-out backup repository. |
| [Disable-VBRArchiveExtentSealedMode](disable-vbrarchiveextentsealedmode.md) | Disables the Sealed mode for the archive extent of a scale-out backup repository. |
| [Publish-VBRArchiveRestorePoint](publish-vbrarchiverestorepoint.md) | Retrieves data from archive storage. |
| [Add-VBRArchiveBackupAvailabilityPeriod](add-vbrarchivebackupavailabilityperiod.md) | Extends the availability period of the backup files from the archive storage. |
| [Get-VBRCloudArchiveRestorePoint](get-vbrcloudarchiverestorepoint.md) | Returns restore points of tenant backups located in the archive tier. |
| [Publish-VBRArchiveRestorePoint](publish-vbrarchiverestorepoint.md) | Retrieves tenant data from archive storage. |
| [Add-VBRArchiveBackupAvailabilityPeriod](add-vbrarchivebackupavailabilityperiod.md) | Extends the availability of the backup files from the archive storage. |
| [Start-VBRArchiveTierSync](start-vbrarchivetiersync.md) | Starts the archiving job. |
