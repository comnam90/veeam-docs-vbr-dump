# Tenant to Tape

In this article

You can use the cmdlets from this section to manage tenant jobs and restore tenant data from tape:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRCloudTapeBackupTenant](get-vbrcloudtapebackuptenant.md) | Returns backups of tenants. |
| [Get-VBRCloudTapeBackupTenantRepository](get-vbrcloudtapebackuptenantrepository.md) | Returns tenant backup repositories. |
| [Get-VBRCloudTapeBackupTenantJob](get-vbrcloudtapebackuptenantjob.md) | Returns the tenant jobs. |
| [Start-VBRCloudTapeRestore](start-vbrcloudtaperestore.md) | Starts a restore session of the tenant data from tape. |
| [Get-VBRCloudTapeRestoreSession](get-vbrcloudtaperestoresession.md) | Returns active restore sessions of the tenant data from tape. |
| [Stop-VBRCloudTapeRestore](stop-vbrcloudtaperestore.md) | This cmdlet stops the restore session of the tenant data from tape. |
