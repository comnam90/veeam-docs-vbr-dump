# Instant Recovery

In this article

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Start-VBRInstantRecovery](start-vbrinstantrecovery.md) | Starts VMware Instant Recovery. |
| [New-VBRHvInstantRecoveryNetworkMappingRule](new-vbrhvinstantrecoverynetworkmappingrule.md) | Creates a new network mapping rule. |
| [New-VBRHvInstantRecoveryHelperAppliance](new-vbrhvinstantrecoveryhelperappliance.md) | Defines settings for a helper appliance. |
| [Start-VBRHvInstantRecovery](start-vbrhvinstantrecovery.md) | Starts Hyper-V Instant Recovery. |
| [Restart-VBRInstantRecovery](restart-vbrinstantrecovery.md) | Restarts a failed Hyper-V Instant Recovery session. |
| [Get-VBRInstantRecovery](get-vbrinstantrecovery.md) | Returns running Instant Recovery sessions. |
| [Set-VBRHvInstantRecoveryHelperAppliance](set-vbrhvinstantrecoveryhelperappliance.md) | Modifies settings for a helper appliance. |
| [Stop-VBRInstantRecovery](stop-vbrinstantrecovery.md) | Stops running Instant Recovery sessions. |
