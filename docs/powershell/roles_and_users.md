# Roles and Users


You can use the cmdlets in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRUserRoleAssignment](add-vbruserroleassignment.md) | Adds a role to a user or a user group. |
| [Get-VBRUserRoleAssignment](get-vbruserroleassignment.md) | Returns a role that is assigned to a user or a user group. |
| [Set-VBRUserRoleAssignment](set-vbruserroleassignment.md) | Modifies a role that is assigned to a user or a user group. |
| [Remove-VBRUserRoleAssignment](remove-vbruserroleassignment.md) | Removes a role from a user or a user group. |
