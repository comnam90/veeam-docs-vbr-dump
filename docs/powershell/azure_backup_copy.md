# Azure Backup Copy Job

In this article

You can use cmdlets in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRAzureComputeBackup](get-vbrazurecomputebackup.md) | Returns Azure backups. |
| [Set-VBRAzureComputeBackupCopyJob](set-vbrazurecomputebackupcopyjob.md) | Modifies Azure IaaS backup copy jobs. |
| [New-VBRRecoveryPointObjective](new-vbrrecoverypointobjective_azure.md) | Creates an interval for backup copy jobs that process backups stored on external repositories. |
| [Set-VBRRecoveryPointObjective](set-vbrrecoverypointobjective_azure.md) | Modifies an interval for backup copy jobs that process backups stored on external repositories. |
| [Get-VBRRecoveryPointObjective](get-vbrrecoverypointobjective_azure.md) | Modifies an interval for backup copy jobs that process backups stored on external repositories. |
| [New-VBRSimpleRetentionPolicy](new-vbrsimpleretentionpolicy_azure.md) | Creates a retention policy for backup copy jobs that process backups stored on external repositories. |
| [Set-VBRSimpleRetentionPolicy](set-vbrsimpleretentionpolicy_azure.md) | Modifies a retention policy for backup copy jobs that process backups stored on external repositories. |
| [Get-VBRRetentionPolicy](get-vbrretentionpolicy_azure.md) | Returns a retention policy for backup copy jobs. |
| [New-VBRGFSRetentionPolicy](new-vbrgfsretentionpolicy_azure.md) | Creates a GFS retention policy for backup copy jobs that process backups stored on external repositories. |
| [Set-VBRGFSRetentionPolicy](set-vbrgfsretentionpolicy_azure.md) | Modifies a GFS retention policy for backup copy jobs that process backups stored on external repositories. |
