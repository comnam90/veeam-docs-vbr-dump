# Veeam Cloud Connect Cloud Gateways

In this article

You can use the cmdlets from this section to perform the following operations with cloud gateways:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRCloudGateway](add-vbrcloudgateway.md) | Creates a cloud gateway |
| [Get-VBRCloudGateway](get-vbrcloudgateway.md) | Returns existing cloud gateways |
| [Enable-VBRCloudGateway](enable-vbrcloudgateway.md) | Enables a cloud gateway |
| [Disable-VBRCloudGateway](disable-vbrcloudgateway.md) | Disables a cloud gateway |
| [Set-VBRCloudGateway](set-vbrcloudgateway.md) | Modifies a cloud gateway |
| [Remove-VBRCloudGateway](remove-vbrcloudgateway.md) | Removes a cloud gateway |

Page updated 9/4/2023
