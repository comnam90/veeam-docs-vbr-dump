# Capacity Tier


You can use the cmdlets in this topic to perform the following operations:

Capacity Tier

| Cmdlet | Operation |
| [Get-VBRCapacityExtent](get-vbrcapacityextent.md) | Returns the capacity extent of a scale-out backup repository. |
| [Enable-VBRCapacityExtentMaintenanceMode](enable-vbrcapacityextentmaintenancemode.md) | Enables the Maintenance mode for capacity extents of a scale-out backup repository. |
| [Disable-VBRCapacityExtentMaintenanceMode](disable-vbrcapacityextentmaintenancemode.md) | Disables the Maintenance mode for capacity extents of a scale-out backup repository. |
| [Enable-VBRCapacityExtentSealedMode](enable-vbrcapacityextentsealedmode.md) | Enables the Sealed mode for capacity extents  of a scale-out backup repository. |
| [Disable-VBRCapacityExtentSealedMode](disable-vbrcapacityextentsealedmode.md) | Disables the Sealed mode for capacity extents of a scale-out backup repository. |
| [Start-VBRCapacityTierSync](start-vbrcapacitytiersync.md) | Starts to sync data with the capacity extent. |
| [Start-VBRDownloadBackup](start-vbrdownloadbackup.md) | Downloads backup files from the capacity tier to the performance tier. |
