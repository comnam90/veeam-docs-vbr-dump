# Nutanix Files Storage Systems

In this article

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRNutanixHost](add-vbrnutanixhost.md) | Adds Nutanix Files storage systems to the backup infrastructure. |
| [Get-VBRNutanixHost](get-vbrnutanixhost.md) | Returns Nutanix Files storage systems. |
| [Set-VBRNutanixHost](set-vbrnutanixhost.md) | Modifies Nutanix Files storage settings. |
| [Sync-VBRNutanixHost](sync-vbrnutanixhost.md) | Starts a rescan session of Nutanix Files storage system. |
| [Sync-VBRNutanixVolume](sync-vbrnutanixvolume.md) | Starts a rescan session of Nutanix Files storage volumes from the backup infrastructure. |
| [Get-VBRNutanixInfrastructureVolume](get-vbrnutanixinfrastructurevolume.md) | Returns Nutanix Files storage volumes added to the Veeam Backup & Replication infrastructure. |
| [Get-VBRNutanixVolume](get-vbrnutanixvolume.md) | Returns all volumes of the specified Nutanix Files storage systems. |
| [Remove-VBRNutanixHost](remove-vbrnutanixhost.md) | Removes Nutanix Files storage systems from the backup infrastructure. |
