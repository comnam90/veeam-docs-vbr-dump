# Managing Microsoft Azure Connection Service

In this article

You can use the cmdlets in this topic to perform the following operations.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Connect-VBRAzureBlobService](connect-vbrazureblobservice.md) | Starts a new session with Microsoft Azure Blob object storage. |
| [Disconnect-VBRAzureBlobService](disconnect-vbrazureblobservice.md) | Ends the current session with Microsoft Azure Blob storage. |
| [Connect-VBRAzureDataBoxService](connect-vbrazuredataboxservice.md) | Connects to Azure Data Box storage. |
