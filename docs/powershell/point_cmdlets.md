# Managing Restore Points

In this article

You can use the cmdlet in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRFilesInRestorePoint](get-vbrfilesinrestorepoint.md) | Returns backup files in a selected restore point. |
| [Get-VBRRestorePoint](get-vbrrestorepoint.md) | Returns restore points. |
| [Remove-VBRRestorePoint](remove-vbrrestorepoint.md) | Removes individual VMs from backup or replicas. |
| [Get-VBRApplicationRestorePoint](get-vbrapplicationrestorepoint.md) | Returns restore points created with the VSS-aware image processing enabled. |
| [Export-VBRRestorePoint](export-vbrrestorepoint.md) | Exports backup files of a specific restore point. |
| [Get-VBRCloudTenantRestorePoint](get-vbrcloudtenantrestorepoint.md) | Returns restore points of non-encrypted backups for AD tenants and tenants managed by the Service Provider. |
| [Get-VBRSOBRObjectStorageRestorePoint](get-vbrsobrobjectstoragerestorepoint.md) | Returns restore points stored on archive extents and capacity extents of a scale-out backup repository. |
