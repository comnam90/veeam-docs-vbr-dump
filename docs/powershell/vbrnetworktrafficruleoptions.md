# VBRNetworkTrafficRuleOptions

In this article

Contains network traffic rule details.

Properties

| Property | Type | Description |
| --- | --- | --- |
| MultipleUploadStreamsEnabled | Boolean | Defines that multithreaded data transfer is enabled or disabled. |
| StreamsPerJobCount | Int32 | Number of streams for each job. |
| IPPreferencePolicy | IPPreferencePolicy | Pv4/IPv6 stack mode settings. |
