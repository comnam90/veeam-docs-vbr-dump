# Tape Drives

In this article

You can use the cmdlets from this section to perform the following operations with tape drives:

| Cmdlet | Operation |
| --- | --- |
| [Disable-VBRTapeDrive](disable-vbrtapedrive.md) | Disables tape drive. |
| [Enable-VBRTapeDrive](enable-vbrtapedrive.md) | Enables tape drive. |
| [Get-VBRTapeDrive](get-vbrtapedrive.md) | Returns tape drives. |
