# Managing Microsoft Azure Storage Containers


You can use the cmdlets in this topic to perform the following operations.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRAzureBlobContainer](get-vbrazureblobcontainer.md) | Returns the array of Microsoft Azure Blob containers. |
| [New-VBRAzureBlobFolder](new-vbrazureblobfolder.md) | Creates a new folder in Microsoft Azure Blob object storage. |
| [Get-VBRAzureBlobFolder](get-vbrazureblobfolder.md) | Returns the array of Microsoft Azure Blob folders. |
