# Tape Decryption


You can use the cmdlets from this section to manage encryption settings for the tape:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBREncryptedTapeMediumHint](get-vbrencryptedtapemediumhint.md) | Returns password hints for the encrypted backups stored on tapes. |
| [Set-VBREncryptedTapeMediumPassword](set-vbrencryptedtapemediumpassword.md) | Decrypts encrypted backups stored on tapes. |
