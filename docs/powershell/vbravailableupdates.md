# VBRAvailableUpdate

In this article

Contains a list of available updates for a Linux server.

Properties

| Property | Type | Description |
| --- | --- | --- |
| ServerID | Guid | Unique identifier of the server. |
| ServerName | String | DNS or IP address of the server. |
| Description | String | List of the available updates for the server. |
