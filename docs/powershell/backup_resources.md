# Veeam Cloud Connect Backup Resources

In this article

Backup resources are quotas of cloud repositories and their settings. Backup resources are part of hardware plans.

| Cmdlet | Operation |
| --- | --- |
| [New-VBRCloudTenantResource](new-vbrcloudtenantresource.md) | Creates object with backup resources |
| [Set-VBRCloudTenantResource](set-vbrcloudtenantresource.md) | Modifies object with backup resources |
| [Sync-VBRCloudTenantResource](sync-vbrcloudtenantresource.md) | Refreshes information about current usage of backup resources by a cloud tenant. |
