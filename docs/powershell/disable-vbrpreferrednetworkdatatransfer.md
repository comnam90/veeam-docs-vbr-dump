# Disable-VBRPreferredNetworkDataTransfer


Short Description

Disables data transfer over preferred networks only.

Applies to

Platform: VMware, Hyper-V

Product Edition: Standard, Enterprise, Enterprise Plus, Veeam Universal License

Syntax

|  |
| --- |
| Disable-VBRPreferredNetworkDataTransfer Â [<CommonParameters>] |

Detailed Description

This cmdlet data transfer over preferred networks only.

Run the [Enable-VBRPreferredNetworkDataTransfer](enable-vbrpreferrednetworkdatatransfer.md) cmdlet to enable the data transfer over preferred networks only.

<CommonParameters>

This cmdlet supports Microsoft PowerShell common parameters. For more information on common parameters, see  [Microsoft Docs](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/about/about_commonparameters?view=powershell-7).

Output Object

None.

Examples

Disabling Data Transfer over Preferred Networks

This command disables data transfer over preferred networks only.
