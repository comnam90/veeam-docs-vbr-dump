# Plug-In Backup Copy Job


You can use cmdlets in this topic to perform operations with the following types of Veeam Plug-in jobs:

* Backup jobs created with standalone Veeam Plug-ins
* Backup copy jobs for Oracle backups
* Backup copy jobs for SAP HANA backups

* Backup copy jobs for SAP on Oracle backups
* Backup copy jobs for Microsoft SQL Server backups
* Backup copy jobs for IBM Db2 backups

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRPluginBackupCopyJob](add-vbrpluginbackupcopyjob.md) | Creates plug-in backup copy jobs. |
| [Get-VBRPluginJob](get-vbrpluginjob.md) | Returns backup jobs that were created with standalone Veeam Plug-ins and backup copy jobs of backups created with Veeam Plug-ins. |
| [Set-VBRPluginBackupCopyJob](set-vbrpluginbackupcopyjob.md) | Modifies plug-in backup copy jobs. |
| [Remove-VBRPluginJob](remove-vbrpluginjob.md) | Removes backup jobs that were created with standalone Veeam Plug-ins and backup copy jobs of backups created with Veeam Plug-ins from the Veeam Backup & Replication infrastructure. |
| [Enable-VBRPluginJob](enable-vbrpluginjob.md) | Enables backup jobs that were created with standalone Veeam Plug-ins and backup copy jobs of backups created with Veeam Plug-ins. |
| [Disable-VBRPluginJob](disable-vbrpluginjob.md) | Disables backup jobs that were created with standalone Veeam Plug-ins and backup copy jobs of backups created with Veeam Plug-ins. |
| [New-VBRPluginCopyJobStorageOptions](new-vbrplugincopyjobstorageoptions.md) | Defines storage optimization settings for plug-in backup copy jobs. |
| [Set-VBRPluginCopyJobStorageOptions](set-vbrplugincopyjobstorageoptions.md) | Modifies storage optimization settings for plug-in backup copy jobs. |
