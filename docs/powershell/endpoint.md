# Veeam Agent Standalone Mode


You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Get-VBREPPermission](get-vbreppermission.md) | Looks for current permissions for backup repositories for Veeam Agent or Veeam Plug-in operating in the standalone mode. |
| [Set-VBREPPermission](set-vbreppermission.md) | Applies user access permissions to backup repositories used by Veeam Agent or Veeam Plug-in operating in the standalone mode. |
| [Get-VBREPJob](get-vbrepjob.md) | Returns backup jobs run by Veeam Agent operating in the standalone mode. |
| [Disable-VBREPJob](disable-vbrepjob.md) | Disables backup jobs run by Veeam Agent operating in the standalone mode. |
| [Enable-VBREPJob](enable-vbrepjob.md) | Enables disabled backup jobs run by Veeam Agent operating in the standalone mode. |
| [Get-VBREPSession](get-vbrepsession.md) | Returns sessions of backup jobs run by Veeam Agent operating in the standalone mode. |
| [Start-VBRRestoreVirtualDisks](start-vbrrestorevirtualdisks.md) | Restores physical disks to virtual disk formats. |
| [Start-VBREpInstantRecovery](start-vbrepinstantrecovery.md) | Starts instant recovery of physical computers to virtual infrastructure. |
