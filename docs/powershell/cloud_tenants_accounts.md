# Veeam Cloud Connect Tenant Accounts


You can use the cmdlets in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRCloudTenant](add-vbrcloudtenant.md) | Creates standalone tenant accounts. |
| [Add-VBRvCDCloudTenant](add-vbrvcdcloudtenant.md) | Creates VMware Cloud Director tenant accounts. |
| [Add-VBRAdCloudTenant](add-vbradcloudtenant.md) | Creates Active Directory tenant accounts. |
| [Get-VBRCloudTenant](get-vbrcloudtenant.md) | Looks for existing cloud tenant accounts. |
| [Enable-VBRCloudTenant](enable-vbrcloudtenant.md) | Enables cloud tenant accounts. |
| [Disable-VBRCloudTenant](disable-vbrcloudtenant.md) | Disables cloud tenant accounts. |
| [Set-VBRvCDCloudTenant](set-vbrvcdcloudtenant.md) | Modifies VMware Cloud Director tenant accounts. |
| [Set-VBRAdCloudTenant](set-vbradcloudtenant.md) | Modifies Active Directory tenant accounts. |
| [Set-VBRCloudTenant](set-vbrcloudtenant.md) | Modifies cloud tenant accounts. |
| [Reset-VBRCloudTenant](reset-vbrcloudtenant.md) | Resets cloud tenant license accounts. |
| [Remove-VBRCloudTenant](remove-vbrcloudtenant.md) | Removes cloud tenant accounts. |
| [Start-VBRCloudTenantBackupEvacuation](start-vbrcloudtenantbackupevacuation.md) | Starts to migrate tenant data between performance extents. |
| [Switch-VBRCloudTenantsQuotaRepositoryToSOBR](switch-vbrcloudtenantsquotarepositorytosobr.md) | Switches tenant quota on the cloud repository to a scale-out backup repository. |
