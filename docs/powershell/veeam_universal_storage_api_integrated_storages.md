# Universal Storage API Integrated Systems


Add Universal Storage API integrated systems to your Veeam Backup & Replication console and administrate them.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRStoragePlugin](get-vbrstorageplugin.md) | Returns plug-ins for adding Universal Storage API integrated systems. |
| [Get-StoragePluginHost](get-storagepluginhost.md) | Returns Universal Storage API integrated systems. |
| [Add-StoragePluginHost](add-storagepluginhost.md) | Adds Universal Storage API integrated systems to Veeam Backup & Replication. |
| [Set-StoragePluginHost](set-storagepluginhost.md) | Modifies the settings for Universal Storage API integrated systems. |
| [Sync-StoragePluginHost](sync-storagepluginhost.md) | Starts a rescan session of Universal Storage API integrated systems. |
| [Remove-StoragePluginHost](remove-storagepluginhost.md) | Removes Universal Storage API integrated systems from Veeam Backup & Replication console. |
| [Get-StoragePluginVolume](get-storagepluginvolume.md) | Returns the volumes of Universal Storage API integrated systems from the Veeam Backup & Replication infrastructure. |
| [Get-StoragePluginInfrastructureVolume](get-storageplugininfrastructurevolume.md) | Returns the volumes of Universal Storage API integrated systems from the storage infrastructure. |
| [Sync-StoragePluginVolume](sync-storagepluginvolume.md) | Starts a rescan session for the volumes of Universal Storage API integrated systems. |
| [Get-StoragePluginArchivedSnapshot](get-storagepluginarchivedsnapshot.md) | Returns archived snapshots of Universal Storage API integrated systems. |
| [Publish-StoragePluginArchivedSnapshot](publish-storagepluginarchivedsnapshot.md) | Retrieves and restores an archived snapshot. |
| [Remove-StoragePluginArchivedSnapshot](remove-storagepluginarchivedsnapshot.md) | Removes archived snapshots of Universal Storage API integrated systems. |
| [Add-StoragePluginSnapshot](add-storagepluginsnapshot.md) | Creates snapshots of Universal Storage API integrated systems. |
| [Get-StoragePluginSnapshot](get-storagepluginsnapshot.md) | Returns snapshots of Universal Storage API integrated systems. |
| [Remove-StoragePluginSnapshot](remove-storagepluginsnapshot.md) | Removes snapshots of Universal Storage API integrated systems. |
