# Adding File Backup Jobs

In this article

You can use the cmdlets in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [New-VBRNASBackupJobObject](new-vbrnasbackupjobobject.md) | Defines settings of files and folders and enterprise NAS system that will be added to the file backup job. |
| [Set-VBRNASBackupJobObject](set-vbrnasbackupjobobject.md) | Modifies settings of files and folders that will be added to the file backup job. |
| [Add-VBRNASBackupJob](add-vbrnasbackupjob.md) | Creates file backup jobs. |
| [Set-VBRNASBackupJob](set-vbrnasbackupjob.md) | Modifies file backup jobs. |
| [New-VBRNASBackupPathMask](new-vbrnasbackuppathmask.md) | Defines an exclusion mask for path for a file backup job. |
| [New-VBRNASBackupWildcardMask](new-vbrnasbackupwildcardmask.md) | Defines a wildcard mask for file backup job. |
