# Replica Re-IP Rules


Configure replica re-IP rules. The re-IP rules are applicable for local or cloud replication jobs.

| VMware | Hyper-V | Operation |
| --- | --- | --- |
| [Get-VBRViReplicaReIpRule](get-vbrvireplicareiprule.md) | [Get-VBRHvReplicaReIpRule](get-vbrhvreplicareiprule.md) | Returns re-IP rules configured for a selected job |
| [New-VBRViReplicaReIpIPv6Rule](new-vbrvireplicareipipv6rule.md) | [New-VBRHvReplicaReIpIPv6Rule](new-vbrhvreplicareipipv6rule.md) | Creates an object with re-IPv6 rules |
| [New-VBRViReplicaReIpRule](new-vbrvireplicareiprule.md) | [New-VBRHvReplicaReIpRule](new-vbrhvreplicareiprule.md) | Creates an object with re-IPv4 rules |
