# VBRAzureIRTargetVmInfo


Contains information on a Microsoft Azure VM created during Instant Recovery to Microsoft Azure.

Properties

| Property | Type | Description |
| --- | --- | --- |
| ResourceGroup | String | A resource group. |
| VMName | String | A name of a Microsoft Azure VM. |
| SubscriptionName | String | A Microsoft Azure resource group. |
| VirtualNetwork | String | A Microsoft Azure virtual network. |
| VirtualSubnet | String | A Microsoft Azure virtual subnet. |
| ApplianceSubnet | String | Microsoft Azure virtual network subnets. |
