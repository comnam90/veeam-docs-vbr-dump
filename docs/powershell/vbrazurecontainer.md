# VBRAzureContainer


Contains Microsoft Azure entities.

| Property | Type | Description |
| --- | --- | --- |
| Region | [VBRAzureSubscription](vbrazuresubscription.md) | Microsoft Azure account subscription. |
| Location | [VBRAzureLocation](vbrazurelocation.md) | Microsoft Azure region. |
| Entity | Object[] | Azure VMs, Azure tags, Azure Availability Zone. |
| ExcludeEntities | Bool | Excludes Azure VMs, Azure tags, Azure Availability Zone. |
| Excluded Entity | Object[] | Excluded Azure VMs, Azure tags, Azure Availability Zone. |
