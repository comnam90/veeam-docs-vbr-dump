# Adding Object Storage Backup Jobs


You can use the cmdlets in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [New-VBRObjectStorageBackupJobObject](new-vbrobjectstoragebackupjobobject.md) | Defines a scope of objects that will be added to the object storage backup job. |
| [Set-VBRObjectStorageBackupJobObject](set-vbrobjectstoragebackupjobobject.md) | Modifies settings of objects that will be added to the object storage backup job. |
| [Add-VBRObjectStorageBackupJob](add-vbrobjectstoragebackupjob.md) | Creates object storage backup job. |
| [Set-VBRObjectStorageBackupJob](set-vbrobjectstoragebackupjob.md) | Modifies settings of object storage backup job. |
