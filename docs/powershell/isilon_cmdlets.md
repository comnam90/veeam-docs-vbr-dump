# Dell PowerScale (Formerly Isilon) Storage Systems

In this article

Add Dell PowerScale (formerly Isilon) storage systems to your Veeam Backup & Replication console and administrate them.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRIsilonHost](add-vbrisilonhost.md) | Adds Dell PowerScale storage systems to Veeam Backup & Replication. |
| [Get-VBRIsilonHost](get-vbrisilonhost.md) | Returns Dell PowerScale storage systems. |
| [Set-VBRIsilonHost](set-vbrisilonhost.md) | Modifies Dell PowerScale storage settings. |
| [Sync-VBRIsilonHost](sync-vbrisilonhost.md) | Starts a rescan session of Dell PowerScale storage system. |
| [Remove-VBRIsilonHost](remove-vbrisilonhost.md) | Removes Dell PowerScale storage systems from Veeam Backup & Replication. |
| [Get-VBRIsilonInfrastructureVolume](get-vbrisiloninfrastructurevolume.md) | Returns Dell PowerScale storage volumes from the Veeam Backup & Replication infrastructure. |
| [Get-VBRIsilonVolume](get-vbrisilonvolume.md) | Returns storage volumes from the Dell PowerScale storage infrastructure. |
| [Sync-VBRIsilonVolume](sync-vbrisilonvolume.md) | Starts a rescan session of Dell PowerScale storage volumes from the Veeam Backup & Replication infrastructure. |
