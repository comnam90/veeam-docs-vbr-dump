# Configuring Security Settings

In this article

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Generate-VBRBackupServerCertificate](generate-vbrbackupservercertificate.md) | Generates a TLS certificate that is used to establish a secure connection from backup infrastructure components to the backup server. |
| [Add-VBRBackupServerCertificate](add-vbrbackupservercertificate.md) | Assigns a self-signed TLS certificate to the Veeam Backup & Replication settings. |
| [Get-VBRBackupServerCertificate](get-vbrbackupservercertificate.md) | Returns TLS certificates. |
| [Get-VBRSecurityOptions](get-vbrsecurityoptions.md) | Returns Veeam Backup & Replication security settings. |
| [Set-VBRSecurityOptions](set-vbrsecurityoptions.md) | Modifies Veeam Backup & Replication security settings. |
| [Export-VBRLinuxKnownHost](export-vbrlinuxknownhost.md) | Exports Linux SSH fingerprints to a file. |
| [Import-VBRLinuxKnownHost](import-vbrlinuxknownhost.md) | Imports Linux TLS fingerprints from a file. |
| [Get-VBRLinuxTrustedHostPolicy](get-vbrlinuxtrustedhostpolicy.md) | Returns the trust policy for Linux hosts. |
| [Set-VBRLinuxTrustedHostPolicy](set-vbrlinuxtrustedhostpolicy.md) | Sets the trust policy for Linux hosts. |
