# Tenant Backups

In this article

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Get-VBREntraIDTenantBackup](get-vbrentraidtenantbackup.md) | Returns backups of Microsoft Entra ID tenants. |
| [Get-VBREntraIDTenantItem](get-vbrentraidtenantitem.md) | Returns Microsoft Entra ID items that were backed up. |
| [Get-VBREntraIDTenantRestorePoint](get-vbrentraidtenantrestorepoint.md) | Returns restore points of a Microsoft Entra ID tenant backup. |
