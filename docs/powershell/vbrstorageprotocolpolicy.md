# VBRStorageProtocolPolicy

In this article

Contains protocol policies for storage systems.

Created by

[New-VBRStorageProtocolPolicy](new-vbrstorageprotocolpolicy.md)

Properties

| Property | Type | Description |
| --- | --- | --- |
| iSCSI | bool | Indicates that the storage works over iSCSI protocol. |
| FibreChannel | bool | Indicates that the storage works over FibreChannel protocol. |
| NFS | bool | Indicates that the storage works over NFS protocol. |
