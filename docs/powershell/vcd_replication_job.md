# Cloud Director Replication Job

In this article

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRvCDReplicaJob](add-vbrvcdreplicajob.md) | Creates Cloud Director replication jobs. |
| [Get-VBRvCDReplicaJob](get-vbrvcdreplicajob.md) | Returns Cloud Director replication jobs. |
| [Get-VBRvCDReplica](get-vbrvcdreplica.md) | Returns Cloud Director replicas. |
| [Remove-VBRvCDReplica](remove-vbrvcdreplica.md) | Removes Cloud Director replicas. |
| [Set-VBRvCDReplicaJob](set-vbrvcdreplicajob.md) | Modifies Cloud Director replication jobs. |
| [Enable-VBRvCDReplicaJob](enable-vbrvcdreplicajob.md) | Enabes Cloud Director replication jobs. |
| [Disable-VBRvCDReplicaJob](disable-vbrvcdreplicajob.md) | Disables Cloud Director replication jobs. |
| [Start-VBRvCDReplicaJob](start-vbrvcdreplicajob.md) | Starts Cloud Director replication jobs. |
| [Stop-VBRvCDReplicaJob](stop-vbrvcdreplicajob.md) | Stops Cloud Director replication jobs. |
| [Remove-VBRvCDReplicaJob](remove-vbrvcdreplicajob.md) | Removes Cloud Director replication jobs. |
