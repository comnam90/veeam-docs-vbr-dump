# Adding Amazon S3 and S3 Compatible Object Storage

In this article

You can use the cmdlets in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRAmazonS3Repository](add-vbramazons3repository.md) | Adds Amazon S3 object storage repository to the backup infrastructure. |
| [Add-VBRAmazonS3CompatibleRepository](add-vbramazons3compatiblerepository.md) | S3 compatible and S3-integrated object storage repositories to the backup infrastructure. |
| [Add-VBRAmazonS3GlacierRepository](add-vbramazons3glacierrepository.md) | Adds Amazon S3 Glacier archive storage repository to the backup infrastructure. |
| [Add-VBRS3GlacierCompatibleRepository](add-vbrs3glaciercompatiblerepository.md) | Adds S3 compatible object storage with data archiving to the backup infrastructure. |
| [Set-VBRAmazonS3Repository](set-vbramazons3repository.md) | Modifies settings for Amazons S3 object storage repository added to the backup infrastructure. |
| [Set-VBRAmazonS3CompatibleRepository](set-vbramazons3compatiblerepository.md) | Modifies settings for S3 compatible object storage repository added to the backup infrastructure. |
| [Set-VBRAmazonS3GlacierRepository](set-vbramazons3glacierrepository.md) | Modifies settings for Amazon S3 Glacier archive object storage repository added to the backup infrastructure. |
| [Set-VBRS3GlacierCompatibleRepository](set-vbrs3glaciercompatiblerepository.md) | Modifies settings of S3 compatible object storage with data archiving. |
