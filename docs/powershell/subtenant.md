# Veeam Cloud Connect Subtenants

In this article

You can use the cmdlets from this section to perform the following operations with cloud subtenants accounts:

| Cmdlet | Operation |
| --- | --- |
| [New-VBRCloudSubTenantResource](new-vbrcloudsubtenantresource.md) | Creates quotas of cloud subtenant resources |
| [Set-VBRCloudSubTenantResource](set-vbrcloudsubtenantresource.md) | Modifies settings of cloud subtenant resources |
| [Add-VBRCloudSubTenant](add-vbrcloudsubtenant.md) | Creates new cloud subtenant accounts |
| [Add-VBRvCDCloudSubTenant](add-vbrvcdcloudsubtenant.md) | Creates Cloud Director subtenant accounts |
| [Get-VBRCloudSubTenant](get-vbrcloudsubtenant.md) | Looks for cloud subtenant accounts |
| [Set-VBRCloudSubTenant](set-vbrcloudsubtenant.md) | Modifies cloud subtenant accounts settings |
| [Remove-VBRCloudSubTenant](remove-vbrcloudsubtenant.md) | Removes cloud subtenant accounts |
| [Disable-VBRCloudSubTenant](disable-vbrcloudsubtenant.md) | Disables cloud subtenant accounts |
| [Enable-VBRCloudSubTenant](enable-vbrcloudsubtenant.md) | Enables disabled cloud subtenant accounts |
