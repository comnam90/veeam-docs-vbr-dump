# Lenovo ThinkSystem Storage Systems

In this article

Add Lenovo ThinkSystem storage systems to your  Veeam Backup & Replication console and administrate them.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Add-ThinkSystemHost](add-thinksystemhost.md) | Adds ThinkSystem storage systems to Veeam Backup & Replication. |
| [Get-ThinkSystemHost](get-thinksystemhost.md) | Returns ThinkSystem storage systems added to Veeam Backup & Replication. |
| [Set-ThinkSystemHost](set-thinksystemhost.md) | Modifies ThinkSystem storage settings. |
| [Sync-ThinkSystemHost](sync-thinksystemhost.md) | Starts a rescan session of ThinkSystem storage systems. |
| [Remove-ThinkSystemHost](remove-thinksystemhost.md) | Removes ThinkSystem storage systems. |
| [Get-ThinkSystemVolume](get-thinksystemvolume.md) | Returns ThinkSystem storage volumes from the Veeam Backup & Replication infrastructure. |
| [Get-ThinkSystemInfrastructureVolume](get-thinksysteminfrastructurevolume.md) | Returns storage volumes from the ThinkSystem storage infrastructure. |
| [Sync-ThinkSystemVolume](sync-thinksystemvolume.md) | Starts a rescan session of ThinkSystem storage volumes from the Veeam Backup & Replication infrastructure. |
| [Add-ThinkSystemSnapshot](add-thinksystemsnapshot.md) | Creates ThinkSystem storage snapshots. |
| [Get-ThinkSystemSnapshot](get-thinksystemsnapshot.md) | Looks for ThinkSystem storage snapshots. |
| [Remove-ThinkSystemSnapshot](remove-thinksystemsnapshot.md) | Removes ThinkSystem storage snapshots. |
