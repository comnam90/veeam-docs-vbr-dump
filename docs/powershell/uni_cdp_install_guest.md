# CDP Guest Service Management

In this article

You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |

| [Uninstall-VBRDiscoveredComputerCDPAgent](uninstall-vbrdiscoveredcomputercdpagent.md) | Removes Veem CDP Guest Service and Veeam CDP volume filter driver. |
