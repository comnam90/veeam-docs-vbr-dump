# CDP Guest Service Management


You can use the cmdlet in this topic to perform the following operations.

CDP Guest Service Management

| Cmdlet | Operation |
| [Install-VBRDiscoveredComputerCDPAgent](install-vbrdiscoveredcomputercdpagent.md) | Installs Veeam CDP Guest Service and Veeam CDP volume filter driver. |
| [Uninstall-VBRDiscoveredComputerCDPAgent](uninstall-vbrdiscoveredcomputercdpagent.md) | Removes Veeam CDP Guest Service and Veeam CDP volume filter driver. |
