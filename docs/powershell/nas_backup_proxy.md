# Adding File Share Backup Proxies


You can use the cmdlets in this topic to perform the following operations.

In This Section

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRNASProxyServer](add-vbrnasproxyserver.md) | Adds the general-purpose backup proxy to the Veeam Backup & Replication infrastructure. |
| [Get-VBRNASProxyServer](get-vbrnasproxyserver.md) | Returns general-purpose backup proxies added to the Veeam Backup & Replication infrastructure. |
| [Set-VBRNASProxyServer](set-vbrnasproxyserver.md) | Modifies the general-purpose backup proxy added to the Veeam Backup & Replication infrastructure. |
| [Remove-VBRNASProxyServer](remove-vbrnasproxyserver.md) | Removes general-purpose backup proxies from the Veeam Backup & Replication infrastructure. |
| [Enable-VBRNASProxyServer](enable-vbrnasproxyserver.md) | Enables general-purpose backup proxies. |
| [Disable-VBRNASProxyServer](disable-vbrnasproxyserver.md) | Disables general-purpose backup proxies. |
