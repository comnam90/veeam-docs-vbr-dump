# Performance Tier


You can use the cmdlets in this topic to perform the following operations:

| Cmdlet | Operation |
| --- | --- |
| [Get-VBRRepositoryExtent](get-vbrrepositoryextent.md) | Returns scale-out backup repository performance extents. |
| [Set-VBRRepositoryExtent](set-vbrrepositoryextent.md) | Modifies performance extents of scale-out backup repositories. |
| [Enable-VBRRepositoryExtentMaintenanceMode](enable-vbrrepositoryextentmaintenancemode.md) | Enables Maintenance mode for performance extents of a scale-out backup repository. |
| [Disable-VBRRepositoryExtentMaintenanceMode](disable-vbrrepositoryextentmaintenancemode.md) | Disables Maintenance mode for performance extents of a scale-out backup repository. |
| [Enable-VBRRepositoryExtentSealedMode](enable-vbrrepositoryextentsealedmode.md) | Enables Sealed mode for performance extents of a scale-out backup repository. |
| [Disable-VBRRepositoryExtentSealedMode](disable-vbrrepositoryextentsealedmode.md) | Disables Sealed mode for performance extents of a scale-out backup repository. |
| [Start-VBRRepositoryExtentBackupEvacuation](start-vbrrepositoryextentbackupevacuation.md) | Evacuates backups from performance extents. |
