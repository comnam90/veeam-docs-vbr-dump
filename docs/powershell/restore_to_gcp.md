# Restore to Google Compute Engine


You can use the cmdlet in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRGoogleCloudComputeAccount](add-vbrgooglecloudcomputeaccount.md) | Adds Google Cloud service accounts to Veeam Backup & Replication |
| [Get-VBRGoogleCloudComputeAccount](get-vbrgooglecloudcomputeaccount.md) | Returns Google Cloud service accounts added to Veeam Backup & Replication. |
| [Set-VBRGoogleCloudComputeAccount](set-vbrgooglecloudcomputeaccount.md) | Modifies Google Cloud service accounts added to Veeam Backup & Replication. |
| [Remove-VBRGoogleCloudComputeAccount](remove-vbrgooglecloudcomputeaccount.md) | Removes Google Cloud service account credentials records from Veeam Backup & Replication. |
| [Get-VBRGoogleCloudComputeRegion](get-vbrgooglecloudcomputeregion.md) | Returns Google Cloud compute regions. |
| [Get-VBRGoogleCloudComputeZone](get-vbrgooglecloudcomputezone.md) | Returns Google Cloud compute zones. |
| [Get-VBRGoogleCloudComputeInstanceType](get-vbrgooglecloudcomputeinstancetype.md) | Returns Google Cloud compute engine instance types. |
| [Get-VBRGoogleCloudComputeVPC](get-vbrgooglecloudcomputevpc.md) | Returns a Google Cloud Compute Engine VPC. |
| [Get-VBRGoogleCloudComputeSubnet](get-vbrgooglecloudcomputesubnet.md) | Returns subnets for Google Cloud compute region. |
| [Get-VBRGoogleCloudComputeSharedSubnet](get-vbrgooglecloudcomputesharedsubnet.md) | Returns subnets for Google Cloud shared VPC. |
| [New-VBRGoogleCloudComputeLabel](new-vbrgooglecloudcomputelabel.md) | Creates Google Cloud labels. |
| [New-VBRGoogleCloudComputeProxyAppliance](new-vbrgooglecloudcomputeproxyappliance.md) | Creates helper appliances for restoring Linux VMs to Google Cloud. |
| [New-VBRGoogleCloudComputeDiskConfiguration](new-vbrgooglecloudcomputediskconfiguration.md) | Creates storage volume settings for Google Cloud instances. |
| [Start-VBRVMRestoreToGoogleCloud](start-vbrvmrestoretogooglecloud.md) | Starts a restore session to Google Cloud. |
| [Get-VBRGoogleCloudRestoreSession](get-vbrgooglecloudrestoresession.md) | Returns active restore sessions to Google Cloud. |
| [Stop-VBRVMRestoreToGoogleCloud](stop-vbrvmrestoretogooglecloud.md) | Stops running restore to Google Cloud sessions. |
