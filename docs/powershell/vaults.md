# Tape Vaults


|  |
| --- |
| ![Tape Vaults](images/icon_note.webp) Note: |
| Vaults require Enterprise or Enterprise Plus, Veeam Universal License product edition. |

You can use the cmdlets from this section to perform the following operations with media pools:

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRTapeVault](add-vbrtapevault.md) | Creates vault. |
| [Get-VBRTapeVault](get-vbrtapevault.md) | Returns vaults. |
| [Remove-VBRTapeVault](remove-vbrtapevault.md) | Removes vaults. |
| [Set-VBRTapeVault](set-vbrtapevault.md) | Modifies vaults. |
