# Veeam Cloud Connect Network Appliances

In this article

The network extension appliance on the user side is required if you plan to replicate to cloud. In the cloud replica scenarios, the network extension appliance is used for partial site failover.

You can use the cmdlets from this section to create and manage network extension appliances on the user side.

| Cmdlet | Operation |
| --- | --- |
| [New-VBRCloudProviderNetworkAppliance](new-vbrcloudprovidernetworkappliance.md) | Creates a user network appliance |
| [Get-VBRCloudProviderNetworkAppliance](get-vbrcloudprovidernetworkappliance.md) | Returns user network appliances |
| [Set-VBRCloudProviderNetworkAppliance](set-vbrcloudprovidernetworkappliance.md) | Modifies user network appliance |
