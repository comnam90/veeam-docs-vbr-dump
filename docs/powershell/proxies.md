# Proxies


You can use the cmdlets in this topic to perform the following operations.

| Cmdlet | Operation |
| --- | --- |
| [Add-VBRViProxy](add-vbrviproxy.md) | Adds VMware backup proxies to the backup infrastructure. |
| [Get-VBRViProxy](get-vbrviproxy.md) | Returns VMware backup proxies. |
| [Disable-VBRViProxy](disable-vbrviproxy.md) | Disables a VMware backup proxy. |
| [Enable-VBRViProxy](enable-vbrviproxy.md) | Enables a VMware backup proxy that was previously disabled. |
| [Set-VBRViProxy](set-vbrviproxy.md) | Modifies VMware backup proxies. |
| [Remove-VBRViProxy](remove-vbrviproxy.md) | Removes VMware backup proxy from the backup infrastructure. |
| [Add-VBRHvProxy](add-vbrhvproxy.md) | Adds Hyper-V backup proxies to the backup infrastructure. |
| [Get-VBRHvProxy](get-vbrhvproxy.md) | Returns Hyper-V backup proxies. |
| [Disable-VBRHvProxy](disable-vbrhvproxy.md) | Disables a Hyper-V backup proxy. |
| [Enable-VBRHvProxy](enable-vbrhvproxy.md) | Enables a Hyper-V backup proxy that was previously disabled. |
| [Remove-VBRHvProxy](remove-vbrhvproxy.md) | Removes a specified Hyper-V backup proxy from the backup infrastructure. |
| [Add-VBRViLinuxProxy](add-vbrvilinuxproxy.md) | Adds Linux backup proxies to the backup infrastructure. |
| [Set-VBRViLinuxProxy](set-vbrvilinuxproxy.md) | Modifies settings of Linux backup proxies. |
| [Get-VBRJobProxy](get-vbrjobproxy.md) | Returns proxy servers assigned to a selected job. |
| [Set-VBRJobProxy](set-vbrjobproxy.md) | Assigns a specific proxy to a job or enables automatic proxy selection. |

|  |
| --- |
| Note |
| Operations with file backup proxies are described in the [Adding File Share Backup Proxies](nas_backup_proxy.md) section. |
