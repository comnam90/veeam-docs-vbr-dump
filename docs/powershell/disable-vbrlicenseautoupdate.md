# Disable-VBRLicenseAutoUpdate

In this article

Short Description

Disables the automatic license update option.

Applies to

Product Edition: Standard, Enterprise, Enterprise Plus, Veeam Universal License

Syntax

|  |
| --- |
| Disable-VBRLicenseAutoUpdate Â [<CommonParameters>] |

Detailed Description

This cmdlet disables the automatic license update option.

Run the [Enable-VBRLicenseAutoUpdate](enable-vbrlicenseautoupdate.md) cmdlet to enable the automatic license update option.

<CommonParameters>

This cmdlet supports Microsoft PowerShell common parameters. For more information on common parameters, see  [Microsoft Docs](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.core/about/about_commonparameters?view=powershell-7).

Example

Disabling Automatic License Update Option

This command disables the automatic license update option.
