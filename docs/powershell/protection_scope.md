# Veeam Agent Management Protection Scope


You can use the cmdlet in this topic to perform the following operations.

| Cmlet | Operation |
| --- | --- |
| [New-VBRADContainer](new-vbradcontainer.md) | Creates a scope of Active Directory objects. |
| [New-VBRManuallyDeployedContainer](new-vbrmanuallydeployedcontainer.md) | Defines a scope of Veeam Agent setup files. |
| [Set-VBRADContainer](set-vbradcontainer.md) | Modifies the scope of Active Directory objects. |
| [Get-VBRADDomain](get-vbraddomain.md) | Creates an Active Directory domain connection object. |
| [Find-VBRADEntity](find-vbradentity.md) | Looks for Active Directory objects. |
| [New-VBRADCustomCredentials](new-vbradcustomcredentials.md) | Specifies credentials for Active Directory objects. |
| [New-VBRIndividualComputerContainer](new-vbrindividualcomputercontainer.md) | Creates a scope of individual computers. |
| [Set-VBRIndividualComputerContainer](set-vbrindividualcomputercontainer.md) | Modifies the scope of individual computers. |
| [New-VBRIndividualComputerCustomCredentials](new-vbrindividualcomputercustomcredentials.md) | Specifies credentials for individual computers. |
| [New-VBRCSVContainer](new-vbrcsvcontainer.md) | Creates a scope of computers from a CSV file. |
| [Set-VBRCSVContainer](set-vbrcsvcontainer.md) | Modifies a scope of computers created from a CSV file. |
| [New-VBRCSVCustomCredentials](new-vbrcsvcustomcredentials.md) | Specifies custom credentials for computers from a CSV file. |
| [Get-VBRAmazonEC2VM](get-vbramazonec2vm.md) | Returns Amazon EC2 instances. |
| [New-VBRAmazonEC2Container](new-vbramazonec2container.md) | Defines a scope of Amazon EC2 instances, EC2 tags or AWS datacenters for a protection group. |
| [Set-VBRAmazonEC2Container](set-vbramazonec2container.md) | Modifies a scope of Amazon EC2 instances, EC2 tags or AWS datacenters for a protection group. |
| [Get-VBRAzureVM](get-vbrazurevm.md) | Returns Azure VMs. |
| [New-VBRAzureTag](new-vbrazuretag.md) | Defines Azure metadata tags. |
| [New-VBRAzureContainer](new-vbrazurecontainer.md) | Defines a scope of Azure VMs, Azure tags or Azure Availability Zone for a protection group. |
| [Set-VBRAzureContainer](set-vbrazurecontainer.md) | Modifies a scope of Azure VMs, Azure tags or Azure Availability Zone for a protection group. |
