# Creating Application Backup Policy

In this article

To back up MongoDB data, you must configure an application backup policy in Veeam Backup & Replication.

Before configuring an application backup policy, [check prerequisites](mongo_policy_before.md). Then use the New Application Backup Policy wizard to define settings for the backup policy.

1. [Launch the New Application Backup Policy wizard](mongo_policy_launch.md).
2. [Specify policy name and description](mongo_policy_name.md).
3. [Select deployments to back up](mongo_policy_deployments.md).
4. [Specify backup storage settings](mongo_policy_storage.md).
5. [Specify advanced backup settings](mongo_policy_advanced.md).
6. [Specify database processing settings](mongo_policy_preferences.md).
7. [Specify the backup schedule](mongo_policy_schedule.md).
8. [Review backup policy settings](mongo_policy_summary.md).
